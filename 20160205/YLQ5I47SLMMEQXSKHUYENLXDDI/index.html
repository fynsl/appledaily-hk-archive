【港女衣學】凍住都要靚　一條披肩無限打法

近兩個星期日日都只得十多度，最低氣溫更只有十度。要保暖除了穿得像隻糭，其實一條披肩圍着頸保暖功效更佳，而且有無限打法造型更百變。

【經典打法】
難度：★
1. 把長披肩對摺成正方形。
2. 對角摺成三角巾。
3. 把三角巾圍在頸上在頸後打結。
4. 把兩端於三角巾下打結便完成。

【修長打法】
難度：★★
1. 把長披肩摺成三角巾。
2. 向下捲兩三次。
3. 把三角巾搭於頸後。
4. 把左端放在右膊上，繞到頸後。
5. 把右端收於頸圈內。

【擯辮打法】
難度：★★★
1. 同樣先把長披肩摺成三角巾。
2. 不停向下捲。
3. 把整卷對摺並放在頸後。
4. 把一端穿入頸圈。
5. 另一端則由下穿入頸圈。

【脆麻花打法】
難度：★★★★
1. 同樣先把長披肩摺成三角巾。
2. 向下捲兩三次。
3. 把三角巾圍在頸前並在頸後打結。
4. 把右端於頸圈內打成一小圈。
5. 把左端穿入小圈即成。
