保羅車禍最轟動

藝人車禍意外經常發生，其中以下數宗造成藝人死亡的交通意外轟動全球。

【美國 保羅獲加】

傷亡：兩死
《狂野時速》男星保羅獲加13年11月在美國加州坐朋友駕駛的保時捷失控撞燈柱，二人遭燒死，保羅終年40歲。

【韓國 Ladies' Code】

傷亡：兩死三傷
女團Ladies' Code 14年9月深夜坐車返首爾時遇車禍，21歲的高恩妃和23歲的權梨世先後死亡，另三名成員停工至今。

【台灣 許瑋倫】

傷亡：一死一傷
許瑋倫07年1月坐女助手駕駛私家車，從台北趕往南投拍劇遇交通意外，頭部重創，搶救兩日後傷重不治，終年28歲。

【香港 陳僖儀】

傷亡：一死
陳僖儀13年4月凌晨獨自駕車返家，在油麻地海寶路天橋狂撼路中燈箱至當場頸骨折斷，送院後不治，終年26歲。
