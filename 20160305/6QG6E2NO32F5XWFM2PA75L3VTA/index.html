【瑜伽教室】三式睡前瑜伽　安眠瘦身去水腫

不少靚靚OL日做夜做，長期坐在電腦前，頸梗膊痛腰又越坐越粗。其 實睡前抽15分鐘做簡單瑜伽動作，除可鬆筋、放鬆心情易入睡，更可去水腫瘦身。我們找來瑜伽導師Alice Ng示範三式睡前瑜伽，既簡單又有效。

第一式 天鵝扭腰

功效：鬆腰及肩頸、舒緩經痛
動作：
1. 俯卧，用雙手撐起上半身
2. 雙膝跪起，雙手向前延伸拉鬆腰部
3. 右手向左打橫伸展
4. 左手放於右邊後腰維持五秒
5. 換邊重複

第二式 俯卧十字分筋

功效：鬆腳筋、肩頸、去水腫
動作：
1. 挺腰坐在地上，雙腳各前伸直
2. 雙手及腰部向左扭90度並俯卧地上
3. 左腳向頭部方向提高，與右腳呈直角
4. 右手拉左腳腳掌
5. 頭轉向望左，左手撐起
6. 維持五秒，換邊重複

第三式 蛇式

功效：紓緩壓力、鬆弛筋骨、促進血液循環
動作：
1. 先挺腰安坐，雙腳伸直
2. 左手曲臂提起，捉緊右邊肩膀
3. 右手垂低緊貼大腿，平躺
4. 左腳向胸提起屈曲
5. 左腳扭向右，右手拉緊腳掌
6. 維持五秒，換邊重複
