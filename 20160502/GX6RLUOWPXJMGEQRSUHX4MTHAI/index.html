【吾keep吾fit】瑜伽fitness齊做一分鐘　擴胸收腹迎夏日

有人喜歡做瑜伽，有人喜歡做fitness體能鍛煉，adidas近期就鼓勵大家，瑜伽和fitness互相配合一齊做。瑜伽導師魏秋琪（Stefanie）認為瑜伽和體能鍛煉是息息相關，「雖然兩者皆是對身體的一種鍛煉，但瑜伽動作的維持時間長，可讓身體部位有一定的擴展，是做體能運動前的最佳橋樑。」而專業教練梁愷傑（Ocean）更覺得做fitness前做一些拉筋的瑜伽動作，可以令效用會更大，還可以有助減低受傷的機會。」二人即席各自示範了一套動作，先做瑜伽動作Turbo dog，再做Fitness動作Hand Release Push up，認為這樣配合更易練出靚線條。兩套動作加起來也只不過一分鐘，只要勤於在家中練習，包你很快成功擴胸收腹迎接夏天的來臨。
怕自己一個人做冇動力？adidas除在添馬公園免費開班外，將於5月2日在銅鑼灣開幕的全新旗艦店，亦有免費瑜伽及運動等課堂，這更是首間設有男女浴室的店舖，方便大家做運動，有人陪就唔怕自己偷懶唔願郁啦。

瑜珈動作：Turbo dog
1. 猶如下犬式的動作趴在地上。
2. 十隻手指撐開。
3. 手肘需要微微彎曲，再將雙肩膀向內拉。
4. 手肘並非指向外，是指向內旋轉。
5. 保持這動作約十個呼吸。
作用：可以強化上身拉力，鍛煉胸部肌肉。
初學提示：如果腳掌未能貼地，可以彎少少膝蓋。

Fitness動作：Hand Release Push up
1. 首先趴在地上，並心口貼地。
2. 雙手放在肩膀之下，闊度大一點也不成問題。
3. 嘗試將雙手離地。
4. 然後呼氣時盡力收緊核心肌肉（Core），再全身用力向上推。
5. 隨後慢慢將身體下降。
6. 雙手再離一離地，繼續重複再向上推的動作。
7. 每一組做十至十五下，然後休息一分鐘，每次三至四組。
作用：雙手離地有助打開肩膀關節的幅度，離心收縮動作可以鍛煉上身力量及腹部肌肉。
初學提示：若感到吃力，可以試將雙膝貼地作支撐點，雙腳交叠做這動作會較輕鬆。
