【電腦巡場】Note 7如箭在弦　Note 5水貨價迫降$3,380

Note 7已經如箭在弦！儘管大家對佢個名還存在深深芥蒂，不過為咗9月份的改朝換代，好多手機舖頭已經開始做落價清貨了！今次被落價的是Samsung Note 5水貨，32GB版機價由上周的$3,880直插落$3,380，下調價格是$500，以一次性的減價幅度來說，可謂非常之高，須知它還是現役的機王級手機。

另一部被借勢下調價格的，是上代機王S6 Edge，不過留意這部機可是行貨呢！S6 Edge 32GB版上周價格為$4,200，最新調整價格為$3,580；至於64GB版，價錢亦由$4,998下調至$4,080。

查詢：泡菜屋（深水埗桂林街141A高登電腦廣場新翼1樓33號）
