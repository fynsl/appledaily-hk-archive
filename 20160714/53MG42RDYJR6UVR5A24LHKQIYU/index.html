【台灣直擊】平薄贏Surface Pro4 Transformer 3 Pro爭做最強Win10平版筆電

在剛舉行的華碩台灣發佈會中，除了焦點的ZenFone 3系列外，其實品牌還有一部值得留意秘密武器，就是真正今天才全新發佈的Transformer 3 Pro，外型跟Surface Pro4相當相似，但Transformer 3 Pro機身只有8.35mm厚，比Surface Pro4的8.45mm更薄了一點。但說到重量Transformer 3 Pro就有輸蝕，比對手較重29克，但考慮到它的12.6吋屏幕比Surface Pro4的12.3吋稍大了0.3吋，更加設了新型的USB-C插口，可算是打過平手。、另外，新機的手寫筆亦跟Surface Pro4一樣有1024級壓感，外型亦頗為相似，畫寫感覺很不錯，但感覺上筆觸跟筆尖的速度較慢了一點，而現場亦有插畫師即場示範畫畫，效果不俗。 規格上新機跟Surface Pro4亦頗相似，配備頗高階的硬件，但香港暫只會有兩種規格選擇，不過就勝在售價比Surface Pro4便宜，加上都預設Win10 Pro，並有頗完善的配件選擇，更特別的是能作兩個4K畫面輸出，甚至可玩近期大熱的VR遊戲，是Win10用家的不錯選擇。

電話方面除ZenFone 3和ZenFone 3 Delux，其實還有一部6.8吋巨屏的ZenFone 3 Ultra，以聲畫效能作賣點，聲稱具有畫質優化及24bit/192 kHz通過Hi-Res Audio認證的聲效輸出，可惜是現場新機並無影片可供測試體驗。另外ZenFone 3 Ultra亦配置有跟ZenFone 3 Delux同級的2,300萬像素鏡頭及4,600mAh電池，更可以1.5A輸出電源為其他裝置充電，功能相當全能。不過個人覺得6.8吋屏幕作為電話實屬過大，未必為一般人受落，幸好它設有簡易的放大縮小屏幕功能，令操作上方便了一點 。

最後同場其實還有一部平板電腦ZenPad 3S 10發佈，以5.32mm窄邊框作賣點，配9.7吋2048x1536 178度超廣視角屏幕，設有跟ZenFone 3 Ultra一樣的高階聲畫效能，加上5,900mAh電池，可連播10小時影片，作為隨身影院的確不錯，不過可惜香港暫未有計劃發售。
