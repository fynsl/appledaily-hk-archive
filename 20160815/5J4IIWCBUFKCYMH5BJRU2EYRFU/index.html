指關節起鱗係癌症？
皮膚暗示你要識

皮膚是人體的最大器官，體內發生狀況，多少也會反映在皮膚上，英國皮膚料醫生加赫（Joanna Gach）列出了乳癌、丙型肝炎等疾病出現時皮膚的症狀，讓大家留意。

1. 皮膚泛黃 → 黃疸病
別以為只有嬰兒才會患黃疸病，當肝無法分解膽紅毒這種毒素，令血液累積過多毒素，便出現黃疸；患者除了皮膚，雙眼也可能會泛黃。

2. 胸部凹陷 → 乳癌
若胸部無故出現像橙皮紋的皺紋，甚至紅腫凹陷，可能是患乳癌的徵象。

3. 小腿、腳背起紅疹 → 丙型肝炎
丙型肝炎主要是經血液傳染，嚴重可致肝癌。患者除了出現感冒症狀和胃痛外，有少部份人的小腿、腳背會起紅疹。

4. 皮膚硬化 → 硬皮症
硬皮症（Scleroderma）是免疫系統出現異常，攻擊皮膚下面或體內器官附近組織，令皮膚變硬、失去彈性，嚴重會傷害器官，有生命危險。

5. 指關節呈鱗狀 → 癌症
指關節有鱗狀紅腫，一般是免疫系統失調，導致肌肉發炎；但也有兩成機會是和體內器官有癌細胞，如卵巢癌等。如果上眼簾有紫色疹，並出現體重變輕、咳嗽、盜汗和體虛等情況，便要小心。

不過醫生也強調，皮膚問題多半是因濕疹或受感染所引致，毋須過於驚慌；建議皮膚出現問題3、4日仍未好轉，甚至有惡化迹象，便要求醫。
