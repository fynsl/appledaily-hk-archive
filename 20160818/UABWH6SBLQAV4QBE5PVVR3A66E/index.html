《Deep Web》無警告字句罰4萬

【本報訊】網絡作家恐懼鳥以犯罪為題材的《Deep Web》系列兩本著作，疑涉烹屍等暴力及色情情節，5月被淫褻物品審裁處評為不雅物品；出版商發行兩書時，未有在封面及封底展示顯眼的法定警告字句及出版人資料，遭票控8罪。出版商昨在東區法院承認全部控罪，罰款4萬元。
辯方求情指被告元點媒體有限公司僅有3名員工，在兩書列為不雅後，主動將書籍下架及包上透明膠袋才再發行。公司初次出版此類內容敏感的書籍，做法有不足之處，未在警告標貼印上出版人資料，但書內其實有刊載這些資料。

書商：積存5,000本

兩書被投訴前共賣了約1.5萬冊，再發行時已很少書局願意上架，現於貨倉積存5,000本。
辯方續指，公司去年營業額180萬元，盈利14萬元，而兩書分別售88及98元，但書價其中一半要用來付發行費，餘下一半扣除成本後才是利潤。
案情指電影、報刊及物品管理辦事處人員於今年5月11日在銅鑼灣商務印書館、誠品書店及油麻地中華書局，購得多本《Deep Web File #網絡奇談》（圖）及《Deep Web 2.0 File #人性奇談》，發現有透明包裝，封面封底沒印上禁止售予18歲以下人士的警告字句及出版人營業地址和電話號碼，部份僅在封面或封底其中一面有警告。
案件編號：ESS26229-36/16
