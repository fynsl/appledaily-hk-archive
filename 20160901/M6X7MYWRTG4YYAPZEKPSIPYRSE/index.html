涉充電爆炸　Note 7暫停韓出貨

南韓傳媒報道，三星電子已暫停向當地流動網絡通訊商供應全新旗艦智能手機Galaxy Note 7，業界相信叫停出貨是同該型號手機發生了多宗充電時爆炸的案件有關。
今個月19日才發售的Galaxy Note 7，三星曾寄予厚望，冀能跟蘋果公司即將推出的新iPhone分庭抗禮，單在南韓已賣出了約40萬部，首批新貨亦已經到了香港。但短短兩周，全球發生了最少5宗聲稱Galaxy Note 7充電時爆炸的事件，有燒焦的手機殘骸照片更在網上廣泛流傳，令到三星非常尷尬。
外界相信，三星停止向國內SK電訊、韓國電訊和LG U+等3大網絡商發貨，是要亡羊補牢，盡快查明爆炸原因並修正手機設計。三星昨表示為進行「額外品質檢查」暫停出貨，但無交代詳情和是否涉海外出貨。
