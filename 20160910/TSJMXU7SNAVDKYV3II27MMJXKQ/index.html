美日籲航機上勿用Note 7

南韓三星的新旗艦手機Galaxy Note 7接連發生爆炸事故，被視為對航班安全的威脅，美國聯邦航空管理局（FAA）、日本交通部門、新加坡及台灣多家航空公司，都相繼發佈針對Note 7的使用及託運指引。
繼澳洲航空、澳洲維珍航空及捷星航空之後，美國FAA前天發佈針對三星Galaxy Note 7的指引，除了「強烈勸喻在客機上不要使用及為這裝置充電」，亦建議乘客不要將手機放在寄艙行李內。
指引昨天得到日本運輸部響應，發聲明表示三星Galaxy Note 7雖然未在日本發售，但鑑於在世界各地發生的意外，故此作出一樣建議。

新航華航長榮禁用

新加坡航空、台灣遠東航空、虎航、中華、復興及長榮航空昨天亦作出相關指引，其中新航、華航、長榮及復興更不僅只是勸喻，而是禁止乘客託運或在客機上使用Note 7。美國達美航空回應查詢時表示正考慮相關措施。
三星上月初發佈旗艦Note手機系列的最新型號Note 7，預售反應較預期熱烈，然而手機交予用家後接連發生多宗電池引致爆炸事故，迫使三星本月初史無前例地宣佈全球回收這款手機，涉及250萬部。
最新一宗矚目事故發生在本周一美國勞動節當日，佛羅里達州車主道納克（Nathan Dornacher）將車停泊在草坪後離開片刻，Note 7留在車內充電、車廂開着空調，詎料返回草坪時只見車子在燃燒，冒出濃濃黑煙，最後只燒剩支架。
道納克打趣道這不是理想的假日燒烤活動，慶幸當時沒人在車內，他上載的照片見到手機燒到完全焦黑。三星回應指知悉事件，正在調查。
