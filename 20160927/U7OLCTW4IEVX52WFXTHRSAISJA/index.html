【抹茶控注意】茶道專家實試抹茶雪糕　中村藤吉輸了？

抹茶雪糕向來是香港人最愛，有見及此，市面上越來越多間抹茶雪糕店，但到底哪間味道最正宗最香濃？位於The One的本地品牌雪糕店，The Fun Store開業至今，主打有四級抹茶濃度（15%,25%,40%及50%）的雪糕，亦計劃於十月推出70%濃度的抹茶雪糕。記者就找到茶道裏千家專任講師Connie前往品嚐。

「其實我們不能將茶道用的茶葉跟抹茶甜品相比，」她認真道，「就像紅茶跟紅茶蛋糕一樣，前者本身能夠不加糖不加奶，但後者無法避免要加入很多香料或配料。」她表示，抹茶甜品所用的抹茶粉，本身已與茶道有一定分別。因此對於抹茶雪糕，她認為抹茶與奶味及甜味能達到平衡為最重要。

由於早幾年香港沒有焙茶機械及相應技術，The Fun Store老闆娘便飛到台灣拜師學藝，從而創立本地茶葉品牌「Ms.Tea」，而The Fun Store便是其首間實體店。「如果成份是以100%來形容15%濃度雪糕，奶味大概佔了80%吧，很適合一些怕苦的女生吃。」Connie細細品嚐後說，表示真正的分水嶺其實是40%濃度，「40%的開始有點像茶道的薄茶，即是樹齡較輕的茶葉，味道亦較濃茶淡。」其後的70%苦得更像黑朱古力，她笑說：「如果你食均全香港的抹茶雪糕都覺得不夠濃的話，我覺得你可以來這兒試試。」

其後我們到了由日藉老闆蒲谷康成所開的Via Tokyo，甜品店主打抹茶跟焙茶甜品，每次經過都發現客人們為抹茶雪糕大排長龍。「剛剛我看到了宣傳單張，它上面寫住用了京都宇治一百五十年歷史的抹茶粉，這令我很驚訝。」她指了指宣傳單上的文字，「對我們學茶道的學生而言，宇治市出產的抹茶是最高級的，從前亦是皇室貢品，這證明他們用了很高的成本。」她輕輕勾起一口品嚐，除了高級抹茶粉外，店家更選用北海道3.6牛乳，奶味、甜度與抹茶三者亦做到平衡到位，Connie給於極高的評價，「或許沒有The Fun Store的那麼濃，但是雪糕入口非常Creamy，牛奶跟抹茶亦沒有爭做主角，我覺得很好味。」不過作為一個茶道者，她最期待的還是歷史悠久的中村藤吉。

步到人滿為患的中村藤吉，我們用了五十多元購買了有白玉紅豆的抹茶雪糕，誰知卻來換來一絲失望，「不是說中村藤吉的雪糕不好吃，但是我想不到它的奶味會如此濃厚，甚至有點蓋過抹茶味，如果以A Fun Store作標準的話，這大概是25%以下的抹茶濃度。」她帶點失望說，這時她作出一個小總結，「我最喜歡的是Via Tokyo，因為它的配方能夠將抹茶跟奶味溶合得恰到好處；A Fun Store則帶給我無限驚喜，想不到本地品牌亦能像日本般，將抹茶雪糕分出幾種濃度，味道亦是不俗。」她頓了頓，繼續說，「如果是中村藤吉的話，將它形容是普通的抹茶雪糕是滿好吃，但欠缺驚喜。」
