【終極解碼】身份證號碼括弧有數計　即刻check check佢

日常生活除了被警察查身份證外，圖書館借書，出入香港關口也需要用到你的身份證，而身份證內除了姓名、性別、出生日期以及身份證號碼，其實還有三粒星、兩個英文字母以及一些中文電碼，你認識自己的身份證嗎？

其實在身份證英文名字下總有一堆四位為一組的奇怪數字組合，那些便是你的中文電碼。每一個中文字也有一組四位數字電碼，0001至9999，每個電碼也可以在網上查閱，好像外籍人士「屠龍麥粒花」擁有5個中文字便有5個相應電碼，分別為1458、7893、7796、4721、5363。日常生活中填寫一些政府表格如選民登記表以及政府職位申請表也需要填上中文電碼，目的是方便職員輸入資料時使用。

另外很多人也誤以為身份證上的三粒星代表永久性居民，其實三粒星代表18歲或以上有資格申領回港證，而一星則代表11至17歲有資格申領回港證。如果你發現身份證三粒星後面有英文字「AZ」，當中的「A」代表持證人擁有永久的香港居留權利，而「Z」則代表在香港出生；「X」代表在中國內地出生、「N」代表在首次登記姓名後曾經作出更改。外籍人士「激烈的海𪂇」英文字母為「CO」，「C」表示其居留權受入境事務處處長所限制，「O」則代表在中、港、澳地區以外出世。

其實身份證號碼括弧內的數字亦有數得計，用頭七個號碼來計，當中的英文字母可換算為數字，A為1、B為2、C為3……Y為25、Z為26。再按排序左起乘以相關數字：×8、×7……×2，然後加出總和除以11，得出餘數的再用11減……好複雜吧，舉例睇就好清楚：

假設身分證號碼為C199707(?)，英文字母C=3
3×8 + 1×7 + 9×6 + 9×5 + 7×4 + 0×3 + 7×2 = 172
172÷11>>>餘數為7
11-7=4
因此身份證號碼括弧內的數字為4，全個身份證號碼是C199707(4)

如果計算出括號數字為雙位數字10，號碼便會變成(A)，在數學角度這種計算方法為核對數位。擁有兩個英文字母起頭的身份證計法也相同：

例如：AB123456(?)，英文字母A=1，B＝2
1×9 + 2×8 + 1×7 + 2×6 + 3×5 + 4×4 + 5×3 + 6×2 = 102
總和102÷11得出餘數3
11-3=8
因此身份證號碼得出了AB123456(8)

大家有空亦不妨計算一下自己身份證號碼吧。
