【恐懼埋身】寫書曾受威嚇作家恐懼鳥憂港人前途

【恐懼埋身】寫書曾受威嚇
作家恐懼鳥憂港人前途

2016年可以稱得上令香港人恐懼的一年，年初先有銅鑼灣書店事件，到近期票王朱凱迪受死亡威嚇，恐懼的氛圍彷彿籠罩整個香港。之前因撰寫《Deep Web File 網絡奇談》及《Deep Web File人性奇談》而掀起一場風波的作家恐懼鳥認為，恐怖事若是真真實實存在，就應該納入思考世界如何運作，而非只停留於驚訝逃避。
記者：余傲芝　攝影：吉吉、Neo

大學修讀犯罪心理學的恐懼鳥，至小對光怪陸離的事感興趣，因而毫無懸念走上驚慄故事作家之路。普遍人以為他撰寫的只有都市傳說，但其實有些個案已經去到BBC報導的犯罪記實層面。近期寫過最令他深刻是一系列來自暗網變態佬聚集的討論區的故事，由仔細觀察到心理分析，「呢啲故事無論恐怖故事或學術角度，都係幾稀有嘅例子」。有感中文地區比較少人涉獵這方面的題材，恐懼鳥希望多些香港人看到傳說與古靈精怪的故事。

經常接觸令人毛骨悚然的故事，但這並不代表恐懼鳥甚麼都不怕。「有時候做資料搜集時候要對住嗰啲好恐怖嘅畫，真係幾難受。」但說到較實在的威嚇，就是在資料搜集時遇到。恐懼鳥在撰寫暗網故事時曾到一些邪教網站，之後就會有不知名的人非法登入他的電郵網站帳戶，雖然在他眼中不算太大麻煩，不過他指如果受更大威嚇都會不會卻步。「以我性格係更加寫落去，因為更確定係真或者背後有更多嘅秘密，我認為有更多人要知道」

之前恐懼鳥寫關於暗網的作品，被部份公眾指責教壞人，令兩本書都曾一度下架，但他堅持繼續寫。「其實好多罪惡都可以用平淡嘅方式講，但即使人睇到每年有幾個婦女死於謀殺，都不會有特別大嘅感情，但以故事形式佢哋會更關心關注，因為恐懼令人產生強烈嘅感覺」。被問到究竟他筆下暗網的世界還是現實的世界比較令人恐懼呢？恐懼鳥笑言暗網呈現出來的世界會較戲劇化，但稀釋一下就是我們的現實世界。他坦言對香港前途都有點悲觀，「以之前選舉為例，大家一直以為會有公正程序，但原來又可以多咗選票，又有某部份人因政治主張而被篩選，香港前景應該會有更多令人震驚嘅事發生」。話雖如此，恐懼鳥認為理解才是治癒恐懼最主要的藥方，「如果你了解一樣事情後，有時都會膽怯，但你可以更冷靜去看事情，而非被恐懼沖昏頭腦。」
