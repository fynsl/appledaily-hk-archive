38歲粉紅生日派對吳佩慈：I like this number

38歲粉紅生日派對
吳佩慈：I like this number

台灣女星吳佩慈三年抱兩，為內地富商男友紀曉波生咗一仔一女，湊夠一個「好」字，雖然遲遲未拉埋天窗，不過男友一直好照顧吳佩慈。而吳佩慈亦過住如同「少奶奶」嘅生活。
前日係吳佩慈38歲生日，唔知係唔係為免被打擾，佢昨日凌晨先喺IG貼圖：「I like this number」，相中見佢打扮高貴，身穿長袖碎花連身裙，枱上放有一個兩層嘅蛋糕，上面有一個「38」字，而另一張相似係facebook截圖，未知係唔係吳佩慈嘅私人fb，圖中寫住「今天主題，我沒跟任何人講過，其實是三八阿花」，見現場佈置以花及粉紅色配搭為主。
