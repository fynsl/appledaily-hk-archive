【秋天野餐GO】識玩要玩埋dress code　英式復古打扮襯到絕

野餐造型其實沒有一個特定的答案，這個源於18世紀西方的戶外社交活動，人們穿上時尚的服飾，懶理是笨重又不方便走動的裙子，總之就要高貴又有氣質。近年港人喜歡戶外活動，野餐絕對是輕鬆得來不用大費周章的玩意，可是千篇一律的文青和山系造型未免太沒新意，只要花點心思配襯，其實野餐都可充滿樂趣！即使野餐dress code實無衰的復古造型也有很多種玩法，遠至四、五十年代的淑女小黑裙造型；近至七、八十年代色彩繽紛的花恤衫、連身套裝及喇叭褲。這群熱愛時裝的朋友便成立Parodise costume實行認真地玩，一切源於朋友的生日聚會，他們想到不如每次都玩特定dress code主題，為主角度身定造主題服飾，事前不嫌麻煩地襯衫、DIY甚至準備道具，原因就是為了影張靚相，玩就是要玩到盡。

通常他們會去迪欣湖野餐，因為地方比較大和空曠，這次則選擇山頂公園，因為較多花草樹木和歐陸式風格建築，很配合他們這次以1910年代為靈感的英式打扮主題。負責這次造型的Jessica本身亦是本地時裝品牌The Layers的時裝設計師，曾推出以復古為主題的時裝系列，這次終能大派用場，加上大家的私伙服飾互相配搭，途人看到都忍不住大讚「Nice costume!」。Jessica提到假如要玩dress code絕不是照辦煮碗，而是將重點元素注入造型之中，所以玩野餐主題的最大樂趣是可以自製道具或衣服，「不一定要讀過設計才做到，其實很多時要看大家有沒有花心思」，不過並不是任何主題都配合野餐，她便舉例說萬聖節野餐主題與草地未必配合，出來的效果就不夠好。

造型方面，這次共有八位朋友參與野餐派對，他們不會為了配襯造型而特地去買服飾，反而會因應自己擁有的衣服去穿搭最近似的造型，的確考起大家的配襯功架。她建議女生最好以素色為主，配襯有層次效果的裙子和有細節的白恤衫，大家的衣櫃都很容易找得到；男生則可選擇深色系衣服，例如西裝背心加西褲，整個造型頓時感覺好紳士，適逢秋涼亦可加西裝外套，配搭皮鞋便完美。為了突出造型，帽和誇張的頭飾就更加不可少。道具方面亦絕不馬虎，將碎布捲起製成花球，用布塊縫製拼色野餐布，更特別買幾支經典招紙圖案的醬汁。「其實深水埗也可以買到很便宜的布料，內地網站甚至十蚊店也是搜尋造型材料的好地方。」
