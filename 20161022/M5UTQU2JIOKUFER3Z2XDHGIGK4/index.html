Note 7大阪機場冒煙

三星智能手機Galaxy Note 7（圖）事故連連，日前更首次禍延日本，在大阪關西國際機場冒煙，幸無造成傷亡。

強行拔出電池出事

國土交通省昨表示，一名外籍人士周二下午在關西國際機場二號客運大樓出境，過關時被關員發現其手提行李中藏有Note 7。職員於是引用日本政府禁令，阻止他攜Note 7登機。該名旅客因此感不耐煩，想大力將電池從手機拔出，手機此時卻突然冒煙。關員事後當場回收肇事Note 7。
Note 7未在日本正式發售，但當局上周六起以可能引發機艙火警為由，禁止旅客攜Note 7登機，並呼籲各大航空公司在登機櫃枱貼出告示通知旅客。
Note 7自8月起在全球12個國家及地區發售後接連發生冒煙及着火事故，三星電子本月初宣佈停止生產及出售。
