天籃地北：Thank God this is the first game - 仙道彬

大勝勇士29分，最教馬刺球迷驕傲的不是在首戰已取得個人生涯新高35分的尼納特，也不是令上季開季連贏24場的勇士首戰即輸，而是後備西蒙斯（Jonathon Simmons）的大爆發，全場20分及四個籃板，還有那記令勇士迷不想回憶的把居里上籃一巴拍走，若連同比賽末段的大力入樽，搶鏡王非他莫屬。
不是說風涼話，也不是在勇士迷身傷口上灑鹽，其實馬刺打得不算太好，過多的單打獨鬥，失誤也多；幸好幾位球員的手感火熱，包括刻意拉到去零度位射三分的阿域治（LaMarcus Aldridge）也攻入26分，是有功勞，但全隊三分命中率高達50%（勇士為21.2%），要多得勇士如同倒水的防守。

為了KD，放走了內線的「擎天柱」保格，也放走了伊斯利（Festus Ezeli），令勇士變成「甜甜圈」。馬刺今仗有50分在內線取得，連同射26中23的罰球，足足73分之多，包括尼納特個人在罰球線上15射全中，還未計大量的進攻籃板。杜蘭特拚命扛起內線防守，除了兩個封阻，也多次以其長手臂盡力干擾馬刺放入內線的高吊球，可是雙拳難敵四手，尤其是馬刺雙塔加素（Pau Gasol）和阿域治都可以拉出禁區外，毫無位置感的柏治利亞形同廢人，打了20分鐘結果是兩分、三個籃板及三次失誤，就算換上大衛韋斯（David West）、麥伊（JaVale McGee）或是華利奧（Anderson Varejao），也肯定做不到之前保格的效果。
比起原班應戰的騎士，或者兩季前已做好換班準備的馬刺，算得上是大破大立的勇士，陣痛期當然更長。現時勇士的問題有二，第一，是上面提到的禁區防守問題，第二，是近兩季兩入總決賽的主力後備，包括巴保沙（Leandro Barbosa）和H班尼斯（Harrison Barnes）等都轉投其他球會，結果今仗兩軍的後備得分，是54:16，成為勝負的最大分野。
KD與浪花兄弟同時落場，前者會主力入楔，以及在禁區背籃單打，以解決勇士外線失靈時的問題，也令KD的得分機會大減；現時球隊的「第二兵團」實力不足，那由KD代替又老一歲的伊古達拉（Andre Iguodala）來出任超級第六人，也是一個好主意。在關鍵時刻才「四核合體」，勇士火力將更為完整，可減輕防守上的壓力。同日紐約人大敗，路斯說Thank God, this is the first game，只是第一場，大家不要太着緊。
