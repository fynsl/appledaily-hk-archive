【先達收風】Pixel XL美國缺貨　水貨價飈升32GB / $7,800、128GB / $8,400

Google親生仔Pixel XL水貨殺到，第一矚目點當然是出現大缺貨情況！

話說自Samsung Note 7進行回收，機王級的手機群龍無首，咁啱呢段時間，Pixel XL開賣，須知此機定位，可是幾近iPhone的高階機級數；機身規格使用了現時最頂級Snapdragon 821處理器，配4GB RAM及32/128GB ROM，顯示屏使用了大得適中的5.5吋，而且還達2K（2,560 x 1,440 / AMOLED）質素，不過最重要還是使用了最新Android 7.1系統，令用家有一種全新的體驗。所以推出首個星期，美國本土都出現大缺貨情況；而所謂的缺貨情況，是一個相對問題！基本連月費計劃的「台機」，如果大家想出機，當然問題不大；但如果你唔想食Plan，要在「Google商店」購買「齋機」的話，暫時仍處於長期斷貨情況，有點一機難求！

Android 7.1系統最大亮點，是加入一個稱為Google Assistant功能；話說使用時只要長按下方圓形虛擬按鍵，就會彈出一個對話框框，這刻開始，你就可以同手機進行語音互動。

記者曾以語音答問方法，問手機有關天氣、iPhone 7價錢等問題，甚至叫佢開啟相機功能；話說開相機就最爽利，基本一叫即開！至於天氣則相當窩心，竟然連結GPS功能，說出來的可是我當時身處旺角的氣溫；不過最鬼馬係問iPhone 7價機問題，基本不同容量的價錢，都有詳細顯示；話說記者曾作逆向測試，反過來用iPhone的Siri功能，試問出Pixel XL的價錢，可能記者與手機舖店長的英文水平較差，反覆試了十多次，Siri都係唔知我哋問乜？

老實說，Google Assistant功能真係有幾分似Siri，不過論實用性，由於Pixel XL仍未有香港行貨推出，所以暫時只能對應英語語音提問，所以使用普及性，有點不如對應廣東話的Siri！不過估計不久將來，要Google Assistant對應中文或廣東話，估計亦唔會有任何難度，希望能盡快開通吧！

價錢方面，Pixel XL有32GB及128GB兩種不同容量，美國官方定價為32GB（約港幣$5,998）、128GB（約港幣$6,778）；至於香港水貨賣街價錢則為32GB / $7,800、128GB / $8,400，好明顯有點炒起情況；不過在美國本土也大缺貨前提下，想搶先入手新機，都真係要有點捱炒的心理準備。

查詢：三禾電氣（旺角亞皆老街83號先達廣場1樓F33）
