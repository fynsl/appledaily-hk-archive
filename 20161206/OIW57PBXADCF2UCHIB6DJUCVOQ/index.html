【8K感壓平板】線條唔起狗牙　專業繪圖師：一筆即畫輕如鴻毛效果

當今電腦年代，愛畫畫的人，都續漸轉投以電腦作畫布！以往的畫法，好多人鍾愛以Notebook配搭專業繪圖畫板；不過自去年Wacom首度推出Cintiq Companion 2後，情況有點轉變，原因是將平板電腦的屏幕，直接變成專業級2,048級感壓畫板，在作畫時有更直感的操作，而且予人有如在畫紙上繪畫的筆觸！在推出一年後，新一代的MobileStudio Pro 16正式推出，它以更強大的8,192級感壓效果，令愛畫人士嘩然！

新機上手，第一件事是先比對硬件，舊版Wacom Cintiq Companion 2，不少專業繪圖師已經相當滿意，起用13.3" TFT LCD屏幕（2,560 x 1,440），配上2,048級感壓，處理器行Intel Core i7-5557U（3.1 GHz），記憶體最高對應512GB SSD配16GB RAM，開價是$19,990。

至於剛推出的Wacom MobileStudio Pro 16，機身只是稍長了一點，但卻起用了更大的15.6" LED屏幕（3,840 x 2,160），配上8,192 級壓感，處理器行上新一代的Intel Core i7-6567U（3.3 GHz），記憶體容量不變，都是以512GB SSD配16GB RAM為最高階配置，但價錢只是輕微上調至$23,990，在硬件全面升級情況下，的確令人感到很吸引。

硬件強是一回事，實際操作下又如何，今次我們找來了插畫師兼3D繪圖師——Akin Wong為大家品評新機；話說他手底下的作品不少，而且大家都可能曾見過，例如去年銅鑼灣時代廣場的Star Wars Lego展--「時代廣場《星球大戰》大型展覽第二擊」就是出自他的手筆！「其實我由Cintiq Companion 2開始，已經使用感壓式繪圖平板電腦，上一代的版本我都非常滿意，由於機身內藏一部強效的i7電腦，在運行不同的專業繪圖軟件時，已經非常順暢，所以我對新一代更加期待。」

新機上手，屏幕由13.3吋升級上15.6吋，是一個非常大的擴充感覺，最大感觸處是軟件放Tools位置見多了，而且可繪畫的範圍更大，不過機身尺寸未見大了或重了很多，只是由Cintiq Companion 2的374 x 248 x 15 mm、1.70kg，MobileStudio Pro 16 輕微上調至418 x 262 x 19mm、2.02kg，Akin甚至說拿在手上作畫，感覺分別不大。

使用上感覺分別最大位，還是將自己的舊畫作，放在新的MobileStudio Pro 16上試睇， Akin說：「舊機放大縮細一些複雜3D圖像時，往往需要較長的一段時間計算畫面，而且畫面放大後，線條會點起格的情況，原因可能是機身沒有獨立顯示卡，只是Onboard的設計吧。新機MobileStudio Pro 16開啟同一圖像時，可能都需要一點時間，但感覺快三分一至四分一時間左右，而且開出來的圖像，線條完美完全冇起狗牙，原因可能是新機加入了顯示卡的原故吧（NVIDIA Quadro M1000M 4GB GDDR5 ），所以操作上明顯快上一點點。」

另一項測試是以是用專業軟件《ZBrush》，以同一幅3D圖像，加入光影的效果，以試出新舊機在 3D演算能力的差異，最後得出結果為，舊機演算一格時間為10秒，而新機則只需要7秒，Akin說：「驟看好似相差不遠，但這只是一格片的演算時間；如果我要做一條短片，以1秒25格作計算，那麼如果在製作一條1分鐘的短片，兩部機3D演算的時間，累積相差可達1至2小時之久！」

不過說到測試重點，當然是8K的感壓效果，Akin以3D繪圖軟件試畫貓鬚：「舊版Wacom Cintiq Companion 2，就是以最輕新畫，感覺都是比較深刻；但新版的MobileStudio Pro 16，就真的能做到輕如鴻毛的感覺。」如此說法，以往是做不了如此微細的效果嗎？「以往用1K或2K筆觸時，其實都可以做到相類的效果，只要多畫幾筆，畫線後，再做平整效果，重複重叠幾次就可以了；至於8K筆觸，只需輕輕一下，就做到想要的輕力效果，但這我覺得是專業插畫師或動畫師，才可以以掌握此效果。」

專業產品太貴買唔起？國產PIPO同期亦出了一部W1s Pro平板電腦，規格低一點屏幕起用10.1" TFT LCD（1,920 x 1,200），處理器為專為平板電腦新出的Cherry trail Z8350（四核心），記憶體為64GB、4GB RAM，不過跟機一樣有感壓筆，而且屏幕還做到初階的1,024級感壓效果，記者再找來Akin作測試：「以Windows繪圖軟件作試畫，明顯見到一樣有粗幼的感壓效果，不過感應能力相對會差一點，畫直線時明顯見到有震動效果，不過老實講，它亦有一定市場，由於$1,999的機價真的偏平，就算用作上網或文書工作，也不失為一部稱職的電腦，至於繪畫功能，閒時讓小朋友作塗鴉之用，都不失為一個實用功能。」
