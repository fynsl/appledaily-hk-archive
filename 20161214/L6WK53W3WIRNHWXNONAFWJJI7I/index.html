Type R撼壆司機斷片噴火龍飛出車

Type R撼壆司機斷片
噴火龍飛出車

【本報訊】一部思域Type R私家車，昨凌晨沿長沙灣青沙公路行駛，轉右彎期間突失控撞壆，如彈珠般擦撞左右兩邊石壆，來回三次始停在快線上，車頭盡毀，司機受傷獲救送院，惟他一度失憶未能說出車禍過程，警方正調查意外原因。意外中，車上擺放的寵物小精靈的噴火龍公仔亦被拋出車外。

意外後「斷片」男子姓黃（27歲），頸及手部輕傷，由到場消防協助救出，清醒送院。昨凌晨2時許，黃駕駛今年初才購入的思域Type R私家車，沿長沙灣青沙公路往沙田方向行駛，駛近呈祥道，私家車疑高速失控先撞向右石壆，再反彈越過4條線撞慢線石壆，再反彈回4線衝前30米外快線始停下，車頭盡毀，水箱損毀一度冒煙，車身凹陷變形，泵把飛脫，安全氣袋彈出，碎片散滿地上，車廂雜物及一隻寵物小精靈噴火龍公仔亦被拋出車外，路面留下10多米長胎痕，黃受傷被困由消防協助救出，黃事後表示「記唔起發生咩事」，對意外經過茫無頭緒。
