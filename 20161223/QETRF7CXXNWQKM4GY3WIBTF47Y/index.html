【又到聖誕】迪士尼聖誕換新裝
《蘋果》直擊教你唔使排長龍

距離聖誕節只剩下三天了，在這個一年中最重要的節日裏，美國人無不抓緊機會，希望感受節慶氛圍。其中，美國的迪士尼樂園也在這段期間換上冬裝，除了超高聖誕樹之外，也把其中幾個人氣遊樂設施以及巡遊「改頭換面」，希望來到這裏的遊客能夠體驗聖誕節的迪士尼。
駐洛杉磯記者：陳志豪

像是經典設施「幽靈大宅」（Haunted Mansion）就把館內大規模改裝，換成知名導演添布頓（Tim Burton）《怪誕城之夜》（The Nightmare Before Christmas）的主題體驗。片中主角Jack Skellington以及女主角Sally都現身「幽靈大宅」，怪誕離奇的裝扮加上充滿「添布頓風」的背景音樂，讓本來小小恐怖的「幽靈大宅」化身為歡樂節日鬼屋。

另外迪士尼知名的歡樂巡遊也都搭配節日，推出聖誕版本。米奇、米妮以及其他知名卡通人物，都換上聖誕毛衣，在花車上大跳節日舞蹈。而園區內原本以墨西哥與森巴舞蹈為主題的遊行，也都加入聖誕元素，把熱帶風情跟冬日聖誕做了完美結合，參與的遊客感受到這樣的氣氛，個個笑得合不攏嘴。

在加州長大的Tanner從小就是迪士尼樂園的超級粉絲，已經來了超過十次之多。他認為迪士尼是每個人心中的夢想樂園，不論大人小孩，都可以在這裏找到自己喜歡的那一塊天堂。Tanner十分推薦大家在聖誕節期間前來，因為特別有節日的感覺；他表示：「這邊很好玩，我以前也常在聖誕節期間來玩。這時候不管是燈光、裝飾、聖誕樹、或是華麗的吊飾，都讓這裏充滿了歡樂的節日氣氛，真的會讓你體驗到聖誕節的感覺」，「我希望大家都來，這個時間（聖誕期間）很好玩，絕對值得一遊」。

不同於Tanner，來自華盛頓的Krimson則是第一次來到迪士尼的新手。對於初次體驗的她，園區裏的一切都顯得十分新鮮。能夠見到童年卡通裏的角色出現在現實生活中，她也感到十分興奮。她表示自己很喜歡看《反斗奇兵》，所以最開心的就是能夠實際看到這些動畫人物，「當然，我最喜歡的是遊樂設施就是『巴斯光年』，因為我喜歡競爭，我也喜歡射擊，當然我也喜歡贏的感覺！迪士尼太好玩了，我絕對會再回來這裏」。

不過氣氛歡樂之餘，高人氣帶來的人潮，對園區的娛樂品質也是一大考驗。許多熱門的遊樂設施像是過山車「飛越太空山」（Hyper Space Mountain）、「馬特洪峰雪橇」（Matterhorn Bobsleds）、以及「奪寶奇兵」（Indiana Jones Adventure）等，排隊一等就要一小時以上。而馬特洪峰雪橇還一度發生故障，甚至出動工作人員人力推車的情況。維修期間也讓遊客望穿秋水，所幸故障很快就修好，立刻又是長長的排隊人龍。

雖然這段期間總是人潮洶湧，來到迪士尼的遊客也依然是興致高昂。然而，有沒有方法讓我們更盡興的暢玩迪士尼呢？《蘋果》記者也特地整理出六項要點，讓讀者遊迪士尼遊的輕鬆、遊的歡樂。

1.詳細研究，做好計劃
在出發之前，先把地圖以及相關資訊研究好，可以幫助你省下很多時間。這樣一來，不但可以少走很多冤枉路，也可以充分了解巡遊的時間和地點，都讓你能夠提前做好安排。

2. 提早入園
一切都研究好了，提早到迪士尼絕對可以幫助你避開大部份的人潮。迪士尼每天的入園人數可是將近6萬人，能夠成為前20%入園的人，一定會讓你玩得比較盡興。

3. 睇App做research
知識就是力量！現在迪士尼官方提供的App中，有許多實用資訊。包含等候時間以及地圖等，都可以直接查閱，好好利用這些App，你再也不用光憑運氣傻等了！

4. 善用Fast Pass
大部份迪士尼的遊樂設施都提供遊客領取Fast Pass，《蘋果》建議讀者一入園就直奔人氣設施，攞咗先算！雖然Fast Pass有領取時間間隔的限制，但是如果安排得好，絕對可以讓你跳過長長人龍。

5. 別在意當個Single Rider
所謂的Single Rider，顧名思義就是一個人去玩這些遊樂設施。許多美國遊樂園為了有效利用空位，都會將空位塞滿，這時就是Single Rider上場之時了！記者實測人氣設施Radiator Spring Racer，發現Single Rider只需等待12分鐘，居然比抽了Fast Pass的人還更快玩到。

6. 下雨天去
12月是加州相對較多雨的時候，如果你不在意下雨天去，人潮也有可能會少了大半，只不過貼心提醒你，別忘了雨具雨衣，否則就算人少很開心，感冒了可就划不來了。
