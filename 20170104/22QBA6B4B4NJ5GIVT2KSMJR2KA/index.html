【停車場地獄與天堂】灣仔胡忠皇帝位過百呎　銅鑼灣嘉柏窄到開唔到門

香港地價高昂，車位更是奢侈品，不過有些停車場就可以如天堂般闊落，有些停車場就如地獄般狹窄。大家又有否想過，天堂與地獄的分別究竟有多大呢？

《佐敦區》

天堂：九龍站圓方（尺碼：8呎 x 15呎10吋）收費：$18/小時
九龍站圓方停車場出名闊落，通道又闊又光猛，車位尺碼有8呎闊，15呎10吋長，咩新牌仔泊車都未驚過啦，時租還要只是每小時$18，性價比極高。
地獄：佐敦寶來大廈（尺碼：7呎 x 13呎1 ）收費：6pm前$17/小時、6pm後$23/小時
同區無遠的佐敦德興街寶來大廈出名地獄，入口窄， 轉上樓的通道更窄，至於車位的尺碼原來只有七呎闊，比起圓方足足窄了一呎。至於長度就短了差不多有三呎，細人咁多仲要貴過人，只怪處於市中心馨香啲。

《旺角區》

天堂：朗豪坊（尺碼：7呎8 x 15呎9）收費：平日$14/半小時、五六日及公眾假期$17/半小時
朗豪坊是旺角區出名闊落的天堂停車場，車位尺碼其實只是輕輕細圓方車位少少，尺碼有7呎8吋闊15呎9吋長，闊只差四吋，長只短一吋，時租$28至$34，收費於這區算合理啦。
地獄：金雞商場（尺碼：6呎8 x 13呎7 ）收費：$30/小時
金雞是旺角出名的地獄停車場，第一個關口是入車用升降機，落到地牢，小車更必須上架。車架闊6呎8吋，只有小車才可進入，加上兩側有壆位，不同一般停車場以線間位，所以就位難度更高，泊車心理壓力更大。至於長度伸埋車頭出去也只有13呎7吋，足足短朗豪坊車位兩呎有多，不過收費卻跟朗豪坊差不多，可以好好體驗何謂天堂價地獄味。

《灣仔銅鑼灣區》

天堂：灣仔胡忠大廈（尺碼：9呎4 x 16呎3）收費：$15/半小時
數到終極天堂一定是灣仔胡忠大廈，最大特色是車位與車位之間有一條兩至三呎半闊的空位，這處車位比圓方還要大一呎，長度連鐵尺也不夠長，大到有9呎4吋長16呎3吋闊，可以大大力推開車門都唔怕撞。而每層皆有一個皇帝位，尺碼比現今的新樓更大，因為有成過百呎，可以做示範單位。
地獄：銅鑼灣嘉柏大廈（尺碼：6呎5 x 13呎2）收費：$25/小時
距離相隔只有1.3公里的銅鑼灣嘉柏大廈，時租平$5，不過車位尺碼就只有6呎5吋闊13呎2吋長，相比胡忠大廈，計埋車位之間的超大空隙，真係差不多細一倍。車與車之間窄到暈，所以開車門落車一定要攝出來，仲要起碼轉上四層先泊到時租，條通道又窄又難彎，可算是地獄同天堂的最大差距。
