「牛肉乾」擬加價五成打擊違泊

「牛肉乾」擬加價五成
打擊違泊

【本報訊】違例泊車問題越趨嚴重，港府擬修例增加俗稱「牛肉乾」的違例泊車定額罰款，建議將罰款額大增五成，由現時的320元及450元，分別調高至480元及680元，但因上屆立法會拉布令議案堆積，政府臨時抽走修訂建議。不過政府今年捲土重來，準備首季再將修例建議提交立法會審議。
修例未過，警方已加強執法頻率和密度。運輸及房屋局局長張炳良昨回覆立法會，去年共進行4次全港反違例泊車執法行動，對可引致交通意外或阻塞交通的違例事項重點執法。行動期間警方不作警告發出定額罰款通知書。由2016年1月至11月，警方共發出147.3萬張牛肉乾，較2015年同期上升21%，罰款收入保守估計逾4.7億元。

行車證研加QR code

未來牛肉乾會電子化，警方研究採用電子定額罰款通知書，當中包括在行車證加入二維條碼，讓執法人員以掃描二維條碼（QR code）方式讀取行車證上的車輛資料，提升前線人員發出定額罰款通知書的效率。
香港汽車會副會長李耀培質疑加價未能解決違例泊車，他指停車位、咪錶位不足導致車主冒險違泊。
此外，中環一帶違例泊車多數涉及「老闆車」，職業司機根本不會理會牛肉乾加價，「唔通叫老闆行幾條街上車？」他認為港府增加泊車位和加強巡邏執法，才是治本之道。
