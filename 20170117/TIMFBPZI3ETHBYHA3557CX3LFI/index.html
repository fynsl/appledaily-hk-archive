人事權力真空影響財政運作

人事權力真空
影響財政運作

作為國內的龍頭企業，三星集團經歷去年的Galaxy Note 7「連環爆炸事件」，公司形象已受重挫。今次李在鎔面臨被捕，不但進一步打擊三星形象，相信亦會對三星財政和運作有實質影響。

Note 7爆炸報告下周一公佈

三星在國內規模之大，令外界戲稱南韓為「三星共和國」，業務除了涵蓋幾乎所有電子產品，也包括醫院、保險、傳媒等範疇，民眾一生都難與三星完全脫離關係。不過伴隨而來的，亦是財閥壟斷、政經勾結等的指控。早前三星Note 7手機出現異常爆炸後，被迫急急召回產品，更令公司聲譽嚴重受損。三星會在下周一就Note 7爆炸發表調查報告，估計是和檢測問題和電池設計有關，但是否能挽回公眾信心已成疑問。
南韓《東亞日報》指出，在今次李在鎔面臨被捕事件後，三星未來營運可能困難重重。由於世界知名的機構投資者，一般會把涉及貪腐的企業排除在投資對象以外，美國亦規定因行賄受罰的企業不准投標，因此若果三星今次因行賄而受懲處，公司隨時被撤資及列入黑名單，對財政和各項業務也會有負面影響。
除了在品牌和資金上，人事上的權力真空也是一大影響。漢城大學經濟學教授金尚祚指：「若李在鎔和其他高層被起訴，必會阻礙三星作出投資、併購之類的大型決定，至少在年底前情況都不會好轉。」
南韓《東亞日報》/美國《華盛頓郵報》
