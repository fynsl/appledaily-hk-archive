兩男捧一女 - 左丁山

一月十四日星期六賽馬，新亞校友團體之「一片光明」出戰第十場1400米，過往「一片光明」雖然戰績穩定，出賽14次得5冠4亞2季3負，但只跑過一次千四米得第三，一般馬迷及評馬人皆認為「一片光明」難與三匹千四專家「旌暉」、「劍斷浪」、「無敵飛龍」匹敵，於是賠率十分冷落，臨場最後賠率為35倍，各校友、親友捧場，皆打定輸數，紛紛下注位置，希望可有奇蹟出現，殺入三甲，打破三匹大熱門的壟斷。
「一片光明」騎師田泰安在沙圈聽練馬師徐雨石之策騎指示，出閘後做足一百分，首先搶出，隨即留守四五位，不貼欄，取得空檔，入直路後二百米，與「劍斷浪」同時展開衝刺，力追前領的「旌暉」，而大熱門「無敵飛龍」在外如飛殺上，與「旌暉」拼鬥，結果「無敵飛龍」勝「旌暉」短馬頭，「一片光明」隨後跑第三，輸一個半馬位，餘駒更後，「劍斷浪」斷纜潰敗得第十三名，則不知是何原因了。「一片光明」的位置派彩竟然高達77元，眾校友歡聲雷動，冠亞軍組成的連贏派彩只有48元而已！
次日南華早報馬經評論第十場，指「一片光明」（New Asia Sunrise）：He does not know how to run bad races。一位政府高人（校友）看後來短訊：「此亦艱險我奮進精神也！」上一季「一片光明」初跑千四，輸給大富豪田北俊的「人民武士」，今次再跑千四，輸給大富豪洪祖杭的「無敵飛龍」，是意料中事，「人民武士」與「無敵飛龍」的來價高出「一片光明」十倍以上，贏是理所當然。
賽後在馬會會所碰到一位商界人士，非常客也，問他為何有逸興到馬場一行，他說：「忙裏偷閒，老友馬主有馬出賽，叫我入來拉頭馬，誰知落第而回，白行一場。不過也好，講馬好過講政治，上次選委選舉我輸了幾十票，真係好彩，我啲入選老友俾人call到失魂，順得哥情失嫂意，壓力好大呀！」講多幾分鐘，略知一二，原來商界對於林鄭之最高工時、全民退保、強積金對沖之立場，完全不同意，對林鄭日後施政有戒心；點知中央卡住莊曾，遲遲不批辭職，故意讓林鄭從後趕上，並在莊曾不能活動期間，兩男特首大力為林鄭造勢拉票，梁特拉攏所有梁粉過票，董特猛搵商界支持林鄭，一心本來諗住支持阿John嘅，推得到Carrie，推唔到董伯伯，難抵人情壓力，在1月16日中央宣佈同時批准兩位司長辭職之前，不少本來預備擁曾嘅人士，已被拉過去林鄭陣營。呢位商界朋友話：「兩位特首猛男力捧，女士勝望幾高。」左丁山一時口痕講多一句：「兩男一女成嬲字呀！」
