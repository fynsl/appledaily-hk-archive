壹錘定音：翻林鄭放暗箭舊賬
（李慧玲） - 李慧玲

林鄭月娥與傳媒高層閉門聚會，解釋參選是要避免選委會選出中央不任命的人，造成憲制危機。上述新聞21日晚先在網上出街，然後多份傳統報章亦大篇幅報道。而林鄭競選辦21日晚深夜則發表聲明澄清，稱林太只是一般評論，絕非針對任何人。
換言之，林鄭競選辦承認林鄭確實講過這番話，所謂絕非針對任何人，就欲蓋彌彰，心照不宣了。如果毫無針對性，何必講這樣一番話？難道林鄭會擔心長毛可以得到過半數選委支持勝出？
林鄭在閉門聚會批評團隊中人以至潛在對手，已非第一次。煲呔時代，她最常私下批評的對象是唐唐，據聞用詞之大膽直接往往令與會的政界和傳媒中人嚇一跳。
梁振英上場後，林鄭過去一年亦有兩次高調閉門放暗箭的往績。先是去年12月底，當時已表明「重新考慮」參選的林鄭出席新聞行政人員協會閉門晚宴，已經向在座數十人數落曾俊華。她以「to run or not to run 嗰位」，批評曾「無能力」、「做唔到嘢」，而她有工作能力、有民望、有商界及公務員支持，並獲中央信任。
再早一點，去年5月，她出訪四川，罕有地主動邀請多位老總同行，然後某個晚上和老總閉門談心，講了一番與會者覺得是參選政綱的施政理念之餘，更暗踩梁振英對政府運作一竅不通，諸如此類。她其後回港後，便一度傳出她與梁振英關係緊張。
坦白和敢言都是好事，但在陽光底下做的才叫坦白和敢言，才值得尊重，閉門放暗箭太小家、太不光明磊落，對被批評者不公平外，其實也自貶身價。即使最終達到預期政治目的，小家鬼祟行徑難以服眾，絕非領袖應有風範。
