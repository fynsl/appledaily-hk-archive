隔牆有耳：
田少石永泰同場論ABC - 李八方

最近香港社會圍繞跑馬仔有好熱烈嘅討論，其實要獲選委投票支持做特首，相信參選人嘅領導才能，都係一個重要嘅考慮因素，咁大家鍾意打得有為嘅上司、抑或愛和諧倡休養生息嘅波士？無論點樣，相信都有一啲係做領導嘅基本條件。

曾鈺成拒參與

公民實踐論壇特別為此舉辦「領導才能ABC」講座（雖則ABC三粒字好敏感，但據知係針對入門條件ABC，唔係為一男子而設喎～），請嚟各界人士分享心得，較矚目嘅當然係薯片叔支持者自由黨田北俊，以及字字珠璣成日有soundbite嘅前大律師公會主席石永泰，唔知呢兩位男士會畀啲乜嘢真知灼見，只知佢哋平時都風趣搞鬼，同台亮相肯定冇乜悶場。
拔萃男書院校長鄭基恩及First Code Academy創辦人辛婥琳都係講者，八方聽聞，主辦單位都有打早前甩底拒同肥彭同場嘅曾鈺成主意，但最終都係食檸檬。講座本月25日（六）舉行，費用全免，報名可到 https://goo.gl/forms/1LFr1R4y0im1GP3i2 。
