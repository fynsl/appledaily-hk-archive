二手Tesla轉手賺40萬
炒家隨時犯法

【本報訊】財政預算案公佈，4月起恢復對電動私家車徵收首次登記稅後，趕不及「尾班車」的準車主今次分分鐘「遲買貴一球」，網上更出現「炒車」的情況，約90萬的七座位Tesla MODEL X 75D賣到130萬元，轉手即賺40萬元，不過「炒車」如果沒有向政府申請及交足附加稅，分分鐘觸犯法例，最高罰款50萬元及監禁12個月。
於28car網站可見到不少Tesla放盤，當中不少都標榜於4月1日前出車，有稅前賣約90萬的Tesla MODEL X 75D賣到130萬元，一個轉手賺45%。更有賣家表示，落訂10萬元留了兩架Tesla MODEL X 75D，尋找買家開價。有賣家更講明現時稅前賣近100萬元的Tesla MODEL X 90D於4月1日後總車價將升到170萬，現放145萬元。

然而，「炒車」的行為有可能觸犯法例香港法例第330章《汽車 (首次登記稅)條例》4A。條例提到，任何人如在未獲得運輸署署長同意前，以高於前述所准許的價格出售已登記的汽車，即屬犯罪，一經定罪，可處罰款50萬元及監禁12個月。根據香港法例第330章《汽車 (首次登記稅)條例》的規定，除非事先經由運輸署署長批准，否則登記車主不得以高於汽車的應課稅價值、免稅配件的價值、免稅保養未滿期部份的價值及已繳付的首次登記稅的總和的價格出售在1994年8月1日或以後於香港首次登記的汽車。汽車出現巨額差價的情況罕見，對於車行炒賣或車主炒賣的行為會否觸犯法例，記者曾向海關查詢，但至截稿前未有回覆。
