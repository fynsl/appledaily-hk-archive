【機迷福音】Xbox將推月費制　每月$78玩齊過百「新作」

唔止聽歌，現時玩遊戲都開始慢慢過渡成月費模式，如之前遊戲公司EA就推出EA Access以月費形式玩如《FIFA》系列等熱門EA遊戲，連Microsoft都剛剛公佈將會在今個春天推出類似的服務Xbox Game Pass，可以用月費模式在Xbox One上玩到超過100款Xbox One及Xbox 360遊戲。

但當然一分錢一分貨，Xbox Game Pass上所提供的遊戲並非最新推出的大作，如官方公開的相片所見，所謂「新作」都是早一、兩年的作品如《Halo 5: Guardians》、《Gears of War: Ultimate Edition》、《Payday 2》、《NBA 2K16》及《LEGA Batman》等等，但好在並不只限於Microsoft自家作品，只要下載到Xbox One硬碟就能玩；雖然名義上唔算真正擁有這些遊戲，但只要繼續有訂閱Xbox Game Pass服務的話都可繼續玩。至於想擁有這些遊戲都得，付出原價8折就可永久擁有，但點計都唔抵，到遊戲店鋪買這些「舊作」肯定較平。香港方面未有定出Xbox Game Pass價錢，但應跟美國的9.99美元（約$78）相差不遠。
