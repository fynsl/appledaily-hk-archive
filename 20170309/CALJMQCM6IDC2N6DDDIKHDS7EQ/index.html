周氏姊妹花十足Twins
嘟嘴齊晒Fendi春夏新袋

忙拍內地劇的周麗淇最近難得返港，當然約家人聚聚，日前周汶錡、周麗淇兩姊妹雙雙在Ig po嘟嘴貼頭相，一個留言：「Great minds think alike！」，另一個說：「We are not twins, but like twins！The same face or not？Hahahahah... are not important, it is important that we use the same bag... hahahah！」

留意返，兩人同用Fendi今個春夏主打的Kan I新裝，boxy flap bag上以candy studs作袋釦裝飾，runway版本有花碌碌的girly花卉圖案、粉色條紋、波浪邊，至於周氏姊妹就齊揀stitching飾邊，簡單不失細節。
