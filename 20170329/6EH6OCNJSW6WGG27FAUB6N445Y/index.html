【1分鐘睇晒】iOS 10.3發佈　新系統隨時慳幾GB

經常嫌iPhone唔夠位嘅你，有好消息啦。iPhone今日公佈了iOS 10.3，讓用家可以即時騰出更多的儲存空間，而且不需要刪除任何訊息、照片以至影片。即時解決影相經常出現唔夠位的問題。

iOS 10.3使用了新的APFS(Apple File System)文件系統，聲稱有效提升iPhone儲存空間，更新的時間亦比以往長。記者實試後發現原本128GB 的iPhone 6s Plus 可用空間為70.47GB。更新後可用空間更增加了4GB變為74.58GB。而每部的iPhone用家安裝不同的應用程式，因此更新後騰出的可用空間也會不同。（留意應用程式及apps本身可能留有快取儲存，更新系統可能同時刪除這類檔案，實際能騰出的空間容量要視乎機種、應用程式及裝置的使用狀況而定。）

另外其餘比較實用功能，增加了尋找AirPods，使用方法如尋找iPhone一樣，有實時和最後定位功能，而且支援單邊或雙邊播放聲音，在屋企唔見左一隻，亦可以開聲找回，同時減了遺失AirPods的機會。地圖功能亦支援3D Touch，可以在地圖上顯示每小時的天氣預測。
