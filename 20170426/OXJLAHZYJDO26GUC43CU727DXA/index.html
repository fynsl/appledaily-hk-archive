【重奪榜首】本田Civic Type R　德國Nürburgring最速頭轆車

你快過我，我又再快過你，車壇就是如此。本田(Honda)Civic Type R上一代曾經是德國「綠色地獄」Nürburgring的最高圈速頭輪驅動量產車款紀錄保持者，但早約4個月前被福士(Volkswagen)Golf GTI Clubsport打破，現在，新一代Civic Type R回來報仇，重奪兵器榜排名第一位。

圈速7分43.8秒，比Golf GTI Clubsport快接近5.5秒，比上一代車款快接近7秒，絕對強勁！落場的Civic Type R和之後賣街的分別不大，只是車廂內加裝了防滾架，拆去後座及影音設備。動力部份依然是2.0公升VTEC turbo引擎，320ps最大馬力，峰值扭力400Nm，配上六前速手動波箱。據香港代理早前透露消息，這部Civic Type R亦會有行貨到港，看完這條影片後，會更期待嗎？
