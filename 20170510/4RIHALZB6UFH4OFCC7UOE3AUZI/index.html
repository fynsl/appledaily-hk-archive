半世紀白宮修復織品

於1921年美國出生，Iris Apfel既是室內設計師、時裝收藏家，更是全美古董珠寶收藏最多的人。Iris早年任職雜誌《Women's Wear Daily》，其後因為對布料的愛好，轉為從事室內設計。1950至1992年一直為白宮做織品修復，近半世紀歷經九屆總統，包括甘迺迪、列根，直到克林頓。

2005年紐約大都會博物館為Iris搞展覽，震驚時尚界，她亦從此成為最潮的明星阿婆。Iris四出開talk分享積極人生觀，拍廣告、上封面，和Happy Socks出聯乘、拍Coach、Kate Spade廣告等，90歲仍在德州做教授教書。Iris多次奪得時尚獎，2013年被《The Guardian》選為Best-Dressed over 50，2015年推出自傳紀錄片《時尚天后的繽紛人生》（Iris）。

Iris說話幽默，不妨妙語重溫

Style is attitude, attitude, attitude.　風格就是態度，態度，態度！
More is more, less is a bore.　越多越好，越少就越無聊。
Fashion you can buy, but style you possess.　你可以買到流行，但風格你要自己掌握。
I don't have any rules, because I'd only be breaking them.　我沒任何規範，因為我只會打破常規。
You only have one trip. You might as well enjoy it.　人生只有一次，所以你最好享受過程。
