【VR大熱】Microsoft VR推科幻體感手掣　LED圍繞有保護罩

影片： http://bit.ly/2qyBX0R

近年VR市場可謂大行其道，繼Samsung推出Gear VR連體感手掣後，微軟(Microsoft)近日正式公佈將會推出全新的體感手掣，配合Acer Windows VR眼罩使用，進一步提升用家的第一身體感。

官方未有公佈太大詳細資料，但從官方的片段中看到，手掣大概一個手掌的大小，分左、右兩邊使用，有圓形的觸控板以及操控桿。手掣前有一個圓形保護罩，以LED燈圍繞，配合全黑的設計，科幻感十足。手掣售價為399美元(約$3,110)，連同Acer Windows VR眼罩，但未有正式發售日期。
