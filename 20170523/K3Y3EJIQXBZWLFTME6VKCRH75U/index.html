【YouTube秘技】Chok出黑色背景　夜晚睇片無咁攰

夜晚開電腦上YouTube睇片，都可能會畀YouTube刺眼白色背景搞到雙眼好唔舒服，不過早排網上就流傳出只要在Chrome瀏覽器上加入一句程式碼，就可以「Chok」出YouTube網頁的深色主題（Dark Mode），睇落又確係無咁攰。

1/ 首先要將Chrome瀏覽器更新至最新版本，並登入YouTube帳號；
2/ 在Chrome瀏覽器中按入快捷鍵「Ctrl + Shift + I」（Windows）或「Option + Command + I」（Mac）進入開發人員畫面；
3/ 在「Console」一欄打入「document.cookie="VISITOR_INFO1_LIVE=fPQ4jCL6EiE”」再拍Enter；
4/ 關閉開發人員畫面，再重整瀏覽器；
5/ 按入畫面右上角Profile Icon，當中就可選擇開啟深色主題（Dark Mode）。
