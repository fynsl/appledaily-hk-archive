【行機到港】HTC U11短揸影相　$5,098有得玩

早前同步在台北、倫敦及紐約發佈的HTC U11終於都來到香港，6月6日就會正式推出，到底這部加入HTC Edge Sense邊框感應功能的新手機好唔好「揸」呢？HTC U11可以設定邊框感應的力度，大力細力隨你喜歡，但建議唔好設定得太細力，以免拎機時容易按錯。預設「短揸」會自動開啟影相功能，在影相程式內「短揸」就影相，「長揸」就轉換鏡頭，初步使用時都就幾就手，配合被評為最高分的手機鏡頭的拍攝效果都幾好，不過單手「短揸」拍攝時會較易晃動機身，低光拍攝時要留意。

在主介面中「長揸」就可以開啟Google Assitant，當然大家亦可以在設定入面自己調校，如果是Facebook精的話，大可以將「長揸」改成開啟Facebook。HTC U11另一特別地方在於採用流線型曲線外觀，5色水漾玻璃表面更會因應外間光源而變色，觀感幾好，不過亦容易留低指模。128GB版本型號的5色機身當中，以銀色同藍色最快推出，6月6日就會開賣，黑色同白色亦會緊隨其後，但最搶眼的紅色機款則要延到7月初先會見街，等多一個月先有得玩！

HTC U11產品資料
屏幕：5.5吋Qual HD屏幕（2,560x1,440）
處理器：Qualcomm Snapdragon 835
記憶體／儲存：4GB RAM／64GB儲存、6GB RAM／128GB儲存
鏡頭：1,200萬像素主鏡頭／1,600萬像素自拍鏡頭
操作系統：Android 7.1 with HTC Sense
電池：3,000mAh
體積：153.9 x 75.9 x 7.9mm
重量：169g
推出日期：6月6日（128GB版本）、6月內（64GB版本）
售價：$5,598（128GB版本）、$5,098（64GB版本）
