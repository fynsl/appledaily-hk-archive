總決賽筆記：
It was 2-0 last year, we lost - 仙道彬

心態，是去季騎士能夠反敗為勝的關鍵。由underdog而來，騎士一直等待勇士信心轉弱的一刻，加上輕敵及種種因素，最後從落後1:3下大逆轉，成為勇士永不磨滅的烙印。從自由市場上找來KD，不止是打法上的改變，而是心態上的改變和提昇，帶來再無退路的飢渴。
在這兩場比賽，KD也用行動證明了自己的決心，無論攻守的表現，都對得住MVP的身價，他不是處身comfort zone的隨便打打，而是主動擔起重任，守要守LBJ，攻也要用極高效率的取得高分。KD已經連續兩仗取得全場最高分，也令越來越多人討論Final MVP是否其囊中物。今場「咖哩仔」的三雙當然珍貴，但平心而論，KD才是贏波的最大功臣。
來到今日才討論騎士的部署，或者勇士的先見之明，已經沒有意思，目前騎士就如他們之前的對手塞爾特人，最少要在主場挽回面子，取得一勝；正如上季總決賽的第三戰，騎士亦成功以120:90大勝對手，才為之後的逆轉埋下伏線；要做得到，至少要有以下變招。
第一，收起JR史密夫（JR Smith）。第一仗，JR的貢獻是上陣28分鐘，得到三分，正負值是-12，防守一無是處；今仗，JR打了13分鐘，一個籃板，兩次失誤，四次犯規，個人包辦咖哩仔和KD的兩次「四分打」，除非你把他出腳疑似「楔」KD也當成貢獻吧。雖然森柏（Iman Shumpert）進攻也不穩定，至少可在防守上減低KD威力，減輕LBJ負擔。
第二，讓LBJ休息。今仗LBJ仍舊打了近40分鐘，有看比賽的朋友，都見到他第三節回到後備席時，如同激戰了120分鐘的倦容；面對勇士的車輪防守，還要單挑對手防線，所花氣力極多。LBJ一直想要的playmaker，其實就是迪朗威廉士（圖）（Deron Williams），雖然此子今日表現差勁，但他的視野和進攻能力仍在，體型上也有優勢，起碼在第二節和第三節要多用他，並且是由其發辦，由他主導，LBJ要把握機會休息，留力後段打真軍。
第三，試用前鋒D威廉士（Derrick Williams）。作為隊中籃板及內線防守主力，T湯臣（Tristan Thompson）一直是不動正選，可是這兩仗完全進退失據，以目前形勢，把謝菲遜（Richard Jefferson）推上正選，打上25-30分鐘，未必會比TT的作用差；起碼後者的防守判斷和進攻力度，會比TT更強；或者再大膽一點，起用今仗只打了不夠四分鐘的D-Will。這位前鋒起碼夠年輕，26歲有本錢和勇士鬥走，在現時的陣容上，勇士必須打得更「小」，才不會被拋離。
其實比賽只是來到半途，勇士既未在客場取勝，上季反先也有先例，怎能開心得太早？賽後卡爾教練也再次提醒隊員們不要輕敵。當勇士心理上做好準備，要贏他們，真的太難。
"Well, it's been a great run but none of that matters unless we can finish the job with this series," Kerr said. "Trust me, we know. It was 2-0 last year, we lost."
仙道彬
