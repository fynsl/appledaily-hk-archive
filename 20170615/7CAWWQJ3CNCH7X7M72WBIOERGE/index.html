【Note7再現】改名Note FE　韓國7月7日推出

不同文化對「7」字的理解都有不同，如韓國方面會認同西方「Lucky 7」講法，覺得「7」字是有幸運意思，所以最新傳出Samsung Galaxy Note 7翻新機將會於今年7月7日在韓國上市，就未必好似香港人所理解的意思。Samsung Galaxy Note 7翻新機正式名稱應會叫做Samsung Galaxy Note FE，除有傳會將內置電池換成較低電量的電池外，另外亦會加入到S8才出現的Bixby語音助理，至少比單純換電池來得有誠意。據報Samsung將會為韓國市場預留450,000部Note FE，比之前預期的300,000部為多，但預計Note FE定價約為700,000韓圜（約$4,845），以一部一年前的機種來說，價錢方面就真係唔太吸引。
