【新車發佈】CR-V七座SUV登場　神秘嘉賓Type R仲搶鏡

Honda推出全新一代CR-V，身形變大了的SUV，最大喜訊莫過於由五座變七座，對於香港車主既愛SUV又愛七人車，市場上甚少呢類型車選擇。新車只有一個版本，搭載1.5公升直四Turbo引擎，馬力輸出193匹，峯值扭力243Nm，性能上有待試車才知道。內櫳走豪華路線，以木紋飾條點綴，叫價約35萬元，實在有一定吸引力。

發佈會上突然出現另一位神秘嘉賓，就是熱血車迷一直引頸已待的Civic Type R。呢部超級火辣小鋼跑外形極之兇悍，尤其車尾的超誇張尾翼、尾擾流及中置三出死氣喉，相信車迷已經無乜可以改，搶盡傳媒目光。搭載兩公升Turbo引擎及六速手波，性能輸出320匹馬力及400Nm峯值扭力，更有+R模式，0-100km/h只需5.7秒。雖然現場未有機會試，但聽到它叫囂了幾聲， 不過沉沉叫聲未算嚇人。

至於內櫳，以紅光及紅邊點綴，液晶體錶板亦比上代靚得多，中控台更有生產編號，感覺上更矜貴。有興趣？只可惜據代理透露Civic Type R今年的配額已滿，想訂車要等之外，亦要由今年的定價55萬幾升至約58萬幾，車紅自然叫價高，無辦法啦。

Honda CR-V車價：約35萬
Civic Type R車價：約58萬
