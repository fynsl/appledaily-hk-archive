【手機唔退燒】17分鐘熱溶牛油　Note FE=翻版Note 7？

Note FE水貨正式登陸香港！有人喜歡、亦有人鬧爆！如果還記得去年Note 7停售及全球回收，或者大家都知道，其實Note FE不過是Note 7的翻新機；喜歡的人會說非常懷念Note 7的高效能，好想將Note FE入手！但亦有說Note FE由於是使用Note 7的底盤，過熱問題可能未解決！老實說，對駡沒意思，想知有否過熱問題，還是來一次正式測試，會來得準確一點。

今次可說是去年測試Note 7熱溶牛油的延續，還記得去年測試中，Note 7以17分鐘高速、比一般手機快一倍時間，將一塊10g的早餐牛油完全溶化成水狀，今次會否是另一次「神話」的延續？為一證這歷史時刻，今次還找來Samsung自信之作、說永遠不會有爆電問題的Galaxy S8+及坊間好多機迷好評、電池管理超佳的htc 10作同步測試。

這項熱溶牛油測試，舊讀者們應該好熟悉！就是先將測試手機放入防水密實袋內，機面放一塊早餐用的10g小牛油，然後以谷盡手機100%效能的《Stress CPU》軟件，令機身發熱，看看誰的「過熱問題」最嚴重，就會以最快的速度將全塊牛油溶掉。測試未開始，先看看三機的「閒置溫度」，Note FE、Galaxy S8+與htc 10的數值表現非常平均，CPU及電池溫度約在30℃間徘徊，但一開啟《Stress CPU》軟件，Note FE及S8+的CPU溫度馬上直標上70℃！相對電池溫度未見有大升溫，大概只有28℃左右；相對htc 10的電池及CPU溫度兩組數值好接近，都只是33℃。

測試去到8分鐘，Note FE與S8+的CPU溫度仍然高踞不下（72℃至75℃），這可能與它們使用了較高配置的硬件效能不無關係，不過電池熱度還維持在32℃至33℃水平；相反htc 10兩組數字溫度，都已接近40℃。同步另一注意點，是Note FE的牛油在這一刻開始有溶化迹象。
去到14分鐘，htc 10的CPU及電池溫度，都是非常平均的42℃，牛油首度出現溶化迹象！再看Note FE的CPU溫度已急升上78℃，但電池溫度還是36℃的較低水平，好像對電池防過熱問題下了一定苦功！再看S8+，可能使用了較新手機設計，CPU溫度竟然由先前的72℃，下降至57℃！而且電池溫度在這刻是三部機最低的35℃。

直至測試去到17分鐘，Note FE以大熱姿態，首先溶掉了所有牛油，成為三機中最熱的一部；至於htc 10，亦在21分鐘溶掉所有牛油，成為第二名的最熱手機！當中防熱效能最高者是Galaxy S8+，在兩機分別溶掉所有牛油情況下，S8+機上的牛油，大概只溶掉了一半，估計三機中以它的防熱效能最佳！如果需要賽後檢討，首先重點會是在Note FE身上，17分鐘溶掉所有牛油，完全是去年Note 7賽果的翻版，所以說Note FE是翻新機？我是完全相信的！儘管CPU的熱度，在Note FE身上沒有明顯的改善，但另一留意點是電池的熱度，肥通發現全程都維持在30℃至37℃的水平，這方面是令人滿意的，明顯「熱到爆電」的問題，已經作出了針對性的改良！

至於HTC10方面，不論CPU及電池的熱度，都可以用穩步上揚來形容，最後在21分鐘溶掉牛油時，CPU及電池可是達到44℃的高溫！不過以密封環境下得出此數據，都算是令人可接受了。

說到最驚喜，還是S8+吧！當中最大轉捩點，可是14分鐘時，由於機身過熱，CPU竟然自動調節溫度，由72℃下降至57℃，第一點是看出Samsung在過熱保險機制上下過苦功，就算手機一直處於高效能情況下，仍能做到降溫效果，這點值得一讚；更重要是電池溫度全程都保持在30℃至36℃，防止過熱的能力的確相當高，估計Note 7給他們的教訓真是頗大，而且對電池的處理，這部手機是相當認真的。
