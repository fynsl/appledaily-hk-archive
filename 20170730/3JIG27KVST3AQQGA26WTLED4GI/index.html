【邊玩邊跑】熱辣辣五大主題跑　變身Barbie超人跑5k

跑步熱潮一浪接一浪，下半年開始將會有多個主題跑。鍾意跑步又鍾意熱鬧的話，就不要錯過了。

迪士尼樂園將於九月舉辦「Marvel 10K Weekend 2017」 ，最大亮點是一班超級英雄會現身帶領賽事，活動共分為4個組別：「美國隊長10公里跑」、「雷神托爾5公里跑」、「蜘蛛俠3公里跑」及「兒童跑」，價錢由$250至$500。

Marvel 10K Weekend 2017
日期：2017年9月16日至17日
地址：香港迪士尼樂園

Barbie Run跑過台北和高雄，今年跑到來香港！參賽者將穿上粉紅衫跑出一片粉紅海，賽事分別有1.5公里和3公里的路程供跑手選擇。鍾意Barbie同時可以做善事，活動將會捐款支持香港遺傳性乳癌家族資料庫，門票將於下星期公開發售，價錢有待公佈。

Barbie Run
日期：2017年9月24日
地址：中環海濱活動空間

全港首個跑步結合瑜珈的大型運動，由新加坡登陸香港！賽事分兩部分，首先是5公里（非競賽）與10公里（競賽）跑步；跑步結束後，隨即於現場舉辦大型戶外瑜伽班。適合愛好運動，鍾意挑戰的朋友。

Yolo Run HK 2017
日期：2017年10月1日
地址：沙田香港科學園

超人打怪獸由細睇到大，今次Ultraman Run有機會見到7個超人真身，仲有香港特別版超人Figure和限量商品，相信粉絲們唔跑都想去睇睇。活動分為5公里跑及繽紛跑，報名費統一為每人$380。

Ultraman Run
日期：2017年10月2日
地址：沙田科學園

最後就有Beer Run，除了跑步之外，亦有美酒美食、啤酒瑜珈、戶外音樂會、攤位遊戲及工作坊。其實更似大型嘉年華，可以一邊玩一邊跑步，不過飲酒後還是輕鬆跑，量力而為就好了。賽事報名費為$348，齋玩唔跑入場只需$50。

Beer Run
日期：2017年11月4日
地址：啟德郵輪碼頭 對出空地
