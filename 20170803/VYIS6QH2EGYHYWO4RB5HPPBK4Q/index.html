亞馬遜跪低　促中國用戶停翻牆

想進入中國市場，必須服從中國規矩！繼蘋果將中國區應用商店的翻牆app下架，另一家美國科技公司亞馬遜，也屈從中國的網絡審查，其網路服務Amazon Web Services（簡稱AWS　圖）中國用戶日前收通知，要求他們停止「翻牆」。

蘋果翻牆app下架

亞馬遜公司在中國網路服務運營商北京光環新網技術公司，上周五（7月28日）和本周一（7月31日）兩度向客戶發郵件，要求停止使用翻牆工具。光環新網客服人員王女士表示，通知是依中國公安部和電信監管機構最新指示：「如用戶不按指導行事，公司提供服務和用戶的網站可能會被關閉……作為運營商，我們也要定期檢查用戶是否使用這些軟體或儲存非法內容。」
隨中共召開十九大，當局7月開始清理、刪除、下架翻牆軟體，並向外國科技公司施壓。蘋果日前已從其中國區應用商店下架了一批VPN、SS技術翻牆app。亞馬遜的AWS雲服務，允許客戶在中國租用服務器來運行網站或其他在線服務，可衍生許多繞過國家防火牆的網絡行為和工具，因此也成整肅對象。
亞馬遜暫未有回應被指向中國網絡審查低頭的舉措。據悉，從去年起AWS中國區域服務已由光環新網運營全權營運，以符合中國電訊法中，必須有中國合作夥伴運營的規定。
