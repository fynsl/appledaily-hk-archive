【幫你慳錢】親子遊不一定住酒店　暑假限定租屋優惠

暑假是親子外遊旺季，酒店價格不便宜，更經常爆滿。計我話，親子遊不一定要住酒店，租間附設泳池與花園的獨立屋，更適合活潑小孩。

早前記者到越南熱門旅遊城市峴港，利用Expedia旗下的HomeAway網上平台租房，便覓得價錢相宜、位於市中心的三層獨立屋，每晚約$3500租逾6000呎單位，合共六間睡房，可容納16人；屋內有花園、露台、泳池、開放式廚房，並設各式娛樂設備如足球機、桌球，適合一家大細甚至幾個家庭渡假。比較另一租屋網站airbnb，峴港設獨立睡房並可容納16人的獨立屋，價錢貴近一倍。

HomeAway租盤涵蓋190個國家，家長可趁暑假完結前（8月31日）瀏覽網站（www.homeaway.hk），結單時輸入promotion code 「summer17」，可享有8%訂房優惠。
