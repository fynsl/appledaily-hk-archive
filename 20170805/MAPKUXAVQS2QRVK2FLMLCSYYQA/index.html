【iPhone 8消息】包裝盒有相睇？ 傳叫iPhone X Edition

好多用家都等緊iPhone 8推出，但對於這部新機王名稱就有不同傳聞，除iPhone 8外，還有傳叫iPhone X或iPhone Edition；不過剛剛有自我描述為Foxconn Insider的Twitter用家貼出一張據稱是新iPhone包裝盒相片，但盒上印上的是「iPhone X Edition」，難道這才是真正名稱？但當相片曝光後，一眾外國網站都對相片表示質疑，如BGR就指這位貼出包裝盒相片的Twitter用家Nick Chen（綽號@foxconn_insider），只是剛在8月才加入Twitter，並沒有往績支持；而且從盒上所見，iPhone X Edition的開關掣跟傳聞中較長設計不同；再加上相片奇怪只得「iPhone X Edition」這幾個字清晰，其餘盒面部份就模糊一片，難令人信服。到底iPhone X Edition名稱是否屬實要留待Apple在發佈會上公佈，不過名字太長並非Apple一向作風，如之前的細芒iPhone都只簡潔地命名做iPhone SE。

資料來源：BGR、Twitter
