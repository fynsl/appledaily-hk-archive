【新機速報】新牌筆電Avita　玩多色+超大觸控板

好一段時間沒有聽聞過有新電腦品牌打入香港市場，成立不久的本地初創公司Nexstgo就剛引入美國品牌Avita手提電腦，首個見面禮是Liber系列。Liber系列設有規格較低而且只有銀色及灰色選擇的標準版，但較吸引明顯會是設有超多款顏色機身顏色選擇，而且硬件規格會較高的Liber Glamour Collection，初出時會提供金、粉紅、黑、珍珠白及紅共5種顏式，繼後還會推出如橙、黃及紫等多種鮮艷色調，以手提電腦產品來說確實少見。

至於手提電腦設計都確實不錯，機身配備13.3吋或14吋屏幕，重量亦不過1.37kg或1.49kg，不算重，而且造工都算紮實；屏幕部份更可翻開180度，適合不同使用需要。另一特別設計是機身配備面積非常大的觸控板，初時還以為打字時容易誤觸觸控板，但實試時手枕位剛好會停在觸控板旁邊，並沒有出現誤觸這個問題。Liber系列都設有不同的硬件配搭，廠方暫時都未有列出所有配搭的價錢，以配備Intel Core M3處理器、13.3吋屏幕、4GB記憶體及128GB SSD的標準版，價錢會是$5,990；如若選配Intel Core i5或i7處理器，又或是更高容量SSD，價錢自然較高。廠方預計會在9月正式推出Avita Liber系列，如Glamour Collection會先推出5款顏色機身；如想早一步買到Liber手提電腦都有辦法，廠方將會在8月25日開幕的會展電腦節上以$825（約一四折）推出只限10部Liber手提電腦，有興趣的朋友到時記得留意。

Avita Liber Glamour Collection規格
處理器：Intel Core M3、i5、i7
顯示屏：13.3吋或14吋IPS屏幕（解像度：1,920×1,080）
記憶體：4GB／8GB
儲存：128GB SSD／256GB SSD／512GB SSD
體積：315×212×15mm（13.3吋）／333×222×16mm（14吋）
重量：1.37kg（13.3吋）／1.49kg（14吋）
售價：$5,990起
