【香港幾時有？】Xbox One X特別版即日有得訂　靠《Forza 7》護航

影片：http://bit.ly/2ijtI5n

Microsoft部新4K遊戲主機Xbox One X將會於11月7日開始在美國、英國及歐洲率先推出，所以選擇在德國Gamescom遊戲展上公佈新主機即時開始預售，都算合情合理。玩家預訂有甚麼着數？機價就無得平，美國售價鐵定499.99美元（約$3,912），歐洲價錢更達499.99歐元（約$4,600），不過好在有得訂Xbox One X Project Scorpio Edition特別版，在手掣以及機身上會印有一行綠色字體「Project Scorpio」，感覺上又好似矜貴啲。

Xbox One X主機性能非常強，不過到底當主機在11月推出時會有幾多遊戲可以用到咁強大的性能？原本預計可以拍住Xbox One X一齊上的《Crackdown 3》就要延期到2018年第1季，目前就只剩低一隻《Forza Motorsport 7》算是大作，而且這款賽車遊戲10月頭已推出，單看牌面並唔容易吸引到Xbox One或Xbox One S用家升級。至於香港方面幾時會推出Xbox One X？從香港Xbox網頁都可見到預訂Xbox One X Project Scorpio Edition特別版的頁目，但一按「立即預購」只會彈出另一個顯示「即將推出」的頁面。
