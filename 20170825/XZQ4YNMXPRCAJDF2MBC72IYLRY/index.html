【加強監控】
馬販搲撈冇王管？華人馬主認為註冊制度......

過去不少馬主因欠缺門路，買馬靠朋友介紹，或道聽塗說，而且不少經手人毫無相關知識，僅為賺取佣金而草草促成交易。

雖云馬匹能力從無保證，但經手人遠離水準令買家造成損失，例子可謂比比皆是。

就連引入上季冠軍一哩賽盟主「詠彩繽紛」的金牌馬販大衛派斯（David Price），也無法做到「面面俱圓」。

馬會上季起正式收集有關資料，並編訂《馬販實務守則》（Bloodstock Code of Practice），經同意下公開已註冊的馬販資料，讓準買家從中了解，以盡量保障自身利益。

有白紙黑字的紀錄之下，起碼可以減免馬販們的角色衝突，皆因現時註冊的60位馬販有現任練馬師、前馬會職員等，訂立《馬販實務守則》某程度上可做到去蕪存青。

對於馬會透過政策實施，對業界馬販進行規管。「機緣再現」馬主李偉然有以下看法：「我的馬大多自行買入，所以相信沒有註冊馬販，我的情況也可如常操作。馬販們做生意，名譽攸關，自會保持良好操守，至於馬會此舉，我估計是萬一馬主與馬販有爭拗，登記時簽訂文件，將來便可多一項證據跟進處理。」
