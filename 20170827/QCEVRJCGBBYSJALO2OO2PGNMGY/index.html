【Fami通】自護軍團軟綿綿手枕　OL返工用咗都話好好瞓

BANDAI 今天公布了三種以自護軍機動戰士款式設計的手枕，目前已經開始接受預訂，喜歡可愛 (？) 精品的朋友不容錯過。

說起「機動戰士」第一時間想起的即使不是高達和渣古這類主角級機體，大概也是其他著名 MS 例如大魔、太空坦克、勇士之類吧，但這次 BANDAI 推出這套手枕的造型卻來自三部超冷門機動戰士，這樣的扒逆水戰略真的沒問題嗎？

這次推出一共三款手枕依據自護軍的三部 MS 為藍本，分別是「Agguguy 亞古加爾」、「Juaggu 隼亞古」及「Agg 亞古」。它們是由著名的 MSM-04 Acguy ( 亞凱 / 俗稱：龜霸 ) 發展而成的間諜工作用機體，可惜它們在動畫中登場的機會不多，只曾經在查布羅攻略戰時比較活躍，但因為造型獨特的關係，人氣雖然接近零但出乎意料地對它們有象印的觀眾也不少呢……
