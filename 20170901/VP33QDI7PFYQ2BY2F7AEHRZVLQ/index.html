【山寨大國】國產iPhone X現身　抄到足行Android

影片： http://bit.ly/2vuoQk0

萬眾期待的iPhone X Edition 9月12日才正式發佈，但網上已出現可以使用的山寨iPhone X，山寨廠的執行力及速度實在驚人。iPhone X Edition應該是歷代iPhone中保密工作做得最差的一代，幾個月前就已經有機板流出，其後系統介面、操作方法相繼流出，更有大量零件在深圳手機市場出現，真的只差iOS就可以組裝出一台iPhone X Edition。

有山寨廠綜合眾多網上流傳的iPhone X Edition資料，搶先Apple發佈會，推出一台可以使用的iPhone X Edition。片段可見，其外形、系統介面都抄到十足，連虛擬Home鍵都有，不過系統就只是基於Android。另外，iPhone X Edition的重點異形芒也不見蹤影，畢竟目前異形芒的成本及技術之高，一般廠商根本不能應付。雖然新iPhone尚未正式發佈，但應該和此山寨機相差不遠了。
