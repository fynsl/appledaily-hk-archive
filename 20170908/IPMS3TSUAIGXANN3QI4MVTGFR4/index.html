【新iPhone情報】傳採用隱藏式Touch ID　外媒報道指未成熟下年才量產

很早之前就傳新iPhone將會使用隱藏式Touch ID，但目前隱藏式指紋辨識技術似乎尚未成熟，Samsung兩款旗艦S8及Note 8都妥協將指紋辨識移到機背。Apple會否掌握更成熟的技術，還是真的會以Face ID取代Touch ID，真的只能等Tim Cook揭曉。

高通物聯網業務高級副總裁Raj Talluri接受外媒訪問時表示，將指紋感應器放到屏幕底下並不易，他們還需要時間完成開發及設計，但他有信心隱藏式指紋辨識技術在明年能正式投入使用，且大部分的旗艦手機都會使用此技術。

高通早於2015年就發佈了其Sense ID 3D超聲波指紋辨識技術，今年6月，高通在上海MWC發佈新一代的超聲波指紋辨識技術，技術已經接近成熟。與傳統電容式指紋辨識相比，超聲波抗污、抗汗干擾的能力更強，在水中也可以順利解鎖。

隱藏式指紋辨識技術成熟後，指紋感應器再不需要設置在奇怪的地方（S8及Note 8），手機設計將更完美。
