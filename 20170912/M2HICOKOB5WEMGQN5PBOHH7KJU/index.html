【汽車籽】交車等到2018　熱血最強日系掀背車

【汽車籽：新車落地】
曾去過一屆東京車展，紀念品店有售以各間車廠廠徽作主題的logo tee，唯一售罄的一款是印上本田（Honda）廠徽的，由此可見，本田擁躉數量應該大幅拋離同鄉對手。本田如此受歡迎，因為它形象sporty，積極參與賽車運動，而且一直有推出高性能型號，例如掀背車Civic Type R便是眾所周知的「平民跑車之王」。

自從1997年推出第一代後，Civic Type R便成為平民跑車中的風頭躉，而且越出越精密，甚至有成為獨立車系的趨勢。第五代Civic Type R（廠方編號FK8）於去年先以概念車身份在巴黎車展首度現身，然後於今年正式開賣。小部份配額好快抵港，代理表示約80部2017年現貨已經售罄，現在走入陳列室訂車，sorry，要等到2018年才可以交車，而且車價已經上調了三萬大元。
紅底廠徽是本田Type R型號的最重要記認，這一抹紅更由外再到內，由車身外觀到車廂內櫳都盡量「染紅」，務求要將熱血形象貫徹到底，新一代Civic Type R緊守這傳統，輪圈、擾流組件、賽車化桶形座位、軚環、控制台、引擎頂護蓋……都滲入紅色細節，無所不用其極地提醒車迷：我仍然一腔熱血。

提升車架硬淨度　疾走動態變穩

前三代的Civic Type R都是沿用自然吸氣引擎，但自上一代開始，廠方轉用加入渦輪增壓（turbo）技術的動力心臟，這是為了配合全球日益嚴格的廢氣排放及低耗油要求，在保持強勁動力輸出的情況下，這可說是兩全其美的做法。這一代的引擎最大馬力及扭力，維持在310匹及40.8kgm，表面上看似乎沒有明顯提升，不過由於車身輕了16kg，廠方亦改善了排氣系統，故此加速效果變得更加爽快，而且高性能車重視的排氣喉聲調亦更動聽、更響亮、更有跑味。
動力輸出的提升不算明顯，不過操控感的確更接近完美，全因廠方着力提升車架硬淨度及行駛動態。廠方表示，新Civic Type R的車架硬淨程度大幅上調了38%，而且尾懸掛系統改用更高級的多連桿（multi-link）結構，過彎時，車身動態明顯更平穩。另一方面，車身與前排座位的高度同時下調，這令駕駛者以至車內人感覺到乘坐感更加貼地，進一步放大駕駛跑車的操控樂趣。

自動補油功能　瞬間變轉波高手

相對以往同系列型號予人「飛仔車」的觀感，新一代Civic Type R已經變得「文明」，因為廠方刻意加入多項功能，令駕駛者及乘客無論揸車或坐車，一樣感到輕鬆舒服，例如新增的comfort行車模式，能夠提升日常駕駛時的舒適程度，不再一味硬崩崩戰鬥格；首次出現在Civic Type R的電子式泊車手掣，附設自動啟動及鬆開功能，也能令日常駕駛更舒服。最窩心的是自動補油功能，由於Civic Type R是高性能手波車，要將車開得好、開得順，駕駛者最好識得heel-and-toe轉波技術，有了這項新功能後，系統會自動補給畀油，以確保轉波過程流暢，即使手波車操控技術未夠班，依然可以瞬間變高手。
