【科技籽】高達迷東京台場朝聖　2000模型任你買

【科技籽：玩物達人】
近日應該很多高達迷去日本東京台場朝聖，因為1比1獨角獸高達Unicorn Gundam終於在9月24日正式展出，不過其實除了樓下睇實物大獨角獸高達外，記得要上7樓同樣新開的The Gundam Base Tokyo，因為絕對不只是強迫消費的模型舖，更堪稱係Bandai高達模型的展覽中心！如果是高達迷記得預留多點時間，亦建議先安頓好未必愛高達的旅伴，否則你全程只會聽到：「走未呀？」「盒盒咪一樣？」「你又買呀？」「紅色嗰隻高達你咪有囉！」嗰隻係渣古呀！

首先有個貼士給未去的fans，如果想周六、日去，建議未睇獨角獸前先上7樓攞籌，因為The Gundam Base Tokyo會限制不同時段的入場人數，筆者當天約下午1點左右去到，要攞籌至下午3點半才可入場，當然當日獨角獸開展正是人流高峯期，不過為節省時間，先攞籌再落去睇高達，之後吃個飯或下午茶再上去，時間剛剛好。

全球最齊款　歷代高達模型列陣

The Gundam Base Tokyo，其實前身就是Gundam Front Tokyo，當年都有Gundam胸像、核戰機及天幕影院等，不過早前3月份1比1 RX-78-2巨像被撤走關閉，就是為了變身成這間The Gundam Base Tokyo。這間店舖位於巨像後的商場Diver City Tokyo Plaza的7樓，商場其餘層數主要銷售不同品牌服裝，也有餐廳及food court，絕對可以拖延女朋友，為自己和高達爭取多一點相處時間。
一進入The Gundam Base Tokyo，率先會看到歷代高達模型列陣，還有超多名人製作的模型作品，如有《機動戰士Gundam》主角阿寶的聲優古谷徹，還有SKE48、本鄉奏多等藝人作品。再來就是派發限量產品購買列表，心急的可以即時填好鎖定入手目標先。模型店有超過2,000款模型商品，而且模型架上方就展示超過1,500件完成品，不只是最新產品，最正是由1980年7月開始發售的模型也會有，有得睇有得買，老鬼睇到眼濕濕呀！

迷你靜岡工場　即場任砌60分鐘

之後看到展示區還有一隻1/20發光獨角獸高達，不過既然你等陣都落樓下睇1比1咯，呢隻可以望一望算。除了之前看過的藝人作品，還有比賽得獎作品，就是GBWC（GUNPLA BUILDERS WORLD CUP）世界大賽作品，連獎座都可以看到。買完模型，還可以參觀有專人示範教學的噴油區域，或去到Builders Zone，有枱有凳有工具，可以即場砌模型，繁忙時間限時60分鐘。
最後係筆者覺得最正的是Factory Zone，內裏有展出製作模型的機器，還有職員即場講解，亦可看到模型的每個製作過程，包括設計圖、說明書、模件，甚至係包裝盒嘅設計都有提到。當中最好看是一個1/144比例的靜岡「萬代模型中心」模型，即是日本生產模型的工廠，當中的生產線、寫字樓也有製作出來，而且有活動部份，按不同區域按鍵就會有發光或活動等反應。其實會去看獨角獸，都一定會上7樓參觀，不過我想說很多人包括我，參觀前以為只是「甜品」的The Gundam Base Tokyo，其實有主菜級的份量！
