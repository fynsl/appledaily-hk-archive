C AllStar邀栢芝捧場

C AllStar前晚在九龍灣為紅館演唱會綵排，四子即場唱新歌。演唱會將於後天舉行，成員均指心情不太緊張，On仔說：「開頭緊張啲，後來好多嘢落實咗就輕鬆好多。」釗峰則說：「第一次成功爭取到喺紅館有升降台，個台夠我哋四個人企之餘，仲可以行多兩步。」之前曾表明完騷後，四人會分開發展，問到會否在演唱會上作告別？他們說：「咁又冇喎，暫時冇咁諗過。」
嘉賓方面，C AllStar謂要保持神秘，兩場會有不同嘉賓，其中一位曾經合作。問到釗峰會否邀請張栢芝捧場？他說：「我有同佢講，唔知到時嚟唔嚟，其實仲有好多圈中朋友，唔逐一數，怕數漏。」
採訪、攝影：曹家誠
