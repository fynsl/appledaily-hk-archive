【遊器王】實試三款防盜背囊　防水防鎅有埋USB接口

全新旅遊專頁，即like籽想旅行：https://fb.com/travel.appleseed



不少人去旅行都會背着背囊到處走，但最怕遇上不法之徒，市面上不少防盜背囊，有些聲稱有防水防鎅功能，今天我們找來三款防盜背囊，分別是Bobby多功能防盜背包（a）、CoolBell防盜雙肩後背包（b）和Alpaka shift 防水多功能背包（c），試試到底有多厲害。從表面看，三款背囊的拉鍊位都是置於裏面，都不能從外面把拉鍊打開，隱藏功能都不錯。

測試一：試防水
三款背囊的物料都會跣水，淋水後水珠會自動滑走，所以我們再在拉鏈位倒水，看看背囊有沒有入水。打開背囊，發現b和c都完全沒有入水，但a就看到裏面已入水，雖然入水情況不算嚴重，但都有數滴。

測試二：試防鎅
找來男同事試鎅三款背囊，得出同樣結果，輕力鎅沒破甚至沒花痕，但當一用力鎅就會穿。雖然鎅得爛，但男同事認為被不法之徒用力鎅背囊，當事人應該會感覺到，所以輕力鎅不穿夠實用。

測試三：鬥功能
背囊a和b都有外置USB充電接口設於左側旁，不用拿着尿袋都可輕鬆充電，如果連拿着電話都不想，更可把電話放入側旁暗格。雖然背囊c沒USB接口，但勝在夠硬淨，底部可放置相機等器材，背囊帶的暗格設有拉鏈，非常安全。相反，背囊a和b雖然背囊帶都設有暗格，但沒拉鏈，不建議放硬幣或鎖匙等東西。

測試四：鬥重量和容量
背囊a最輕身，尺寸42 x 34 x 16cm，重0.85 kg，比較適合女性。背囊b尺寸為46 x 31 x 14cm，重0.92 kg。背囊c尺寸42 x 28 x 17cm，重1.5 kg，淨重1.5kg對女性來說，有一點吃力，男性使用會較理想。

測試五：鬥性價比
背囊a售$548，背囊b售$368，完全對得起價錢，a更附有保護罩和環保袋；背囊c較型格，容量較大，但售$1799，選擇那個就視乎你是外貌協會成員，還是性價比專員了。
