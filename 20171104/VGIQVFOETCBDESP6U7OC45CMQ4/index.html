iPhone X變果然燶！
果迷：同幻想差好遠

紀念iPhone面世十周年的iPhone X昨晨開售。在銅鑼灣Apple Store門外持續有約100人聚集，等候取機和收機，食環署派人在街上巡邏，以防「收機黨」在行人路擺檔阻街。

iPhone X 64GB的官方售價為8,588元；256GB 則為9,888元，提供銀色和太空灰。Apple Store昨晨提前至早上8時供在iReserve預約成功者取機。到昨日早上10時，Apple Store 正式開門供公眾入內，蘋果職員一邊拍手歡呼，一邊歡迎「果迷」入場取機。截至昨日早上10時半，iPhone X 256GB市面收購價為10,500元，64GB則約8,800元，惟價格持續波動。

高先生成功預約2部太空灰iPhone X 256GB，昨晨9時半與太太到銅鑼灣取機。他稱自己都有意換手機，但希望在首輪先賺一轉，現在候價待沽。但他取機後聽聞收機價約只有1.1萬元，他稱若價錢再無上升都會選擇放機。他承認，早前曾希望收機價可達3萬元，「但現實和幻想始終差好遠。」他認為是炒家唱高價錢以吸引果農入貨，令現在貨量充足就壓低收購價。

內地旅客李女士則在店外向朋友大喊：「給我拿兩台，我趕著上飛機。」她向記者說，想為自己和朋友買3部iPhone X 256GB自用，但到埗才發現原來要事先經網上預訂，所以只能在店外湊熱鬧。她聲稱自己不懂廣東話，不敢自行向收機客問價，又指自己或回內地官網買機。有收機黨即場放機止賺，一名不願透露姓名的男子稱，特意替上司到現場購機自用，每部iPhone X 256GB購入價僅11,300元，只較官方價多千餘元。

泰國遊客Sutkhet特意來港買iPhone X。他指朋友之前iPhone 8在港開售時都有替朋友代售和轉售，今次特意請假兩天來港購買iPhone X，自己買入2部，同行朋友另買4部，每部iPhone X 256GB購入價為1.1萬元，並計劃即晚返回泰國。他指，雖然不清楚泰國的官方售價，但香港較泰國早個多月發售，可以「早買早享受」，認為價格合理，亦值得來港。

港人陳先生亦到場幫老婆買新機。他說之前都無上網預訂，覺得網上買手續繁複，倒不如今日到場買機更直接。他同樣以11000元購得iPhone X 256GB一部，認為價錢合理。

「極高價收機」、「超高價收機」、「嚴重高價收機」，更震憾的口號，都離不開市場的現實。來到放機集中地、旺角的先達廣場，到場放機的人流疏落。iPhone X 64GB的官方售價為8,588元；256GB則為9,888元，惟部分手機店的64GB回收價，在下午2時前已跌穿官方售價，256GB回收價亦僅較官方售價高300至1,000元。

而截至昨日下午2時，現場舖位開價iPhone X 256GB太空灰回收價為10,700元（賺812元），銀色為10,200元（賺312元）；64GB則8,800元（賺212元）。至於已開盒的上台機，256GB回收價約1萬元（賺約100元），64GB為8,350元（蝕238元）。

有手機店則將256GB機款定於10,850元（賺962元）回收，64GB則為8,100元（蝕488元）。店主稱，回收價較昨日早上低位回升約300元，亦預計下班時間會將再有一批機供應，機價或會再受壓。
