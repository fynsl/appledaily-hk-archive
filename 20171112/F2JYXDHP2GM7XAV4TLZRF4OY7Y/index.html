美駐俄使館聘前KGB負責保安

美國國務院早前聘用一間私人公司為駐俄羅斯莫斯科大使館進行保安工作，但原來該公司是由前蘇聯國家安全委員會（KGB）的特工成立，該特工曾跟俄羅斯總統普京共事。
該公司名為Elite Security Holdings，由前KGB特工布達諾夫（Viktor Budanov）創立，現由其子管理。布達諾夫與英國著名蘇聯雙重間諜金．費爾比（Kim Philby）相熟，1980年代曾與普京一同在東德執行任務。
美國前總統奧巴馬為回應俄羅斯涉嫌干預大選，去年底驅逐俄羅斯外交人員，俄方還以顏色限制美國外交相關職位數量，包括4間駐俄使館保安人員。

稱只有該公司合要求

美方因而被迫聘請私人保安公司提供護衞員，在審視各公司規模後，發現只有Elite Security Holdings符合要求，合約總值280萬美元（2,184萬港元）。
另方面，調查「通俄門」的特別檢察官米勒團隊據報已向總統特朗普前競選團隊共同主席克洛維斯（Sam Clovis）問話，調查特朗普是否知悉親信下屬接觸俄羅斯人員之事。
