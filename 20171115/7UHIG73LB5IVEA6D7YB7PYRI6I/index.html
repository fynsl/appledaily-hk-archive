越南專家製千元面具破解iPhone、Face ID

越南專家製千元面具
破解iPhone、Face ID

蘋果iPhone X本月初正式開賣，但越南有網絡安全專家很快就研製出可破解該手機Face ID容貌辨識解鎖功能的面具。

河內Bkav公司的研究員，利用3D打印機打印出面具的輪廓，「面皮」以手工製，用2D圖為「眼」，「鼻」用矽膠製，「皮膚」面積較大的「面頰」與「額」經特別處理。他們之後用製成品進行測試，幾秒就騙過Face ID，將iPhone X解鎖。

製作一個可破解Face ID的面具需時六天，價錢約1,165港元。

Bkav網絡安全總監吳俊英（音）指出，Face ID技術並未成熟，而他們的製面具技術可能被黑客盜用，富豪政要可能成為針對目標，呼籲聯邦調查局等部門多了解Face ID。

英國《每日郵報》
