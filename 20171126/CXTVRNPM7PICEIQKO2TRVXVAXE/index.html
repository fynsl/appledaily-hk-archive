《Let It Go》被指抄襲男歌手告迪士尼

《Let It Go》被指抄襲
男歌手告迪士尼

迪士尼動畫《魔雪奇緣》（Frozen）歌曲《Let It Go》於四年前推出時風靡全球，無論動畫同歌曲都大受歡迎，近日一位男歌手Jaime Ciero就入稟指控《Let It Go》涉抄襲其於2008年推出嘅西班牙文歌曲《Volar》。

Jaime Ciero話兩首作品結構相似，而其作《Volar》都曾登上音樂流行榜，喺樂壇都極具人氣，佢除咗告迪士尼，仲控告演繹電影版《Let It Go》嘅Idina Menzel，同埋推出單曲版嘅Demi Lovato。Jaime Ciero望分得歌曲同電影所得嘅利潤，如佢獲勝訴，料會獲得大批賠償。
