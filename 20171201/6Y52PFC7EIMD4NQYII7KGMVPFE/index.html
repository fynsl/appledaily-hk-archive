寶馬波子Benz傾巢而出！
2018新車搶先睇

一年一度美國西岸最大型的洛杉磯車展（LA Auto Show）將於周五開幕，各大品牌2018年度新車將於本周三及四會在汽車科技展周三及四的汽車科技展（LA Mobility）搶先發佈，《蘋果》記者當然唔執輸，帶你到現場搶先看最新熱門車款。
駐洛杉磯記者：張紫茵

現代汽車 － 新運動型休旅車（SUV）Kona

不少品牌推出新款車時，均是以舊型號為基本改良，而現代汽車（Hyundai）今年則選擇推出嶄新運動型休旅車（SUV）Kona，大膽的顏色加上俐落的線條，甚具玩味，令人耳目一新。現代汽車產品設計經理黎浩年（Trevor Lai）表示，Kona是繼Tucson、Santa Fe後最新的小型SUV，希望吸引心境年輕的客戶，未來現代汽車會繼續設計更多小型SUV。

「從正面看會發現整個布局都很不一樣，日行燈（DRL）與車燈分開擺，而頭燈融入水箱罩，以SUV 來說是很標誌性的。」北美Kona首席設計師Daniel Kim解釋。Kona除了車身設計更具線條外，顏色上亦相當大膽，清新的檸檬綠﹑帶海洋氣息的藍、令人血脈沸騰的紅，如果想要低調型格，亦有傳統的黑白灰銀色，五顏六色任君選擇。

寶馬－i8 Roadster, i8 Coupe, M3 CS, Mini SE Countryman ALL4

寶馬（BMW）為華人最愛品牌之一，近年走向電動化，品牌目標於2025年推出25架電動車，而周二的發佈會上最新的油電混合跑車i8 Roadster及i8 Coupe正式亮相，將於2018年5月推出市場；加上較早前推出2018 i3s，為旗下i系列再添新員。另外，寶馬旗下英國老牌Mini亦有革新，推出首台油電混合車Mini SE Countryman ALL4，為可愛外表再添「電力」。

凌志－RX 350L, RX450hL, LX570

一家七口出遊，往往一輛普通房車載不下，往往都要犧牲車的外型買架外表「老實」的7人車。豐田旗下主打高端豪華車的凌志（Lexus）了解多人家庭的需要，今次車展跳出框框，熱賣的SUV增多一排，由五人車變身成三排的六人或七人車RX 350L，後排可隨意摺起成儲物空間，亦可選節能的油電混合版本450hL。反而亦然，傳統有三排座位的八人車LX570 則推出兩排選擇，車尾儲物空間立刻增加15%，增加不同選擇。

平治－S Class Cabriolet, CLS

豪華房車當然少不免平治（Mercedes Benz），洛杉磯四季氣候得宜、風光明媚，可算是開開蓬車的最佳城市，平治公佈最新旗艦跑車S Class Cabriolet，同樣配備早前加入Saloon的伸延駕駛系統。此外，第三代CLS配上最新4氣缸或6氣缸引擎，新款延續車型低矮修長設計，車頭以三角形燈組配上繁星式水箱罩，整體感覺修長優雅。

保時捷－911Carrera T, 718 Boxster GTS, 718 Cayman GTS

超跑少不了保時捷（Prosche），保時捷少不了長青車款911，雖然最新911 Carrera T配備較低，引擎改為3.0公升Turbo，但最大馬力亦達到370匹，車身重量減輕令到速度不會被拖慢，真正做到「Less is More」。另外公佈兩台718 系列，分別是718 Boxster GTS 及 718 Cayman GTS，升級後0到每小時100公里只需4.1秒，最高時速達每小時290公里，帶來不一樣的風馳電掣體驗。

是不是意猶未盡？周四於LA車展多個日本大牌子將會公佈年度新車，請緊貼《蘋果》現場直擊！
