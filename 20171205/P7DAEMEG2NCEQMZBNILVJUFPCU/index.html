新iPhone雖然支援快速充電，但隨機只附送5W的充電器，要享受快充，還要額外購買配件。由於新iPhone採用USB PD充電標準，所以副廠USB PD充電器也支援iPhone的快充，不過售價$188的USB C to Lightning連接線就要買原廠。

內地媒體最近從不知名廠商得到一款USB C to Lightning的轉換器，這個售價僅$11（9元人民幣）的轉換器可令普通USB C to USB C連接線支援iPhone快充。此「神器」外觀與普通轉換器無異，體積略大，經媒體測試後的確支援iPhone的18W快充，相信此轉換器一上市就會引起新iPhone用家瘋搶。

雖然這個「神器」以極低的價格實現快速充電，但品質始終令人擔心，要安全免叉壞機，還是買原廠線吧。
