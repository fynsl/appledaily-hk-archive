生意之道 - 高慧然

這天在日本網站見到一個有趣的通告，由「旭酒造株式會社」社長桵井一宏的名義刊出，拜託顧客不要用高價購買獺祭系列。
香港人對這隻酒絕不陌生，獺祭系列的純米大吟釀一向甚受香港人歡迎，其中的二割三分，是獺祭系列中最高品質、也最受歡迎的純米大吟釀。二割三分指的是它的精米度，一隻米被磨走外層76%，只取最中心的23%用來釀酒。除了製作步驟繁複，價格也相應提高。所以獺祭二割三分常常被它的粉絲搶購，一推出巿場，很快被搶購一空。自從香港人及中國人愛上它之後，價格更被不斷搶高。
對此，製造商發出通告，列出原廠所訂的價格分別是：獺祭三割九分720 ml （2418日圓）、1800 ml （4835日圓），獺祭二割三分720 ml （5142日圓）、1800 ml （10285日圓）。製造商更列出旗下各大分銷商的名及地址，要求大家購買原價產品。
製造商要求大家直接去分銷商那兒購買，這不奇怪。不過有趣的是，製造商明明看到獺祭被人搶高價格，而無論被搶得多高，仍不影響顧客購買慾，證明買家認為它值更高價格。在這種情況下，製造商仍堅持賣原定價格，而不是趁機提高價格，這個，大約就是日本人的生意之道和他們的堅持了。
在商言商，商品的價格由供求關係決定，若因此而提高價格，也完全可以理解。但日本人有他們的執着。
