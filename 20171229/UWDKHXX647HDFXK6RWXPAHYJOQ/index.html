【影相館】富士Instax-Wide VS Impossible project　即影即有選邊種好？

踏入數碼年代，大部份人日常接觸的都是數碼攝影，舊式的菲林攝影漸漸淡出市場，被認為是文青玩意，受到年青人的喜愛。但其實除了膠片攝影和數碼攝影外，一直以來歷久不衰的還有即影即有－這種拍完後吐出相紙立即顯影成像的攝影媒介。

即影即有本來的用途是什麼？

史上第一張即影即有的人像照是1965年拍攝的，它是為了方便使用在證件和身份證之上。當時亦有電影團隊使用，他們為了保持電影的連貫性，拍下演員的衣服、道具等各項元素對照，務求各方面不會出錯。至於日常使用，由於即影即有的畫質（銳度／可輸出大小）等等不夠菲林底片好，而且價錢偏貴，所以在以往一般作為上流社會階層開派對時候的玩意，故上世紀以此創作的藝術家不多。

富士Instax-Wide和impossible project 其實各擅勝場！

目前市面上最大行其道、市場佔有率最高的是富士Instax-Wide, mini 和square 系列，三者基本上只是片幅大小不同、藥水配方和顯影顏色大抵是相同的。
Instax Wide有著最高的穩定性，顏色偏藍綠的討喜特點，再加上機器和相紙便宜等眾多優點，基本上新手也能簡單操作，所以在市場上風頭無兩。

富士在價錢上擁有巨大優勢：

如果是Instax mini的話，有眾多卡通造形的框可以選擇，最便宜的白框價錢大約是＄35-＄40港元左右（10張）Instax -Wide 大約是＄50-＄65左右，相比起impossible project 的SX-70 film 的＄150-＄170 (8張)便宜得多。
相機方面，富士的即影即有系列相機機身很便宜，大約＄400（mini）-＄1200 （wide）左右，而impossible 的sx-70 約為＄2500，高階機種的670M 或670S更要價$6000港元以上。富士即影即有系列在價錢上的優勢顯而易見。

不過，要是說富士完勝，那也是不妥當的。
機身如此親民的低價是有代價的。富士即影即有相機的機身以塑膠為主，拍攝的鏡頭以塑膠制造，光圈小、成像一般，而且便宜版本的機身基本上不能控制對焦、光圈快門等曝光條件，因此就白了就是玩味重、易操作，但在創作上局限很大的機器。在室外足夠光亮的時候、不使用閃光、在室內昏暗的環境則開放閃燈，使用者可以調整的範圍，大概只有這些而已。

相對上，雖然價錢貴了一大截，但SX-70 等機器還是有不少優勢。在造功上，雖然都是使用工程塑膠物料，但上手的感覺金屬感很強、機身相當紮實。可以摺疊的機身和單反式的取景器令質感提昇了不少。鏡頭光圈雖然也不大，但畫質優良、最近對焦距離亦相當近，可以拍攝的題材很多。

我應該選擇那一種？

如果你是新手的話，不論你的財力如何，我是先推薦你使用富士系列的即影即有。
Impossible project的底片穩定性低，在不同光線、紫外線照射、溫度、貯存情況等環境下發色和顯影效果差異很大，而且因為習慣了數碼攝影的人，不自覺地喜歡拍攝大景、逆光等迷人題材，然而這些卻非即影即有適合的拍攝條件，新手上路的話挫敗感會非常大。甚至最初的十盒八盒相紙全軍盡墨也是常有之事。
相反，富士相紙穩定性大，無限遠對焦方式加上有內建閃光燈，可控範圍少但也意味著拍攝失敗機率低，如果你是一般玩家，以即影即有作生活記錄，這應該是一個性價比高的方案。三種不同片幅當中我較推薦Instax-Wide，而正方形的Square版本則總有片幅不夠大的雞肋之感，價錢亦較昂貴，比較不推薦。

如果你是有一定經驗的玩家，有兩個建議：

流行的富士即影即有相機，搭配的鏡頭都不算很優秀，在創作上會造成一定的局限（當然還有一些用此器材相當優秀的攝影師）。所以，有部份人以過去因為底片停產而變得沒用的相機如：Polaroid 110 , 180 等型號進行改裝，切開原本的片倉，加上Instax-Wide的菲林後背，成為了一台使用Instax-Wide、並具備優質大光圈鏡頭的相機，實拍效果非常出眾，只是在香港能完成這種機器的人不多了。當然，也有很多不改機的玩家。不改機的人較重視概念和意像，改機的人則傾向重視使用時的趣味性和更豐富的技術展現。

另一方面，如果是創作型的即影即有玩家，嘗試一下impossible project 的相紙也是非常特別的。Impossible project 的相紙雖然極不穩定且價格高昂，但只要掌握到溫度和曝光時的光線條件，可以拍到相當出色的作品，我會以「通透感」來形容，即便是以數碼作模仿也非常困難。
如果你喜歡這種顏色、對這種獨有的成像有偏執的需求，我是挺建議大家嘗試一下impossible project 的相紙的。當然，背後可能要以「年」作單位的訓練了。
