膝關節置換　術後伸展防變長短腳　
東院為長者推QR code　一嘟睇片掌握動作

【本報訊】嚴重膝關節炎病人常受膝痛困擾，接受全膝關節置換手術，以人工關節取代受損關節，配合術後復康增加關節伸展幅度，有助改善活動能力。但術後若不注重物理治療，膝關節會變得僵硬，嚴重會變「長短腳」，甚至要再次開刀。東區醫院去年2月起加強家居復康支援，由物理治療師示範膝部、四頭肌運動及正確步行姿勢等，已換膝關節的病人以智能電話或平板電腦掃描QR code，連結至網頁參考動態圖片，除可幫助記性不好的長者病人掌握正確姿勢，亦鼓勵家屬參與復康。
記者：袁樂婷

全膝關節置換手術病人在術前一個月至術後須做物理治療，訓練膝關節及腿部肌肉的強度，以加快痊癒。該院二級物理治療師陳義安表示，病人在術後一至兩日已可做復康運動，一般住院5天，其後2至3個月就要靠在家自行復康配合。

懶郁動令發炎組織黏連

手術會令病人部份身體組織受損及發炎，假如郁動較少，發炎組織便會黏連，如疤痕般缺乏彈性，阻礙關節屈伸，因此須透過復康運動避免這情況。他舉例，病人坐在椅上，用另一隻腳借力將患腳往後推，可訓練膝關節屈曲，將膝部往下壓則訓練患腳伸直，避免「長短腳」。動作是否準確影響膝關節的伸展幅度，例如膝部下壓至伸直腳，做錯變成膝部凌空，便無效果。
該院一級物理治療師文鎮邦指出，尤其長者病人一般學習速度較慢，記性亦不好，未必能記熟指示，亦因這些動作會令患者感到痛楚，部份人不敢放膽，「擔心痛係要停止嘅signal（訊號）」。他稱，物理治療師會為病人度身訂做復康計劃，期望膝關節靈活度恢復至能應付日常生活，包括上落樓梯，若不做訓練，最差情況是將來走路「趷吓趷吓」、坐車時「伸隻腳出嚟俾人踩」，甚至要在同一隻腳再施手術。
東區醫院物理治療部一直有製作訓練單張，供病人在家使用，參考動作的正確姿勢，去年2月試行加入QR code和動態圖片。文指，新計劃盼鼓勵病人家屬參與復康過程，「多對眼督促病人」，助敷冰消腫之餘，亦可從旁指導，有助掌握準確動作。部門經理溫淑盈表示，接受全膝關節置換手術的病人普遍是長者，過去兩年約170名病人中，就有86.5%為65歲以上；且相關的復康運動較複雜，因此以這手術為試點，觀察長者及其家屬對QR code的接受程度，現逐步擴展至乳癌病人。
