隔牆有耳：胡杏兒姐夫　係Chef Le Mon鐵粉？ - 李八方

隔牆有耳：
胡杏兒姐夫　係Chef Le Mon鐵粉？ - 李八方

民主派就為九西、新東補選搞緊初選，至於建制派一樣有動作，好似盛傳代表建制出戰九西補選嘅民建聯、胡杏兒姐夫鄭泳舜（Vincent），早前就獲港九潮州公會邀請，同近日憑住「單線solo」人氣再度爆燈嘅Chef Le Mon謝霆鋒，一齊出席一個關壞社區活動，同「深水埗明哥」陳灼明一齊派飯，仲去深水埗一啲長者嘅家中探訪。
Vincent就話，佢透過公會認識Chef Le Mon，而Chef Le Mon對深水埗民生同埋飲食文化都非常有興趣，所以就由佢安排Chef Le Mon去北河同行，幫明哥手派飯。

霆鋒落區裝飯

不過由於Chef Le Mon魅力太驚人，只係現身一陣，已經有過百人圍觀，為咗安全起見Chef Le Mon最終只係負責裝飯，而Vincent就將飯盒送去劏房畀長者。
八方問Vincent有冇趁機同霆鋒jam歌，Vincent就話冇，不過自己同霆鋒只係差一年，「細細個就聽霆鋒唱歌」，所以見到面都覺得回憶返晒嚟。
