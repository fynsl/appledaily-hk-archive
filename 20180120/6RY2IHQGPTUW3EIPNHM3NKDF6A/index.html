華人商家群起反對不果
蒙市硬過400萬美元單車徑議案

在地大脈博而公共交通不發達的美國洛杉磯，開車是很多人民必需通勤選擇，大都會交通局（Metro）為了支持滅碳環保，與地方市政府合作建單車徑，讓居民通勤有多一個選擇。華人聚居的洛杉磯蒙特利公園市（Monterey Park）亦響應環保，剛通過與交通局合資399萬美元（約3,112萬港元），於Monterey Pass工業區建1.6英哩（約2.5公里）長的單車徑。不過此舉讓很多當地居民感到不滿。

駐洛杉磯記者：張紫茵

是次工程從Garvey街一直延伸到Cesar Chavez街總長度1.6英哩，重鋪後5條行車線及停車位維持不變，而路旁的15呎花槽位將會削一半建單車徑。工程由大都會交通局與蒙市各出資199.4萬美元（約1,555萬港元），分4年付清，2019年7月動工，料於2020年底完成。

市內增設單車徑原本應是樂事一樁，不過在周三晚上的市議會會議上，甚少露面的華人居民都爭相發言，不少發言者是最受影響的於Monterey Pass Road商家，現場二十多名發言者中，逾八成持反對意見，原因是Monterey Pass Road是工業區，平日沙塵滾滾，不少重型大貨車以時速40英哩（約64公里）駛過，加上路段彎位多造成盲點，如果設置單車徑恐怕會更危險；亦有商家指該路段意外頻生，絕對不是一條理想的單車徑。另外亦有二十多份書面意見，逾半為支持建單車徑，但大部份是鄰近城市居民，並不是蒙市居民。

4位出席的市議員聽完各方意見後持不同看法；副市長陳贊新及市議員梁僑漢贊成計劃，陳贊新指如果1月底未能通過建單車徑議案，交通局就不會再出資，Monterey Pass Road未來亦只可以維持舊樣，相信加入更多安全的設計下，此單車徑不只利好踩單車人士，對商家亦有利。

市議員塞巴斯蒂安（Teresa Real Sebastian）則反對計劃，因早前交通局曾在延建710公路上的撥款出爾反爾，因此對於他們的誠信存疑，而合約中亦指明如果交通局沒有足夠資金，撥款有機會取消，項目超資亦是由蒙市承擔；另外她又指蒙市爛路多，修路的基金明顯不夠，希望市政府的資金可以用得更實在去改善民生。與其設單車徑在杳無人煙的工業區，她認為設在人流較多的商業街Atlantic 大道更實際。

市長林達堅表示，大都會交通局的資助只限於這條路亦有時限，無論如何先通過拿到資助，即使交通局最後不批准，亦沒有損失，但拿到資助就一定可以改善Monterey Pass Road。在民眾反對聲音下，最後投票結果為3票贊成，1票反對，正式通過於於Monterey Pass工業區設單車徑。

支持議案的埃爾蒙地市（El Monte）居民Amy Wong表示，她認為街道安全是十分重要，改善Monterey Pass可以讓更多行人、踩單車人士更安全地前住鐵路站轉乘公共交通工具，所以對於議案通過感到開心，設計亦不會影響到現有行車路及停車位，踩單車人士亦有更安全的行車徑，是雙贏局面。

而反對的蒙市商家Daniel Wang就感到失望，他指商家們踴躍發言就是希望可以阻止此議案通過，奈何市政府似乎想趕絕市內的工業發展，以圖將該地段改為商業區。作為蒙市多年居民，他了解Monterey Pass Road路段除了工廠外，亦有拖車公司、消防局等，而工程期間將會影響到前線人員的出入，商家生計亦會大受影響。
