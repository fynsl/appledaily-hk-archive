【科技籽】拍片直播無手震　邊款手機穩定器入得過

【科技籽：出閘搜新】
手機拍片功能越來越強，不過拍片影相大不同，手持拍片很難避免手震又搖cam，所以很多KOL拍YouTube、做直播都會用手機穩定器幫手。不要以為穩定器太pro很昂貴，近年三軸穩定器越玩越平，航拍界一哥「DJI」即將推出的Osmo Mobile 2，電量長氣了兩倍有多、又有新操作，售價居然不升反跌，定價$999。開盒評測之餘，再找來同樣是千元價位的Swiftcam M4 Lite和智雲Smooth Q比較，試完你就知平價三軸棍其實都可以好穩定好高水準。

當Osmo Mobile初推出時，售價達$2,269，現在市價約售$1,500。新款Osmo Mobile 2一拿上手，便發現改用塑膠質感物料為主，感覺不夠金屬製的第一代貴價，不過若論功能的話，Osmo Mobile 2算是「平咗又勁咗」。首先電量由980mAh大升至2,600mAh，使用時間由約5小時激增至15小時，雖然沒有了換電設計，但對於用家來說，狂玩15小時一定方便過換電。實試連續拍攝20分鐘，電量也只用了約4%，長氣程度驚人。

支援直度拍攝直播更方便

除了電量增多外，Osmo Mobile 2在手持位也多了一個USB輸出可幫手機充電，不過手機放上支架時一般都會擋住了充電口，要邊拍邊充電就不太可能了。另一進步是更換了支架設計，扭動機背轉盤就可直放手機，支援直度拍攝，用來做facebook直播更方便。另外操作上也新增了zoom掣，用來控掣手機的變焦。其餘功能大致和第一代相同，測試中使用的「DJI GO」app也和舊版一樣，玩到追蹤物品和縮時攝影。基本上除了用料，這次比上代便宜，功能也更齊全。
另一方面，Smooth Q上年推出時跟足Osmo Mobile，做到自動追蹤物品和移動縮時拍攝（Motion Time Lapse），再加上支援直度拍攝和千多元售價，得到不少好評。而另一支測試的Swiftcam M4 Lite則是香港設計的本地薑產品。三棍比較，先講設計，Osmo Mobile 2和Smooth Q如出一轍，扭動轉盤能夠直放機身；Swiftcam M4 Lite比較特別，連按4次joystick掣就可自動扭直手機，切換至直度模式。

本地薑M4 Lite安裝方便

實試三支穩定器安裝大屏幕Note 8，都能成功穩定機身，但Osmo Mobile 2和Smooth Q要先用轉盤扭闊支架，M4 Lite則可直接安上，論安裝方便，M4 Lite稍勝，其餘兩機相若。操作上，M4 Lite和Smooth Q能夠透過按掣，切換「全鎖定」、「全跟隨」、「左右轉向跟隨」三個模式；而Osmo Mobile 2可按開關掣切換「全鎖定」和「全跟隨」，雖然可在「全跟隨」時用app鎖定俯瞰軸，效果和「左右轉向跟隨」一樣，但有少許不便。
三機在重量和穩定能力也相若。若談特別功能，三機皆可追蹤物件拍攝，以追蹤功能而言，Osmo Mobile 2有較明顯的優勢，就算車子在面前快速經過，也能順暢追蹤到一段距離。三機也能拍到縮時攝影，Osmo Mobile 2和Smooth Q會比較多功能，例如自動「pan鏡」的移動縮時縮影。M4 Lite在功能上有較明顯差距，而Osmo Mobile 2和Smooth Q則表現相若，若論細節，Osmo Mobile 2在追蹤物品和操作反應會好一點，但仍然做不到Smooth Q的左右轉向360度。

Osmo Mobile 2適合新用家

如果是新用家，DJI品牌效應以及$999超抵價，Osmo Mobile 2不會令你失望；但如果已有Smooth Q的話，功能已十分齊全，也不用急着換新機。
