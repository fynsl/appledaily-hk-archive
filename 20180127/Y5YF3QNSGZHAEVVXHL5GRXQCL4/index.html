【上海直擊】Nike新科技跑鞋輕巧夠彈　徐濠縈：適合初試跑者

雖然今年渣馬跑完，不過真正喜歡跑步的跑友又怎會停下來？Nike剛剛在上海發佈了旗下最新中底技術的跑鞋Nike Epic React Flyknit。React泡棉已非第一次跟大家見面，之前就率先搭載在Hyperdunk 2017和Jordan Super.Fly 2017兩款籃球鞋上，今次是首次加入跑鞋款中，並以輕盈、柔軟、能量反饋和耐用作四大賣點。

Nike跑鞋類產品創新總監Ernest Kim在發佈會說：「我們知道籃球和跑步的需要不同，籃球需要大量橫移動作，所以我們會在React外加上外底，加強多向移動的穩定性，但跑步則着重向前直線的動作，所以今次新鞋就用上一整塊React泡棉作鞋底，不設中底承托結構、膠水貼合等設計，盡量減少不必要材料，以減輕重量。」雖然是一整塊React物料，但透電腦計算，會在鞋底不同位置設計出不同厚薄和紋理，來提供不同緩震和支撐度。鞋底是今次新鞋重點，但鞋面也沒有馬虎，用上一塊過的單層輕薄Flyknit物料，高透氣、輕巧兼堅韌，能提供不錯的支撐和保護。

大會安排實試環節，讓傳媒和嘉賓試跑，大會請來徐濠縈、李榮浩、中國田徑運動員及2014亞運女子200米跑金牌得主韋永麗等一同試跑。徐濠縈跑完大讚：「暫時覺得它非常適合日常跑步，也適合初試跑步的人，我特別欣賞它的輕巧和力量回饋，的確能幫我減少虛耗力量，剛剛試跑時，我感覺好像越跑越快。」她覺得鞋形夠slim夠簡約，也特別喜歡它的白色配色，女生日常配襯也不錯：「或者不少人像我一樣，去外地工作旅行都想跑步，這對鞋出街夠靚，跑步亦舒適，一對搞掂，就可以不用浪費行李位置多放一對鞋了。」

而韋永麗雖是短跑手，但原來她也需要做慢跑和肌肉訓練，所以新鞋對她來說也是相當好的訓練工具：「它對保護腳部有很大幫助，在我們的訓練中有一些肌肉訓練，對腳與地面的感覺要有很高的要求，它的高能量反饋性能正好適合訓練。之前我也有穿Lunar底的跑鞋，但它太軟，易有左右擺動的情況，相對來說React就提供保護性之餘，同時能有更好的支撐和穩定性。」雖然得到好評，不過兩位嘉賓和Kim的都說Nike Epic React Flyknit適合一般訓練和每日日常跑步，至於用於比賽，就要再次實試才知，Kim：「我們相信世上沒有完美的跑鞋，因為每個人需要都不同，不過Nike Epic React Flyknit作為我們最新的跑鞋，其舒適和耐用會是每日享受跑步的人的不錯選擇。」
