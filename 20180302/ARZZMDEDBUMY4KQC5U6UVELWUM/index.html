本田(Honda)Civic Type R，頭驅揭背靈活好玩，是很多日系小跑迷的最愛。上年去到德國N籄rburgring Nordschleife賽道，更加破盡福士(Volkswagen)olf GTI Clubsport S創的7:47.19圈速，以及Renault Megane RS 275 Trophy-R的7:54.36，成為綠色地獄頭轆王！今年本田又要再挑機，預告去各大賽道「踢館」！

其實早於2016年，上一代Civic Type R已經在歐洲五大賽道寫下頭轆車紀錄，今年新車又要去玩，官方發佈「Type R Time Attack 2018」這條影片，預告會去到英國Silverstone、比利時Spa-Francorchamps、葡萄牙Estoril等賽道計時，而且還會由不同車手駕駛，包括Jenson Button、Tiago Monteiro、Esteban Guerrieri、Bertrand Baguette，絕對令人期待！至於這部2.0公升Turbo引擎、306hp馬力、六速手波的頭轆車表現如何？就要留意過幾日舉行的日內瓦車展。
