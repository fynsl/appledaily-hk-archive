【add hot】Supreme聯乘Nike誘NBA迷

潮牌Supreme上周五推出三方聯乘：Supreme x Nike x NBA的「限定商品」，又找來NBA球星J.R. Smith拍宣傳照，成為潮人及籃球迷喪搶潮物。官網自然一早售罄，日前東京澀谷店開賣，內地十幾名黃牛排隊黨，被拒買貨後，狂打保安，非常離譜！
這批Supreme x Nike x NBA三方聯名的潮物，以多個NBA隊伍Logo佈滿整個設計，包括Tee及外套等，官方熱門貨在各大拍賣網炒高一兩倍價位。而向來熱愛籃球的周柏豪則不用排隊，日前在微博post出球衣外套、波鞋等，鄭秀文（Sammi）亦晒聯乘外套，兩人率先入手。

Puma藉BTS攻全球

要在Nike、adidas等強勁對手中突圍而出，Puma自有策略，近年陸續請樂壇天后、超模如Rihanna、Cara Delevingne及Selena Gomez等加持，陣容龐大。
而繼15年與韓團防彈少年團（BTS）合作出韓國限定聯名系列後，今年再度聯名BTS，但今次不似過往「韓國限定」，而是全球販售，希望藉着BTS的人氣打入年輕市場。Puma X BTS聯名系列包括Turin復古鞋款及各款運動裝，並已於韓國開賣，而香港將會在7月推出。
