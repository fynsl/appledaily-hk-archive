花展想影靚花　掌握微距事半功倍

【特約報道】
香港花卉展又開始喇，今年的主題是「心花放」，寓意今年的主題花大麗花盛開時美態萬千。講到影花，好多人都以為花係唔識郁任你影，應該沒有難度，冇錯花係唔會黑你面，但如果你唔記熟幾個秘訣，最後影到古靈精怪，就浪費了精彩的花展！以下提提大家幾個實用技巧，包你易學又易精。

1.畫面越簡單越有張力

不少人有誤解，以為顏色多就會好看，不過相信你也聽過「花多眼亂
，嘗試在畫面中找出重點，有時是花蕊有時是花瓣，甚至是在採蜜的蜜蜂，總之一定要有一個焦點，拍出來的花照才會好看。

2. 對焦要全時手動對焦作微調

大家現時影相都習慣了自動對焦，一般人幾乎都未試過手動對焦！但以高放大倍率影花的話，因為花有機會太過細小，有時需要適度使用手動對焦微調焦點，而要做好微調就最好選擇有全時手動對焦功能的鏡頭，例如EF 100mm f/2.8L Macro IS USM就支援這個功能，可以在鏡頭的自動對焦模式之下，先半按快門再轉動對焦環作微調，這樣對焦就才可以萬無一失。

3.用大光圈是常識

大光圈鏡頭影花有很多好處，一來可以用淺景深虛化背景，畫面較乾淨看上去更簡潔；大光圈鏡頭另一個好處是透光量大，快門速度相對可以提高，相片就不易鬆郁朦。

4. 微距鏡選擇有學問

要影花就一定要配備微距鏡，如果要選擇一支就手又高質的微距鏡，不妨考慮Canon 的EF 100mm f/2.8L Macro IS USM，支援1.0x放大倍率，適合高質素微距生態拍攝，最吸引是配備混合型光學影像穩定器 (Hybrid IS)，需知道越近距離拍攝，手震影響就越明顯，這支鏡頭的Hybrid IS於正常拍攝時提供4級快門防震，而在1x放大倍率時，更可以做到2級快門防震，影像細節清晰銳利，非其他只採用普通防震功能的鏡頭可比擬，加上防塵防水滴，就算落毛毛雨都唔怕！

至於其他相機系列如APS-C格式或無反相機，就可以考慮Canon EF-S 35mm f/2.8 Macro IS STM及Canon EF-M 28mm f/3.5 Macro IS STM，兩支鏡都已經內置LED微距補光燈，更可視乎需要左邊或右邊亮起，補光就更加方便，後者更支援最近工作距離僅13mm的超微距拍攝模式，進一步突破微距界限。
