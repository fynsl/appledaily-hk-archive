【跑友注意】JOOX增名人語音跑導　運動女神鼓勵：跑埋佢！

Nike的Epic React Flyknit可說是近期最火熱的鞋款之一，鞋迷們對這對全新科技的跑鞋都議論紛紛，雖然有人就話鞋底有點容易磨蝕，但亦有唔少人大讚新底夠舒服，行路跑步感覺都一流。不過計我話，跑鞋最重要的還是「跑」！所以在新鞋推出同時，Nike最近在音樂串流程式JOOX中加入「Let’s Go! NIKE語音跑導」。相信絕大部份跑友都愛邊跑邊聽歌，放鬆之餘亦有助保持節奏，而今次Nike就特別邀請了多位名人為跑手錄製鼓勵及打氣說話，包括林海峰、細So、Donald、Nike+ Run Club總教練蘇凱男，更有運動女神歐鎧淳及楊文蔚，鼓勵大家繼續向前跑，一步一步挑戰自己！而JOOX上架的播放清單共有三種模式，包括輕鬆跑20分鐘、輕鬆跑30分鐘及挑戰30分鐘節奏跑，每個歌單前後，以及在不同時間中，都會出現適合時機語音播放，例如跑到一半時就會講：「就嚟跑到一半喇……」；跑到臨尾女神又可能會同你講：「唔好亂，Keep住呼吸，跑埋佢！」等等，唔多唔少都真係幾有激勵作用，仲好似多咗個同伴同你一齊跑咁添！

另外，除了JOOX外，Nike+ Run Club應用程式同時亦提供更多種語音跑步指導模式，例如Speed Run、Long Run、Recovery Run等。Nike 的教練及運動員Mo Farah、Eliud Kipchoge等，都會在語音跑步指導運動計畫中「現聲」，與跑者一起進行跑步訓練。
