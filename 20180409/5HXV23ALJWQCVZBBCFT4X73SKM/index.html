親美黑客襲俄國伊朗

俄羅斯和伊朗等地電腦系統上周五同遭親美黑客襲擊，影響當地互聯網服務供應商、數據中心及部份網站運作；遇襲電腦螢光幕都出現一幅美國國旗圖像以及一句說話：「別來攪亂我們的選舉。」（圖）

「別攪亂我們的選舉」

黑客代表前日接受科技網站Motherboard電郵訪問，直認發動攻擊是為「發出明確訊息」；他們不滿「親政府黑客」一再襲擊美國及其他國家，矛頭指向涉嫌干預2016年美國大選的俄羅斯。
美國國土安全部轄下「電腦緊急應變小組」（CERT）上月警告，指受僱俄羅斯政府的黑客，正以能源等關鍵基建作目標。思科系統（Cisco）指黑客是利用該公司路由器軟件Cisco Smart Install Client已公開漏洞入侵，部份是針對重要基建系統，Motherboard相信親美黑客利用同一漏洞作反擊。
雖然親美黑客稱行動只針對互聯網內的俄語系統，但伊朗通訊及資訊科技部說，伊朗、歐洲各國、印度以至美國本土用戶亦受牽連，涉及全球20萬組路由器開關（router switch），其中3,500組在伊朗，並指伊朗95%路由器已恢復正常。
