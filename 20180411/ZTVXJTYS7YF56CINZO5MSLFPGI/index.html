隔牆有耳：神沙車增值用QR Code - 李八方

隔牆有耳：
神沙車增值用QR Code - 李八方

相信唔少人都試過攞住一大堆毫子等硬幣到金管局收銀車換紙幣或入落八達通，無重一身輕；兩架「神沙車」2014年10月落地以來確實大受歡迎，至今已為40.6萬人次提供服務，合共收集3.4億枚總面值4.53億港元嘅硬幣。
因應香港使用電子錢包日漸普遍，兩部神沙車最近升呢，提升設備令市民兌換硬幣時可選擇用QR Code為電子錢包增值，參與嘅電子錢包包括支付寶香港、八達通O!ePay、拍住賞、TNG Wallet同WeChat Pay。
兩部神沙車會繼續輪流喺全港18區為市民提供免費收集硬幣服務，服務時間係朝10晚7，出沒地點可以參考金管局網頁。
