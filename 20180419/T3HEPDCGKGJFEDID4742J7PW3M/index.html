數碼廣告：無登入，無賬戶，fb照追你行蹤 - Jansen Lu

Mark Zuckerberg就Camrbidge Analytica資料外洩事件出席白宮聽證會。除了資歷深的參議員不懂科技，成為「焦點」之外，另一個焦點就回到facebook如何追蹤及記錄網上行蹤的問題。

網絡廣告興起，對於追蹤用戶行蹤，在甚麼網站逗留多久、由甚麼網站跳到甚麼網站，對甚麼內容有興趣等，其實又何只facebook。所有以賺取網絡廣告收入的公司，例如Google、Twitter等，其實都想盡辦法去追蹤，務求盡量清澈了解每個用戶的profile，這樣才可以讓廣告商精準落廣告。facebook廣告其中一個吸引人之處，也是夠精準的ad targeting，而且賬戶更是行實名制。

在兩天的答問中，其中一個很值得留意的話題是「shadow profile」。我們用facebook，Mark Zuckerberg的團隊一直追蹤着我們也是意料中事。我們看過甚麼專頁的內容、like過哪位朋友的旅行相片，甚至unfriend了哪誰，facebook都記錄在案；但原來就算你沒有登入facebook，甚至你不是facebook用戶，其實facebook都有你的紀錄。

facebook有個工具叫pixel，讓廣告商放在自己的網站上，就可以記錄用戶的動態。當用戶去到一個安裝了facebook pixel的網站，facebook就辨認到這人。如果他有facebook賬戶，facebook甚至可以知道這是哪一位；就是沒有facebook賬戶，又或者已經登出了facebook，facebook雖然沒有一個user profile，但也有一個所謂的「shadow profile」。當這人建立一個賬戶或登入之後，這個shadow profile就變成一個已經有相當多資訊的user profile。

這個shadow profile當然是facebook廣告的一個重要工具啦，但為何這個工具會惹起爭議呢？當然很多人認為facebook在自己的網站／App上追蹤用戶就較合理。離開了自己的地盤還追蹤，就難以接受。

另外，Mark Zuckerberg在聽證會上，一聽到新墨西哥眾議員Ben Lujan問及shadow profile的問題時，他只說自己對這工具沒有印象（不是說I'm not familiar，就是說I don't know。兩人的對答可看Slashdot的節錄： http://pse.ee/5SDQ6 ）。這令人覺得他在迴避甚麼，好像不願正面回應Ben Lujan的提問。

雖然你說I don't know，但其實facebook的help有一個頁面是「I don't have a Facebook account and would like to request all personal data stored by Facebook.」( http://pse.ee/5SDRP ) 。這頁面提供的方法就是叫你建立一個賬戶，然後就可以下載一整個copy儲存在facebook，你的個人資料。不過及後又有人發現，即使登入了facebook使用這裏提到的工具，用戶還是無法下載shadow profile內的資料。在沒有得到用戶同意，加上用戶不能控制收放多少個人資訊給你時，就容易令大家感到不安。

眾議員Ben Lujan接續問：用戶可否opt out不讓facebook記錄這個shadow profile呢？也有人問，你有否問准我的同意再記錄我這個profile呢？

用戶其實已經很難放棄或離開facebook、Google甚至其他網絡公司，所以更避免不到他們的追蹤嗎（人人都說delete facebook，有幾多delete了呢）？我看就是很難。但如果這些網絡公司在不問准更不通知，亦沒有opt out的情況下，就可以收集大家的資料，那就真正變了一種霸權。可能你會說，就算有甚麼用戶條款詳列facebook的私隱條款，我也只會scroll到最底click「I agree」繼續用，但到底這些都會對facebook他們的措施或多或少有點壓力或規管。
