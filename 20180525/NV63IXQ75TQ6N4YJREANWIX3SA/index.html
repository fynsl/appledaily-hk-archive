【電競神器】Razer北京公佈15.6吋電競筆電　85%屏佔比打機超爽

最近不停有手提電腦的資料發佈，這次的發佈就輪到電競配件起家的Razer了。Razer近日在北京舉辦了一個全球發佈會。邀請了大量粉絲和傳媒一同見證新產品的發佈。一踏入會場就會看到一塊巨型簽名版讓各位粉絲和傳媒朋友簽個名。Razer向來識冧fans，派發的記念品包括三頭蛇手指陀螺，幾有噱頭。現場狂射招牌綠色燈光，令觀眾變成「變型俠醫」顏色來看發佈會。

這次公佈的重點分別是Razer Blade 15的新版本和Razer Core X 外置顯示卡盒，除了有靚仔CEO陳民亮親身介紹產品之外，還請了一隊《PUBG》職業隊「4am」來到現場打表演賽！

這次的Razer Blade 15為整條Razer Blade 的產品線填補了中間尺寸空缺，CPU 一律採用Intel Core i7-8750H，15.6吋屏幕亦由最低的60Hz 至144Hz 都可以選擇，最高解像度達4K，還有 Touch Screen，顯示卡為GTX 1060 或者GTX 1070，雖說不上是最頂尖配置，但電腦僅重2kg，再配上超薄機身，睇落極修身，Razer就號稱是全球最細電競notebook。

再睇真啲spec，Razer Blade 15 採用 4.9mm 窄邊設計，屏幕佔比達 85%，厚度只有 1.68cm (GTX 1060)、1.73mm (GTX 1070)，重量為2.07kg (GTX 1060)、2.1kg (GTX 1070)。用同樣輕薄自居的Asus Zephyrus M比較，Zephyrus M高1.7cm、重2.2kg，Razer Blade 15係輕咗個馬鼻。售價方面，香港版設有四款規格，由$15,699起至最貴$21,499，分別在於屏幕、顯示卡及 SSD 容量規格之不同。

另一重點Razer Core X外置顯示卡盒，其實是上一代Razer Core V2 的延續，雖然為了推升最大支援的顯示卡尺寸，令機身比上一代大了一點，但價錢由超過$4,000減至$2,499，在用文書機而又想加顯示卡的朋友絕對可以考慮一下。
