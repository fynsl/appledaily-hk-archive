C AllStar吳崇銘忙電音節未諗單飛

C AllStar吳崇銘
忙電音節未諗單飛

C AllStar去年底在紅館舉行演唱會後就正式休團，成員吳崇銘（King）、陳健安（On仔）、梁釗峰與何建曦（Jase）作獨立發展。據悉他們所屬的寰亞唱片公司已落實阿King成為首位單飛成員，已經錄好新歌就快派台，但寰亞未有正式公佈。
身為當事人的阿King，昨日率先在一個似是錄音室的地方拍片，否認是第一個單飛派新歌：「唔係我喎，我暫時冇新歌推出，呢排忙緊6月嘅ULTRA！」而鍾情EDM（電子舞曲）的他，最近忙於準備下月在亞洲國際博覽館舉行的「香港ROAD TO ULTRA 2018」電子音樂節演出。
撰文：鍾國仁
