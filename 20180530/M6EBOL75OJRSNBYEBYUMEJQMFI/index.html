薛凱琪認鞋癡家藏數百對
軒仔買樓　接父母同住

張敬軒、薛凱琪、歐鎧淳及方大同等昨日到中環出席Chanel Beauty舉行的CoCo Game Center活動。軒仔接受訪問時表示現場除了化妝品外，還展示了很多90年代的街機及夾公仔機，他說：「呢度見到好多絕版game同佈置，覺得呢個團隊特登開發一啲符合嘅遊戲，真係好有心，自己玩時都好開心、忘形，我係CoCo Chanel fans，亦都有好多佢嘅products，好多化妝品甚至經典NO.5香水，自己都係用家，我好喜歡CoCo Chanel精神。」

買鑽石蝕10萬

軒仔透露方大同原來是夾公仔高手，但自己以往就經常奉獻給機舖。軒仔笑稱童年時最喜歡《魂斗羅》及《雙截龍》遊戲，是出賣年齡秘密的集體回憶。而今年除了開演唱會外，亦會出碟。
對於昨日在電台節目中自爆曾在巴黎買鑽石戒指受騙一事，軒仔表示是六、七年前舊事，當時買貴了10萬，他說：「多年嚟都蝕咗一間豪宅啦，但過程好開心，賺到嘅係經驗。新目標係喺香港買樓，接爸媽一齊嚟住。」

讚方大同琴技

張敬軒、薛凱琪、歐鎧淳及方大同等昨日到中環出席Chanel Beauty舉行的CoCo Game Center活動。薛凱琪喜歡收藏Chanel的鞋，因為是鞋癡一名，她說：「我收藏咗幾百對鞋，尤其係成日要着高踭鞋去跳舞，要揀一對好嘅鞋，唔係會傷到條腰，試過着一啲好靚但唔舒服嘅鞋，着到對腳損晒。」
她自爆每次搬屋裝修時，都會叫設計師幫忙設計一間房放鞋，現在家中有一間房放了幾百對鞋。而昨日她在現場跟方大同玩一款彈琴遊戲機，最後輸了10萬分給對方，她慨嘆方大同現實中彈琴出色，想不到打彈琴機都一樣出眾。
