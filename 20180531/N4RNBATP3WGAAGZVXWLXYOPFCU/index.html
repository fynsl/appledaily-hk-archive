【跑手召集】全港首屆鴨迷慈善跑　報名送限定選手包

黃色鴨仔到底有幾受歡迎？諗諗當年尖沙咀停泊一隻巨型鴨，有幾多人衝去睇就知。品牌B.Duck嘅主角鴨仔，同沖涼鴨鴨一樣受盡大人細路喜愛，嚟緊七月就由時裝踩過界，舉辦第一屆「B.Duck Run 2018 HK」，大大隻黃色鴨仔陪住大家跑步做運動！

當日賽事分5公里、2公里繽紛跑家庭組及個人組，以及800米親子跑，仲有依家時下小朋友最潮最型嘅運動──平衡車大賽，全家上下都有機會大展身手。現場亦會有巨型充氣鴨仔、B.Duck見面會、吹氣彈床等等滿足各位鴨迷慾望。報得主題跑，選手包亦係重點，限定版主題T-shirt、毛巾、獎牌固然唔少得，參加家庭組會有可愛鴨仔背囊，頭1500位報名選手仲會送B.Duck鴨仔毛公仔吊飾。報名費由2公里個人組350元，至2公里3人家庭組880元不等，其中部份收益會撥捐「苗圃行動」支授貧困兒童。做運動健身、潮聖鴨仔之餘，再做埋善事，有興趣嘅鴨迷記得準時報名。

「B.Duck Run 2018 HK」
日期：2018年7月15日
地點：大埔科學園
（6月1日開始報名，額滿即止）

記者：李珮禎
