VISA擬競投港鐵QR Code

【本報訊】港鐵（066）二維碼（QR Code）閘機系統招標正展開前期工作，VISA香港區總經理戴嘉倩指VISA已提交計劃書的資格預審申請，未來亦有興趣參與投標，但相信應用在交通系統最成功的支付技術，仍然是開放式非接觸式支付系統。
倫敦交通局商務分析部主管Jeremy Itzcovitz指出，多年前該市亦研究過在交通工具使用二維碼，但發現入閘前需要諸多準備，不便利乘客及容易出現人龍，故此2014年起開放支付系統，便利各項非接觸開放式支付，並指自推出以來，已錄得超過17億次感應式信用卡及移動設備支付。

稱開放支付系統沒難度

被問到港鐵入閘機應否開放支付系統，即一個收款器（POS）容納多個支付工具，例如非接觸式信用卡、Apple Pay、八達通等，戴嘉倩指香港本身有不少用家使用非接觸式支付工具，開放支付系統完全沒有難度，又指與香港相似的倫敦市早於2014年時已開放支付系統，信用卡亦可以入閘，不用買票或使用Oyster卡。她認為，當港鐵見到外地城市的先進後，將會考慮開放支付系統，又指開放與不開放是商業問題多於技術問題，原因是現時網絡連線已非常先進，開放式系統不會慢於封閉式系統，「10年前封閉式支付系統較快，今日開放式支付系統已經一樣優勝」。
