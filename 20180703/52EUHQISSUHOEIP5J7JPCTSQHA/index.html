【即影即有】評測寶麗來5款新系列相紙

在數碼攝影的主流時代，還有多少人記得相片的溫度，以及那種不完美的美？

歷史簡介

在即時攝影的年代，寶麗來獨占鼇頭，而且整個市場也是由它一手打造起來，Dr. Edwin Land的瘋狂想法最終成就出一年二十億美金的產業。從此寶麗來亦成了即影即有的代名詞﹝請注意，寶麗來是即影即有的其中一種﹞。在1970年代，全世界每年拍攝約十億張寶麗來的照片。直到現在，Polaroid「寶麗來」是地球上最家喻戶曉的名詞之一。 但隨著時代不斷轉變，攝影模式由底片轉為數碼，拍攝即影即有的人愈來愈少。最終2008年初寶麗來正式宣布停止底片生產，寶麗來的王國正式完結。

2008年寶麗來底片正式停產，一班寶麗來愛好者組成了即影即有公司IMPOSSIBLE，並購入有關生產線，重新投入相紙生產，IMPOSSIBLE 於2017年收購了寶麗來有關的業務，並於同年改名為Polaroid Originals（下稱PO），以經典的名字延續了寶麗來的夢。

底片介紹

藥水是即影即有的靈魂，藥水配方決定相片的成像，當中包括：顯影過程中的穩定性、顏色、線條等。PO的底片按感光度分70底片及600底片兩種，70底片ISO較低，成像顏色濃郁，線條較細緻；600底片ISO較高，成像顏色柔和，線條相對粗糙。

有很多人說富士即影即有底片較穩定，顯影時速度快上數倍。小編曾嘗試過不同的即影即有，終也改變不了對PO底片的熱愛，PO的底片追不上舊寶麗來的效果，但10以上分鐘的顯影時間，不穩定的成色，正正是PO底片吸引的地方，儲存方法、溫度、光線、紫外光等都會為影像帶來不同程度的影響。不但記錄影像，還會記錄當下的溫度。

於本文測試的是PO於2018夏季的新系列。

實拍感受

PO於夏季的大動作讓小編都大感震驚，原因是PO於今個夏天一共推出三款600彩色底片及三款600 I-type彩色底片，這裹先測試了夏季系列五款相紙。

本文測試的分別為

1. PO 600 Color Deep Sea Dive Edition
2. PO 600 Color Tropics Edition
3. PO 600 Color Ice-cream Edition
4. PO Color I-type Gradient Frame Edition
5. PO Color I-type Rose Gold Frame Edition

新一代夏季系列以新藥水生產，成色比上一代濃厚，對比度提高了，同時線條上亦比上一代清晰。相信大加對於上一代的評價是太多火紋，這一代的底片確實有明顯改善，小編於測試時只有Ice-cream Edition曾出現過火紋。乘著特別框之名推出，每款底片都有特別的框邊，Deep Sea Dive以深海為設計，深藍色的主色配上四款不同花紋；Tropics以熱帶顏色為主，黃色、綠色等為主；Ice-cream以八種不同的淡色為主；Gradient由淡藍到粉紅的漸變框；Rose Gold配上大熱的玫瑰金。

值得一提的是，Gradient及Rose Gold都是率先在亞洲區推出的特別框。其實我對於新底片有一點竟見，特別框確實十分吸引，亦看得見PO希望進一步提升I-type底片的市場，Rose Gold非常適合拍攝情侶及婚照；Gradient的漸變色相框亦俘虜了不少用家，但對於70系用家，就只可以?黑袋?為I-type底片更換為電盒版，頗為麻煩。其實五款底片官方都聲稱是用同一藥水，小編測試時就發現只有Ice-cream Edition 出現火紋，或估計生產上有時間差距。新底片都出現偏色問題，偏紫及紅，此舉相信是討好外國用家提升對比的?副作用?，對於香港或亞洲想信偏紅的問題較上代嚴重，用家需要以小技巧作預防。上述提及的對比度，小編建議用家可增加一至半格EV，以避免對比太高出現陰陽色問題。另外小編亦發現新底片於影像底部偶會出現「蝴蝶紋」的問題，未知是否意味著影像放久後會出現模糊化的情況。期望未來有更適合亞洲地區的新配方推出市場。

現在還喜歡寶麗來的，都不會介意她的不穩定性。底片的不確定性，是讓影像活起來的原因。你有拍攝寶麗來嗎？把照片上傳在留言中與大家分享吧！
