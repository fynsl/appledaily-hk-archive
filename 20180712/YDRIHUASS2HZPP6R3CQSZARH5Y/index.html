【頂級筆電】MateBook X Pro配91%屏佔比觸控芒　隱藏鏡頭撳掣先識彈

Huawei早在2月底巴塞隆拿MWC展覽上，已經發表頂級手提電腦MateBook X Pro，這部配備13.9吋3K屏幕的新機終於抵港開賣，開價$14,988，比13吋的MacBook Pro價錢更高！

當然價錢開得高，MateBook X Pro都有畀番對應的高階硬件規格，如配上16GB記憶體、512GB SSD及GeForce獨立顯示卡，而且顯示屏更升級至3K解像度及全觸控式設計，這幾點毫無疑問都較13吋MacBook Pro好。另一方面，MateBook X Pro更參考手機潮流用上高屏佔比設計，屏幕邊框特幼，令屏佔比提高至91%。但要做到高屏佔比，移走鏡頭是其中一個最簡單做法，Huawei今次就聰明在將鏡頭隱藏在鍵盤上方第一排功能鍵上，要按鍵才會彈出鏡頭，對於平常基本上唔用電腦鏡頭的記者，確實唔覺有大影響。

MateBook X Pro配備4喇叭，亦支援Dolby Atmos全景聲音響系統，實試亦覺幾好聲；另外機身亦配備大容量電池，官方表示足夠連播12小時影片。雖然記者未有機會將電腦Full Loading測試，在正常使用下，機身算得上幾薄的MateBook X Pro散熱效果都唔錯。初步測試下，MateBook X Pro在各方面表現都算幾到位，但售價高開確有一定阻力：撫心自問，畀得起$15,000買一部手提電腦，大家會揀MateBook X Pro抑或MacBook Pro？
