【潮店開張】UNDEFEATED進駐銅鑼灣　超限量KOBE入手方法公佈

講到香港人熟識的潮牌街牌，UNDEFEATED一定有份，事關佢地不時同唔同品牌合作，好似前排佢同KOBE crossover出嘅一系列產品，到而家都仲有唔少人高追緊，一白一迷彩的聯乘Nike Zoom Kobe Protro炒價更一真高企，冇4千蚊都幾難入手。雖然UNDEFEATED在港有不少fans，但想行專門店，近近地都要去東京原宿，不過好彩，呢個潮牌終於黎到香港開舖，成為香港，亦係中國首間專門店，以後想行唔駛搭飛機，搭「叮叮」都得！

今次UNDEFEATED開舖，就以籃球場做概念，有木裝飾木長凳，更大量用上黑色鐵網配襯，有強烈街頭感覺，店舖面積亦頗大，比東京舖寬敞得多，賣嘅潮鞋潮物當然亦更多。而為了隆重其事，今次UNDEFEATED亦特別帶來了新一對紫白色聯乘Nike Zoom Kobe Protro，據消息，呢對鞋可謂極度超限量，全香港就只有5對咁大把，不過佢嘅定價係「免費」，因為佢係唔會公開發售，而你只需要喺正式開幕日（7月21日），喺店中買夠$1000，就可以參加抽獎，有機會免費得到這神鞋！喺呢度我就先祝大家好運喇！

除咗KOBE，新張期間亦有一系列潮鞋炒鞋排隊在店中上架，包括YEEZY 500 Utility Black（7月22日）、Off-White X Nike Air Preto 2.0 Black（7月27日）、Off-White X Nike Zoom Fly Mercurial Flyknit（7月28日）及NIKE REACT ELEMENT 87（8月4日），實行用潮鞋帶旺個場。當然喇，除鞋之外，新店亦右一系列服飾，好似A BATHING APE X UNDEFEATED 2018系列（7月21日），和UNDEFEATED自家系列等，都會全部上架，潮人們以後又多個排隊同洗錢嘅地方喇。
