【動漫節2018】排隊搶Hot Toys 6款會場預訂figure

Hot Toys是每年動漫電玩節的注目攤位，今年將有28個展覽攤位，佔地約2,700平方呎，將展示最新Figure產品。除之前介紹過的會場版figure，還有最新公佈的6款會場率先預訂figure，包括：Iron Man Mark L武器配件裝、War Machine Mark IV、Iron Man Mark VII、遊戲版蜘蛛俠、Darth Maul連載具套裝和1:4黑武士，咁多正嘢，到時又要排隊搶訂。

1.Iron Man Mark L 1:6 figure配件套裝
在《復仇者聯盟3：無限之戰》電影登場的Iron Man Mark L，以納米技術隨時變換出不同武器，今次的武器配件套裝是參照電影中變出的武器而製作，共9款武器配件，並有戰損面罩供替換。記住呀，個主體Figure係唔包架！
預訂價：$1,530（會場預訂贈送1:6戰損頭盔面罩）

2.War Machine Mark IV 1:6 figure
《復仇者聯盟3：無限之戰》中的War Machine Mark IV，有着多種重型武裝，例如一把可動的外置式重型機槍、一對隱藏於背部的火箭炮、兩組可裝拆臂膀導彈炮組件和打開飛彈的前臂甲等，當然唔少得一個全新塗裝的James Rhodes頭雕啦！
預訂價：$2,630（會場預訂贈送1:6比例Outrider人偶情景地台）

3.蜘蛛俠Advanced Suit 1:6 figure
來自9月推出的PS4遊戲《Marvel’s Spider-Man》的主要服裝造型，緊身戰衣以高質彈性布材縫製，並採用滴膠和絲印技術呈現蜘蛛圖案和網線等細節。配件有3款不同功能的蜘蛛絲發射器組件、蜘蛛無人機、禿鷹干擾器、手機、炸彈及地雷等，係爭在無頭雕箒。
預訂價：$1,500（會場預訂贈送1:6戰損頭盔面罩）

4.Darth Maul連Sith Speeder 1:6 figure
《星球大戰前傳：魅影危機》Darth Maul連Sith Speeder套裝，附有嚴肅和怒吼兩款頭雕，同時具備可動眼球，營造不同神情。Figure附有一把可結合右手手臂內置LED發光的雙刃紅色光劍，可換上揮動形態的紅色光劍組件。套裝版加配專用的Sith Speeder載具，車頭與噴射器位置有LED燈。
預訂價：載具套裝$2,580/淨figure $1,780（會場預訂贈送Darth Maul仿全息影像迷你人偶）

5.黑武士1:4 figure
取材自電影《星球大戰VI：武士復仇》的1:4黑武士figure，高逾50cm，具備30個可動關節。黑武士的頭雕與頭盔合而為一，揭開頭盔上的面罩後，可看到頭盔内部細緻複雜的機件，還有黑武士佈滿傷疤的真面目。黑武士胸前的維生裝置器和腰帶上的兩個控制盒都內置LED燈效，而且閃爍方式是按電影設定而製作，完全滿足粉絲。
預訂價：$3,680（會場預訂贈送LED發光珍藏頭盔支架）

5.Iron Man Mark VII 1:6 figure
會場優先預訂還有一款《復仇者聯盟》的Iron Man Mark VII 1:6 figure，目前資料及售價待定。此Figure同時於網上接受登記以預留購買資格。

動漫電玩節日期：7月27至31日
地點：香港會議展覽中心一號展館「Hot Toys展區：F01」
時間：上午10時正至晚上9時正
