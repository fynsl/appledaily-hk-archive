MyGut者言：十年雷曼幾番新 - 徐家健

雷曼兄弟破產十年，金融大鱷索羅斯曾斷言：「有效市場假說已經破產！」經濟學者十年窗下，今天我們對市場效率又有甚麼新的領會呢？
從次按危機到雷曼倒閉再到金融海嘯，本地評論多是道聽塗說。十年前我身在美國，身邊兩位同事的經歷值得與大家分享。
其中一位，當年剛好在白宮經濟委員會出任經濟師。據他的內幕消息，雷曼宣佈破產那天，整個經濟委員會的專家都不明白金融市場在發生甚麼事，在白宮忙了大半天，最終要請教金融業專家拆解火燒連環船所為何事。
另外一位，曾在聯儲局身居要職，他一向不滿自80年代起貨幣經濟學誤入歧途，培養了一整代漠視貨幣金融市場運作的宏觀經濟學者。這位前聯儲局要員雖未能預視金融海嘯，卻懂得在美股大跌後大手撈底，皆因他對現代貨幣政策救市信心十足。
女友Babe嬲嬲豬咁話：「有效拍拖市場假說已經破產！」我仲未識死：「有效市場大意係話股價充分反映市場訊息，但唔代表我哋可以預視危機。假如我哋有本事預視危機，咁股價反而冇效率喎。」Babe見我火上加油：「咁你預唔預視到我哋感情出現危機？」我死不悔改：「好似08金融危機，十年事後孔明我哋知道主要係兩大風險：一個係只知輸打贏要嘅交易員過度冒險，另一個係過份依賴歷史數據令投資者低估一些不顯眼事件嘅風險，而兩者最終導致金融機構系統性風險過高。」
Babe嬲到冇聲出，我唯有繼續我嘅有效拍拖市場新領會：「The course of true love never did run smooth。我唔敢再低估激嬲你嘅系統性風險，但我對我𠱁女嘅救市方式信心十足。」

徐家健
美國克林信大學經濟系副教授
