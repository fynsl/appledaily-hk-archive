醉駕吉澤瞳紀念碟被消失

日本組合Morning娘（晨女）的33歲前成員吉澤瞳本月6日早上涉醉駕撞傷兩名途人不顧而去，她被捕後以違反道路交通法及過失駕駛致傷罪名遭起訴，上周四以300萬日圓（約21萬港元）獲保釋，及至前日宣佈退出娛樂圈，結束其18年的演藝生涯。

吉澤瞳在大前日晚上離開警署後，隨即被送往埼玉縣一間醫院接受檢查及休養。由於吉澤瞳已跟事務所解除合約，故今後將無法回歸晨女行列，而今年8月舉行的20周年紀念騷「Hello！ Project 20th Anniversary！！ Hello！ Project 2018 SUMMER」會如期在12月12日推出DVD，據知產品將刪除所有吉澤瞳的鏡頭。
