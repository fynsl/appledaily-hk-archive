【玩物達人】80後回憶90後試玩　玩轉20款迷你PS懷舊遊戲

【玩物達人】
自從任天堂推出迷你紅白機大受好評後，其他廠商都效法推出復刻版的迷你主機，主要都是70後的童年集體回憶，數百元一部的相宜售價，令不少人願意買來收藏。今次，終於輪到80後最有回憶的主機復刻登場，是迷你版的PlayStation——PS Classic，內置20款遊戲，將於12月3日推出，已有不少人預訂。80後對PS的遊戲當然了如指掌，大部份遊戲都回憶滿載，但當時還未出世的90後對PS Classic又會有何感覺呢？

PlayStation於1994年推出，迷你版的PS Classic正是當年第一代元祖PlayStation的迷你版，機身的外觀設計以至包裝盒都依照當年PlayStation一樣，只是全部縮小了，80後的朋友對這部機應該不會陌生，但對於90後未出世的人又如何呢？我們就找來今年只有19歲的Gadget Girl小悠來試玩這部PS Classic，看看她有何感想。

由於是傳媒率先試玩，所以時間有限，未能玩齊全部20個遊戲，只能選擇其中一部份來試玩。小悠首先試玩的是兩款格鬥遊戲《鐵拳3》和《鬥神傳》，由於《鐵拳》後續還有很多集，所以連不太玩格鬥遊戲的她都知道《鐵拳》的名氣，她進入遊戲看到角色才十多人可選用時，有點驚訝說：「角色那麼少！」因為後期的《鐵拳》及其他格鬥遊戲，一般都有數十名角色可用，而實際試玩過後，她讚嘆：「以當年的畫質而言實在很好，還以為會像《盜墓者羅拉》般會有起角的胸部，而且動作也算流暢。」不過到她試玩《鬥神傳》時則笑言： 「角色都起角的呢！很懷舊！」

舊式操作沒有「蘑菇」　保留讀碟等待畫面

PS Classic內置了三大經典系列的大作《BIO HAZARD DIRECTOR'S CUT》、《FINAL FANTASY VII》和《METAL GEAR SOLID》，小悠就對《BIO HAZARD》比較有興趣而試玩了。對於當年《BIO HAZARD》的開機和選擇難易度的畫面，她表示很有型，不過遊戲開始時她連行走也有點不順，「不太習慣舊式的操作，而且沒有了兩個蘑菇（analog stick），有點難操作。」的確，由於年代太久遠，現在大家早已習慣了analog stick的操作方法，所以就連筆者試玩也不太適應。不過，對於遊戲開門進入房間的讀碟等待畫面還是很有印象，想不到連這個也有保留下來。

小悠也有試玩了《I.Q Intelligent Qube》、《Mr. DRILLER》、《GRADIUS外傳》和《R4 RIDGE RACER TYPE 4》等遊戲，她表示對考智力的《I.Q》和舊式的橫向射擊遊戲《GRADIUS外傳》都很不在行，連第一關也過不了，而對《R4 RIDGE RACER TYPE 4》則覺得很有速度感和不難操作。然而她最喜歡的是《Mr. DRILLER》，「這款遊戲1999年推出，跟我同一年出世呢！遊戲很容易上手，有點像現今的手機遊戲。」

PS Classic內置即時儲存功能，只要在某款遊戲中，按下機身的reset鍵就會離開遊戲回到選擇遊戲的版面，這時就會問玩家要不要儲存進度，對先前玩的遊戲作即時儲存，非常方便，但要留意即時儲存只有一個。所以，要儲存遊戲進度，還是得在遊戲中按正常方法儲存遊戲進度，而儲存則會記錄在虛擬記憶卡中，可以在選擇遊戲版面查看。

PS Classic的20款遊戲，港行主機和日版是同一樣的，而歐美版的遊戲名單則有數款是不同的。不過，名單公佈時曾惹來網民爭議，指很多遊戲不夠經典，爭相指出欠缺的遊戲。其實大家都希望自己喜歡的遊戲能納入其中，而Gadget Guy們亦有幾款PS遊戲希望PS Classic會推出的，包括經典賽車遊戲《Grand Turismo》、格鬥版《三國無雙》。雖然PS Classic內置的遊戲已成定局，但大家又有甚麼滄海遺珠想再重新玩呢？
