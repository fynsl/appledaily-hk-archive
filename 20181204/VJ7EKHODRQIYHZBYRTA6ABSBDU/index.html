兩朝民政副局患癌　許曉暉44歲病逝

兩朝民政副局患癌　
許曉暉44歲病逝

【英年早逝】

【本報訊】民政事務局前副局長許曉暉昨因病逝世，終年44歲，消息稱她因癌病去世，有傳她患乳癌。出身商界的許擔任民政局副局長九年，經歷兩任特首，2012年時任特首梁振英提出成立文化局，曾傳出她「坐正」升任文化局長的消息，惟該局最終未能成立。多名與她共事的政界人物都對其離世感可惜，醫管局前主席胡定旭形容許是「a true lady（真淑女）」，讚揚對方面對大事仍處之泰然。

記者：梁穎妍

許曉暉去年卸任民政局副局長後，仍有出任不同公司董事，包括今年3月獲准出任故宮文化博物館有限公司董事。她今年1月曾現身為參選港島區補選的新民黨陳家珮站台，許當時較做副局長時消瘦，但仍相當精神，所以昨突傳出許離世的消息，令不少政界中人感愕然。有政圈中人向本報證實許因急病離世，另有消息指許患癌病去世，有傳是乳癌。

幾名最近曾見過許曉暉的建制派人士均指，許近半年突然消瘦，當時許沒透露患病，其中一人引述許笑稱可能長期食齋致瘦，上周曾與許會面的政界人士也指，許當時精神未見異樣，對其突然離世感震驚。

特首林鄭月娥發聲明對許曉暉辭世表示哀悼，並向她的家人致以深切慰問，讚揚她擔任民政事務局副局長期間，致力推動公民教育、文化、康樂及體育工作。民政事務局局長劉江華讚揚許在民政事務局任職期間貢獻良多，民政事務局上下仝人對她離世感到極度悲痛。

胡定旭：a true lady

身在英國的醫管局前主席胡定旭表示，與許相識十幾年，形容許是「a true lady」，讚揚對方對家及朋友都非常好、做事認真，「無論幾大件事，都可以處之泰然，令周圍嘅人都好放心」。胡又憶述兩、三星期前相約許飯敍，但自己因臨時有事要到北京，未能出席，非常遺憾，對許突離世感難過。

陳家佩則表示，過往與對方多在區議會公事上接觸，認為許為人友善，「幫到佢都幫」，感激對方卸任後都走出來為她助選。陳指年初已見許比以往消瘦，但不知道對方患病，認為許尚年輕，對她突然離世感到很可惜，「雖然佢冇坐正，但都仲有好多機會，佢對社會都好熱心」。

許2008年加入政府出任民政局副局長，屬首批政治問責副局長，由曾蔭權政府過渡至梁班子時更曾有望「坐正」，盛傳被梁看中出任當時擬增設的文化局長，當時在社會引起很大迴響，質疑她無文化背景，惟最後政府架構重組方案被立法會拉倒，許繼續擔任民政局副局長，至去年離開「熱廚房」。
