山遊記：大帽山蘊藏打卡旅程山系閨密

山遊記：大帽山蘊藏打卡旅程
山系閨密

秋高氣爽，登上大帽山，一切都很美，如畫芒草、無敵山景、醉人夕陽、連石屋旁的枯樹別有味道；兩位90後體壇閨密王蘊妮（Mimi）及鄧伊程（Tiffany），周日有街不逛，有戲不看，選擇行山，結果驚喜處處，在大自然環抱下過足打卡癮。
記者：徐嘉華
攝影：陳海威、徐嘉華

今年25歲的Mimi是亞運速度滾軸溜冰代表，19歲的Tiffany專項跳高，原本風馬牛不相及，兩、三年前一個運動品牌拍攝活動，促成二人成為閨密，Mimi說：「我們一相識就很投契，天天互傳短訊無所不談，有次我等了的士很久也whatsapp告訴她，我的一切她最清楚。」

這邊廂的Tiffany不甘示弱：「我們笑點好低，很多事都一起大笑。」二人結伴同行，這是第一次，去全港最高的大帽山（海拔957米）也是第一次，從早上11時半在荃灣港鐵站集合，由川龍端記茶樓起步到大帽山，到晚上六點回程，差不多六小時，發掘了10個「打卡」靚景。

1.全自助得意@川龍端記茶樓

路線：約5.24公里，車程15分鐘（從荃灣港鐵站乘的士到川龍的端記，收費約六十元，下車後沿小路入，五分鐘內即到）

check point：從1993年黃秋生的《八仙飯店之人肉叉燒包》到2015年楊千嬅的《五個小孩的校長》，都曾在端記茶樓取景，這間開業超過五十年的茶樓特色是全自助，由拿碗筷、選茶葉泡茶、拿點心都要自己一手包辦，雖然當天嚐不到最出名的即摘西洋菜，但小白菜亦十分美味。

二人評：「吃飽行山更精神，端記的自助經驗好新鮮；更重要是好吃，推介食物是燒賣、粥類、豆腐花。」

2.約瑟爹利互動@大欖郊野公園牌坊

路線：約1.5公里，步行半小時，車程10分鐘（離開端記，沿荃錦公路及扶輪公園自然教育徑）

check point：大欖郊野公園牌坊的吸引力不及狗狗。

Mimi評：「如跟家人行山，一定會帶家中的約瑟爹利同行，今次竟然碰到同種狗，怎會不自拍留念？」

3.蓮姐人間有愛@大帽山茶水亭

路線：從大欖郊野公園牌坊步行約三分鐘

check point：來到大帽山茶水亭，亭內貼滿蓮姐跟發哥周潤發的合照，兩年前冰封大帽山，蓮姐不眠不休為被困山上的行山人士提供飲品及小食，今年七月更支持環保走塑，鼓勵市民自備水樽，是眾人打卡對象。

二人評：「蓮姐很nice，知道我們是運動員就送我們燒賣跟豆漿，環保概念也很好，蓮姐指此舉減少了八成水樽！」

4.與車同行@大帽山道

路線：茶水亭往東南方向至大帽山道，有少許暗斜上，約1.9公里，步行38分鐘

check point：人車同行。

二人評：「以前去過的路線，大部份都是山路，大帽山這裏有不少車路，見到很多人都是沿大帽山道走上去，要邊走邊注意安全。」

5.我唔夠高@淒美石屋

路線：從大帽山道路牌至石屋，約1.7公里，步行33分鐘

check point：香港為殖民地早期時，大帽山有不少軍營、看守屋及瞭望台等，這小石屋是廢棄軍營，但屋外枯樹襯托，有點淒美。

Tiffany評：「Mimi不夠高，我即時抱起她看，但空無一物，破爛的外牆滲出陣陣歷史味道。」

6.芒草靚到呢@觀景台

路線：石屋附近就是觀景台，步行約十分鐘

check point：芒草配美少女，美不勝收。

二人評：「假日喜歡到多點綠色的地方，這些芒草令整個平原生色不少。」

7.遠眺山景唔跳唔得@觀景台

路線：大帽山觀景台

check point：大片草原，加上「大地在我腳下」的山景，遠望林村、錦上路及元朗等，Mimi及Tiffany忍不住手舞足蹈，跳個痛快。

二人評：「終於感覺到大帽山的高了，覺得自己高高在上，跟天空的距離從未如此近過。」

8.霧中波波@雷達站

路線：大帽山觀景台至天氣雷達站的「白波波」，約2.9公里，步行約一小時

check point：大帽山另一個標誌就是「白波波」，如果天氣好，從大帽山道上已可清楚看到兩個「白波波」在頂𥧌上，但當天遇大霧，「波波」時見時不見。

二人評：「今次的終點本是『白波波』，可惜遇上大霧，所以只可遠觀，今次去不到，下次再來吧。」

9.黃昏景@觀景台

地點：石屋對面觀景台

check point：黃昏景致一向受都市人追捧，「鹹蛋黃」配合四周樹景，美得像畫，難得大帽山觀景台夠開揚，美景盡入眼簾。

二人評：「太陽日落西山的景色，顏色美到像幅畫，不枉此行。」

10.遇尼泊爾人大食會@石屋草地

地點：石屋對面草地

check point：採訪當天遇上十多名尼泊爾人野餐，原來其中一人在香港開餐館，有人經過，都會邀請他們品嚐美食，另一種人間有情。

二人評：「走到黃昏五時多已有點餓，難得這群尼泊爾人十分好客，主動叫我們吃東西，很有愛！」

後記：

站在香港最高𥧌的大帽山，兩位體壇甜姐兒也期望自己的運動事業起飛，Mimi希望後年的速度滾軸溜冰亞錦賽能有好成績，今年浸大一年級的Tiffany就期望跳高突破1米80，明年能與師姐楊文蔚一起參加意大利世大運。
