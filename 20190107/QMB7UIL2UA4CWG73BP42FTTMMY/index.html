網站眾籌 - 左丁山

特朗普入主白宮後，最著名、廣為人知嘅一項每日必做事情，就係睇電視，而且只係睇霍士電視，朝早睇Fox and Friends當作白宮官員簡報新聞，晚上睇霍士talk show，最鍾意Sean Hannity。霍士電視台係特朗普之傳媒護身符，似足央視為習近平服務，世界上一間電視台與總統關係如此親密，實在罕有，於是美國有人講霍士係State Run TV（官營電視），亦有人惡搞，話美國政府係TV Run State（電視統領政府）。

特朗普罵CNN、紐約時報、華盛頓郵報及其他主流傳媒不遺餘力，有恃無恐，得罪人唔駛本，原因好簡單，有全國收視率最高之收費電視台（Cable TV）撐腰，怕乜同CNN鬥過？日日講大話，都有Hannity同佢護航也。

特區政府會唔會向特朗普學習，故意拉攏一個具影響力嘅傳媒呢？唔敢講，怕且識得學，但唔識得做，香港或者亦冇一個梅鐸咁嘅人物可以幫特區政府講好說話。但好明顯，一啲有心人係知道要搶奪傳媒陣地嘅。印媒方面，全力封殺蘋果，互聯網方面，傾全力發展網媒，希望減輕蘋果動新聞之威力。上一次李嘉誠基金會捐二百萬元畀中大一個兒童診治基金，動新聞嗰段捐款新聞有六七十萬點擊率，加上讀者在社交媒體廣傳，更加不得了，可見動新聞之影響力巨大，惹人妒忌。

不過網站好難經營，一個網站若非幕後老闆有大水喉，有大老闆政治願景，點樣可以聘請700人，淨係記者都過百，規模大過暢銷報紙呀。如果冇大水喉，屬兄弟班形式辦新聞網站，就認真辛苦咯，唯有出到眾籌與捐款呢張牌，希望香港還有有心人，肯支持意見獨立，評論不偏不倚嘅新聞網絡。大記者打電話嚟幾次追左丁山捐錢支持「眾新聞」，佢話：「蘋果加價至十元一份，唔該你繼續訂閱，另外，由楊健興、劉進圖、李月華等資深傳媒人辦嘅眾新聞網站，捱得好辛苦，唔該你捐錢支持，正徵求2000名訂戶，每位每月100元，好多人畀得起啫。」唉，同仁辦報自古不易，今日兄弟班辦新聞網站，同樣辛苦到極，左丁山見過楊健興（Chris Yeung）（前南華早報資深編輯）幾次，以佢咁好英文程度，不如去搵份工好過，駛乜咁辛苦，但Chris等係有抱負有獨立意見嘅資深新聞人，係要為香港新聞界出一分力，係好難得嘅，但面對一個有700名職員嘅網站，點競爭呢，唯有置諸死地而後生，見一步行一步。左丁山為表示敬意，於是寫張支票畀大記者轉交眾新聞。
