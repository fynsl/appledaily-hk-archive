【楊崢心導遊】歸隱田園的L’Auberge sur Les bois

【楊崢心導遊】
在倫敦、巴黎這些世界頂尖大都會打滾了超過十年後，巴黎的人氣餐廳 Les Déserteurs（創辦於2014年的時尚bistro，在巴黎的地位與Septime 不相伯仲）主廚兼老闆Chef Daniel Baratier，自兩年前寶貝女兒Jeanne出生後，為了給她一個更美好的成長空間，去年決定與拍檔Alexandre Céret分道揚鑣。毅然放棄巴黎的一切，攜同曾任米芝蓮三星L'Arpege與Astrance餐廳經理的太座Charlotte，一起回歸法國東南部著名旅遊勝地Annecy的大自然懷抱，共同建立前舖後居（地下與樓上）的安居樂業之所。

森林上的驛站

Daniel與Charlotte去年四月在Annecy 附近的Annecy Le Vieux覓得理想地點後，整個夏天兩家人總動員參與翻新改建工程。只是拆除地下室舊酒窖過百公斤的木地板和十多斤的螺絲，便用了一星期。付出了無數汗水與努力在日夜趕工下，只用了三個月的時間就把之前保守陳舊的設計，換上時尚簡潔的裝潢。

餐廳的名字L'Auberge sur Les bois 很有詩意，L'Auberge 是以前給馬車休息的驛站；Les bois則指森林，所以這個集bistro、fine dining、外賣服務的「森林上的驛站」，為整個村莊帶來煥然一新的氣息。家鄉在中部Saint -Étienne，自4歲起已在Annecy 生活的Daniel，對於能夠回歸成長地重拾初心，打從心底笑出來：「我覺得在這裏才是真正的生活。每天在大自然裏守護着太太與女兒的同時，又可以創作我喜歡的菜式。」Daniel會選用附近的有機地道食材，更愛落手落腳由麵包開始全部自家製作。閒時就鑽進廚房準備Bocaux（以玻璃瓶盛載的燉湯、肉醬、鵝肝醬等傳統菜式），方便村民外帶回家享用。這一切都是與別不同的構思，從心出發惠澤社群。

L'Auberge sur Les bois分為fine dining的Restaurant和吧枱間隔為主的Bar À Vin。一莊重一悠閒的氛圍以黑白森林印花牆紙貫徹其中，中央有一個玻璃貴賓室。全透明的簡約大氣設計，給人一種心曠神怡的感覺。三盞圓形的鳥籠照明燈，把柔和的光線反射在天花與長形合桃木餐枱上，清雅脫俗。

挑戰再苦也是甜

我抵達當日陽光普照，選了Bar À Vin最後排靠近Restaurant的位置。戶外的遠山近林盡收眼底，瞬間明白Charlotte與Daniel為何會捨離巴黎的五光十色。抬頭看吧枱上的環保餐牌，經理推介還未到午後一點已只剩最後一份的慢煮牛肋排（plat de cote）。上桌時，香味四溢的牛肋排肥而不膩，色澤紅亮，酥軟香糯，入口即化。伴吃的粗薯條炸得金黃鬆脆，不禁厚着面皮低聲地問了一句：「這裏會有ketchup嗎？」順帶一提，在巴黎若問餐廳拿ketchup 可是死罪一條，有機會再詳談。魁梧壯碩，滿手紋身的經理笑着說：「沒有啊，但有每日新鮮自家製的mushroom ketchup，可以嗎？」看着那深啡色的mushroom ketchup，原先還有些遲疑，一放入口，那酸中帶甜的滋味滲着濃郁的新鮮蘑菇清香，比起充滿化學物質的紅色版本好吃得多。其後與Daniel談起這可口的新發現，他即笑笑口說：「很容易弄的！只要把新鮮的巴黎蘑菇（白色的圓形蘑菇）攪拌後，加醋加糖慢煮，最後再攪拌均勻便大功告成了。」此時剛忙完招呼客人、落單、打點一切的太座Charlotte過來閒聊片刻：「只是試業了兩個多月，很高興附近各地的米芝蓮星級廚師們已親臨支持。」問明顯地消瘦了的伊人辛苦嗎？她臉上立刻露出燦爛的笑容：「雖然現在睡眠時間少了，要兼顧的工作多了，但我每天隨時隨地都可以看到女兒與丈夫。在辦公室辛勞工作的時候，一抬頭就可以看到風景如畫的景致，夫復何求？」

未來的歲月或許仍是挑戰重重，但能夠與摯愛一起奮鬥，再苦也是甜。
