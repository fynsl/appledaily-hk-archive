明愛易擋，暗戀難防 - 馮睎乾

明愛易擋，暗戀難防 - 馮睎乾

今天情人節，宜談戀愛，本欄便趁機談談，什麼是最理想的愛情？戀愛表面上千姿百態，但本質上跟特區政府高官差不多，就是摧毀你內心的平靜，破壞你生活的安寧。戰爭、地震和戀愛，一脈相承，都是不同程度的飛來橫禍。但戀愛是最快樂的兵荒馬亂，因為你墮入愛河的時候，通常也會喪失理智，變得比較蠢，於是也比較快樂。

戀愛很快樂，而暗戀又比相戀更快樂。相戀，是兩個人的戰爭；暗戀，是一個人的地震。戰爭有勝負，所以也有俘虜，主子的快樂是建築於奴隸的痛苦之上，只能說苦樂參半；但地震沒有勝負，只有生死，生還的很快樂，死掉的不會不快樂，所以地震只有純粹的快樂。兩人熱戀，總有感情冷卻的時候；但暗戀可以直到永遠，由現在到宇宙滅亡，甚至宇宙滅亡之後。人的心理就是這樣：越是得不到，越會覺得好。沒有結果的初戀，尚未開花的暗戀，都是一億元的空頭支票，因為從來沒有兌現，醜陋的真相也就不被發現，於是永遠美好，永遠期待。從這個角度看，最理想的愛情非暗戀莫屬。

暗戀雖然是最瘋狂的，但也最像得道，因為人在暗戀的時候，近乎忘我，沒有我執。茲威格在〈一個陌生女子的來信〉把這種狀態描述得很到位：「塵世間，沒什麼比得上一個孩子暗地裏不為人知的愛，因為它毫無希望，卑躬屈膝，俯仰由人，熱情洋溢。」（nichts auf Erden gleicht der unbemerkten Liebe eines Kindes aus dem Dunkel, weil sie so hoffnungslos, so dienend, so unterwürfig, so lauernd und leidenschaftlich ist）耶教的信望愛，佛家的人我空，都似乎能在暗戀者身上體現。

暗戀即使不能在愛情上修成正果，但能在智慧上覺悟成佛，也算是不幸中的大幸。但不管你是否成仙成佛，基本道德觀念還是要有的。作為有良知的暗戀人士，你若不直接告白，最好就一暗到底，千萬不要半明半暗，讓對方陷入兩難：到底該假裝不知，讓你繼續下沉，抑或冒着被你嘲笑「諗多咗」的風險，先發制人拒絕你？

沒有得到過的，從來不會失去。最刻骨銘心的愛情故事，是什麼也沒有發生的故事。在張愛玲〈愛〉，那個十五六歲穿月白衫子的女孩，在春天的晚上扶着桃樹，遇見了對門住的少年。他忽然走過來，站得不遠，輕輕說了一聲：「噢，你也在這裏嗎？」然後大家沒有再說什麼，「就這樣就完了」。不愧為祖師奶奶，這就是愛。茲威格和張愛玲都深深明白：愛是永恆，當所愛唔識你。
