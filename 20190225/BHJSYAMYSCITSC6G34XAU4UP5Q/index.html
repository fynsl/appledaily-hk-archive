華排隊黨掃EVA模型　日人不滿

哪裏有炒賣，那裏就有大排長龍的強國人！日本人氣經典動漫《新世紀福音戰士》金屬製模型Metal Build EVA初號機前日在日本發售，吸引大批粉絲在東京秋葉原排隊搶購。但有不少日本動漫迷指，大批來自中國的排隊黨佔領商店，不僅插隊，還將初號機搶購一空，令他們撲空。有網友更拍到中國排隊黨將「戰利品」塞滿客貨車。

肆意插隊　喝罵店員

秋葉原的發售點前日一早已有大批粉絲排隊。有日本動漫迷昨在Twitter發文指今次與以往不同，長龍中有大量中國人，不少人更肆無忌憚插隊，但他們聽不懂日文，當店員詢問他們排隊買甚麼商品時也不懂回應，無法拿號碼牌。他們為此發怒，包圍店員並大聲恐嚇，甚至出手搶去店員手上的號碼牌，後來保安人員到場，雙方展開罵戰。

正式開賣時，中國排隊黨不斷搶購初號機，之後將貨物交予路邊的接頭人並放到客貨車上。有網友拍到他們把一袋袋的戰利品塞滿客貨車的情景。有日本動漫迷到店外排隊但買不到初號機，十分不滿，埋怨排隊黨囤積模型再轉賣到中國圖利。Metal Build EVA初號機定價約為2.3萬日圓（約1,631港元），但市場反應熱烈，網上炒價達逾2,500港元。此款模型與以往的製品不同，是以雕刻細緻、可動性高為賣點的合金玩具。
台灣《蘋果日報》
