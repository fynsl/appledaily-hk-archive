投資客掃Island Residence 三伙

【本報訊】樓市回穩，新盤再現投資客掃貨。會德豐（020）筲箕灣現樓新盤Island Residence早前推出三年成交期付款計劃吸客，昨日錄得3伙開放式戶成交，據悉由同一組買家大手購入，合共套現2,417萬元，平均呎價約2.9萬元水平。

市場消息指，成交的3伙單位位於Island Residence 29至31樓D室，實用面積同為272方呎，開放式間隔。

該買家選用早前新增的三年成交期「置業攻略1088升級版付款計劃」，於簽約時付上5%定金，一個月後加付樓價5%，於1,088天後才需要付餘下90%樓價。單位成交價分別為798.5萬至812.8萬元，呎價介乎29,357元至29,882元，共套現2,417萬元。以上資料以一手成交紀錄冊作準。

事實上，日前信置（083）觀塘凱滙亦錄得一宗大手客入市個案，單位位於3座35樓至37樓F室，實用面積909方呎，三房套房間隔，分別以1,903.5萬至1,918.9萬元成交，呎價20,941元至21,010元。

另外，會德豐將軍澳南入伙盤Monterey昨天招標售4間洋房，包括「Marina Blue」7號屋以及「Lagoona Blue」A號、E號及G號屋，平均呎價28,463元，套現逾2.14億元。其中7號屋實用面積2,004方呎，連874方呎花園、707方呎天台、47方呎露台及36方呎梯屋，售價6,352.7萬元，呎價31,700元。

海日灣招標沽三伙

大埔白石角再爆新盤戰，億京海日灣昨招標沽3伙，共套現3,656.2萬元。成交冊顯示，最貴一伙為Court C 1座17樓F室連天台特色戶，實用面積789方呎，三房套房間隔，成交價1,539.5萬元，呎價19,512元。另外，Court C 2座11樓B室，實用面積505方呎，兩房間隔，成交價766.3萬元，呎價15,174元。
