魑魅魍魎：銀行界科技軍備競賽 - 十一少

踏入金融科技Fintech年代，一眾銀行密鑼緊鼓推出新服務和功能，頻頻有新發佈，恍如一場軍備競賽，初步看來，外資銀行與中資銀行比較有誠意，在這場軍備競賽叮噹馬頭。

過往銀行將大量研發開支投放於「硬實力」之上，例如替「炒房」交易員編寫程式交易、高頻交易、衍生工具及理財產品等，目的只有一個：賺更多錢。但時代變了，大deal難求，整個銀行業都更重視retail banking，加上一眾虛擬銀行來勢洶洶，搞「普惠金融」，即是做散戶生意越來越重要，所以銀行之間就要比拼「軟實力」，為用戶創造價值。

為先發制人，銀行為用戶推出一系列新服務，在數碼化、用戶體驗上花工夫，先比較一下幾間發鈔行，較量如何割據Fintech。

最積極的要數渣打，除盛傳將拿下虛擬銀行牌照，與李澤楷的香港電訊拍住上，最近推出的「無卡提款」也挺方便，讓用戶在App預設提款額，在櫃員機用QR code提款，縮減人均使用櫃員機的時間，是發展Fintech比較積極的銀行。

不過，近日內地發生一宗「女子排隊撳定QR Code 男子從後偷拍隔空盜竊」新聞，乍聽之下令人憂心，惟畢竟外資科技不是國產貨，國產會把QR Code放在用家手機，相反這個服務是把QR Code放在櫃員機上，要登入才能掃描，安全得多。有些事情，反過來處理就可解決問題。

除了渣打，同為外資的滙豐則用創投形式投資Fintech居多，入股一系列初創公司，而自家製的Fintech項目仍是PayMe，推出兩年以來仍在做蝕錢的C2C生意，但身邊不乏朋友使用，最近才發力搞商用版。

中銀香港作為中資代表，有不少中港融合服務，如在香港開內地戶口，又大搞生物認證及跨行業與新世界搞Blockchain買家，算得上有做嘢，但摸不着頭腦的是，中銀櫃員機為何要用標楷體？

在互聯網金融世界，用戶數量比一切重要，儘管Fintech短期內未必能回本，銀行仍願意投放成本，發展金融科技不一定贏，但不發展就肯定會輸。
