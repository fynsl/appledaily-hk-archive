行車證將設QR code
電子牛肉乾6月試行　一嘟即抄

【司機愁爆】
【本報訊】警方近年積極打擊違例泊車，為提高抄牌時發出違例泊車定額罰款通知書（俗稱「牛肉乾」）的效率，消息稱警方最快今年6月起試行「電子牛肉乾」先導計劃，屆時警員使用狀似手機的數碼裝置，掃描違泊車輛行車證上的二維碼，可即場以便攜式打印機發出電子牛肉乾。惟有職業司機擔心在新方法下更易中招，大呼無奈。
記者：黃麗英　翁鈺輝

警隊電子牛肉乾的樣本及抄牌設備近日曝光，消息稱警方五大總區交通督導員及警員，及大埔警區一隊SD team首先試行，會用一部狀似手機的數碼裝置及一部隨身打印機外出，因為6月起運輸署發出的車輛行車證將首次印有二維碼（QR CODE），屆時警員使用該部數碼裝置，掃描行車證上的二維碼，電子告票便有車輛資料，而抄牌人員只需要輸入日期、時間、地點及違例事項便可。

即場打印　減少「寫錯告票」

警員會即場以便攜式打印機，列印出電子牛肉乾。輸入裝置的資料會上傳至警署的電腦伺服器作進一步自動處理。警方消息又稱，該手機可最多拍攝五張相片，所以不用畫圖，初步是不用落記事冊，上庭亦不用給口供，資料將提交法庭。

根據警方數字，去年共發出近202.7萬張牛肉乾，較前年184萬張增加約一成，而全港五大警察總區之中，以西九龍區發出的牛肉乾最多，去年達到51.8萬張，佔全港發出牛肉乾總數的四分一。消息稱，電子牛肉乾可大幅減省警員人手撰寫違泊告票的時間，亦可減少警員因寫錯車牌等資料導致告票無效的情況。至於警員發出電子牛肉乾時在現場所拍攝的相片，亦可作為證據。

不過，不少司機都擔心更易中招，貨Van司機王先生認為路邊泊位根本不足夠，即使遇上警方抄牌亦無可奈何。他不滿搵食車經常遭警方抄牌，自己偶然僅離開車輛上廁所片刻，即已被抄牌，批評警員做法鬼祟。

校巴司機廖先生稱，以往警員抄牌時，若司機及時返回或可以講「人情」，但新方法直接掃描行車證便較容易被抄牌，大感無奈。貨Van司機施先生認為新方法只是方便了警方，直斥政府「大把錢」，卻想盡辦法剝削市民。至於的士司機姚先生則支持警方加強執法，認為新的電子告票有阻嚇作用。
