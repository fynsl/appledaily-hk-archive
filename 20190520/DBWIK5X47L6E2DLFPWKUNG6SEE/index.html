朝藉河內北韓餐廳售軟件賺外滙

北韓被指利用聯合國制裁漏洞，以越南河內市一間北韓餐廳為幌子，實是出售高科技面容識別系統軟件，賺取外滙。

藉制裁漏洞賣面容識別系統

兩個美國智庫——高級國防研究中心（C4ADS）和詹姆斯‧馬丁防核擴散研究中心（CNS）發現，一間名叫Future Tech Group的軟件公司，跟那河內北韓餐廳和一間名為Glocom的神秘馬來西亞公司有關連。聯合國早已指北韓利用Glocom非法售武。Future Tech Group和Glocom的網站共享一個IP地址，雖然Future Tech Group現已移除網頁，但在緩存版本中，可見該公司出售面容識別產品並設計計算技術，包括軟件。

至於Future Tech Group與河內北韓餐廳的關係，根據越南商業登記，姓名譯音金正吉的北韓男子在越南擁有一間餐飲公司，地址正是該北韓餐廳。金正吉透過多個在網上賣廣告出口軟件的自由身匿名人士賬戶，跟Future Tech Group扯上關係。C4ADS分析員阿特伯恩說：「我們看到的是，有機構東主以該餐廳作登記地址，同時跟一些向全球出售先進面容識別技術的自由身人士賬戶有關。」

聯合國制裁禁止北韓向海外售武，但非軍事用途中的高科技軟件是否列入禁運名單就不清楚。面容識別系統軟件或提供一個漏洞，讓北韓賺外滙。阿特伯恩說：「那些在國外工作的程式員，每年可為北韓政府賺取數以十萬美元計外滙。」
