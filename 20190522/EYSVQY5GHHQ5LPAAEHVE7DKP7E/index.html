摧毀旅遊點 - 左丁山

在中環常有夜生活的朋友M傳來英文免費飲食報、網站及「明報周刊」的報導，講及位於中環怡和大廈（很多人還是繼續稱呼之為康樂大廈）地庫的Grappa's Cellar將於今年七月，租約期滿，結束營業。一位女士Karen Tong（據說是女律師）感到非常遺憾，在網上發動簽名，向業主置地集團陳情、請願，懇請置地不要一手摧毀已營業20年的Grappa's Cellar，已有數千人簽名。

左丁山很久已沒有光顧這意大利餐廳，原因很簡單──唔就腳，康樂大廈在1970年代初落成，當時是香港最摩登、最高的大廈，有「千窗大廈」之稱，左丁山曾在此大廈工作六年，對大廈十分熟悉，後來改名為怡和大廈，漸失光芒，晚上成為孤島，地下及地牢的餐廳根本人客寥寥可數。Grappa's的前身是文華酒店經營的The Galley，走中高檔，在90年代末，支撐不住，關門大吉，原址丟空了一段時間，1998年股市大跌，市面蕭條，置地集團找到餐飲M，給予長免租期（裝修）及優惠條件，於是餐飲M租下來，經營Grappa's Cellar，起初午市生意奇佳，但晚上和假期一如中環其他地方，十枱九空，餐飲M與謀臣苦思之下，想出辦法，就是引入現場表演爵士樂隊，設個人棟篤笑表演，不收入場費，只求推銷餐飲，這原是1950年代60年代的夜總會方式（例如夏蕙夜總會），但在21世紀局部復活，竟然一擊即中，十多年前香港棟篤笑天王黃子華也曾到Grappa's表演，利用小場地小眾來測試他的「獨白反應」。香港在回歸以後，外國居民的活動空間逐漸減少，比較大的餐飲場地，能夠容納近三百名觀眾食客，現場live band可以嘈到坼天而又不會引起附近居民投訴噪音的地方，絕無僅有，怡和大廈（如中環孤島）的地庫（天然隔音）就是此類活動的最佳場地。

近幾年餐飲M的兒子留學英國回來，負責每星期的現場節目策劃，廣邀本地爵士樂隊、外國棟篤笑藝員到場表演，大收旺場之效，令Grappa's成為外語界及留洋派客人的夜生活勝地，上述的Karen Tong女士原來在星期三晚在Grappa's搞swing dance舞會，播50、60年前的英文金曲，給男女顧客跳舞，吸引大批擁躉。上星期四有馬來西亞棟篤笑Harmful If Swallowed表演，入場費350元，入場券一早銷清，適逢當日中午左丁山獨自一人，故意走到Grappa's吃午飯，經理云：「馬來西亞有一群fans參加旅行團來港，團費四千元，包括一晚酒店、一晚Grappa's現場表演及機票，這批馬來西亞客已佔了入場人士七成以上！」不說不知，Grappa's已成外來遊客蒲點，置地摧毀之，可惜也。
