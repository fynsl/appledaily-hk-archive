Note 10月底開售「fing筆」遙控手機

【本報訊】三星前日於美國紐約舉行發佈會，兩部Galaxy Note 10系列手機正式登場。新機特點在於螢幕佔比較上一代高，手寫筆S Pen亦更強大，可感應不同動作，用家「fing筆」可遙控更多相機和其他功能。兩款新機皆設LTE及5G版本，本月23日開售，售價949美元（約7,450港元）起，惟香港的銷售詳情尚未公佈。

曲面螢幕　接近無邊界

新的Galaxy Note 10和10+的螢幕佔比較上一代有明顯改變，增加至90%，尺寸分別為6.3和6.8吋，上下和兩側邊緣窄小，加上是曲面螢幕，塑造出近乎無邊界感覺。三星亦為新機外觀花了不少心思，如加入Aura Glow漸變色，其天藍色機底會在不同角度透出彩虹亮光，增強視覺效果，前置鏡頭亦由右上角移到頂部中央。

S Pen作為Galaxy Note系列特色，新機當然亦有新功能，其遙控距離不但增至十米，更加入了陀螺儀，可感應更多動作，除文字辨識能力增強，亦可向相機發出不同指令，如向左打圈鏡頭就會拉近，向右打圈鏡頭拉遠，操作感覺順暢。S Pen未來會開放予程式開發者，或為用家帶來更多遊戲體驗。

Note 10及Note 10+分別設有三個及四個後置鏡頭，拍片方面，超級穩定模式擴展至伸縮超廣角鏡和縮時攝影都可使用，以往相機才有的Live Focus亦支援拍片，可控制景深效果和添加特效。

此外，發佈會同時亦介紹了新筆記簿電腦Galaxy Book S，配備13.3 吋螢幕，重960克，充電後可連續播片23小時，更設nano SIM卡槽，支援流動數據。會上亦介紹上周發佈的Galaxy Watch Active 2智能手錶及Galaxy Tab S6。
