C AllStar成員Jase曬太陽幫女友捽背

C AllStar成員Jase曬太陽
幫女友捽背

【淺水灣】本地四人男子組合C AllStar前年完成紅館騷後，宣佈休團，自稱「拉麵哥」的成員何建曦（Jase）就轉做飲食KOL，在社交平台寫食評，以及嘗試做作家及編劇工作。Jase工作量不算多，但就有更多時間陪女友。日前，《蘋果》目擊Jase與女友拍拖去沙灘曬太陽，享受慢活的時光。

當日下午4時左右，Jase一個人現身淺水灣泳灘附近，當時他正獨自坐着玩手機，原來在等女友，大約幾分鐘後，就見其女友出現，兩人各自拿着一個環保袋，然後走進超級市場購買飲品。穿上短衫短褲，一身夏威夷look的Jase，與女友站在放飲品的凍櫃前左揀右揀，研究買甚麼飲品，最終女友選了一大盒椰子水，他就買了一樽疑似酒精類飲品，二人買完飲品後，便一齊步行去沙灘。

搵位落腳

起初，Jase與女友一先一後行，直至行到沙灘找啱心水位置落腳，兩人便合力打開沙灘墊，準備曬太陽。體貼的Jase坐定後，第一時間拿太陽油幫女友捽背。Jase的女友當日穿上黑色一件頭泳衣，最初穿了一條短褲配襯，但躺下曬了一會，把短褲脫下，見她拿着褲子和Jase研究，之後再用毛巾蓋臉瞓低曬太陽，但粉紅色鏡片的太陽眼鏡就依然戴着，而Jase似乎未及女友放鬆，不時四圍望當女友貼身保鑣。
