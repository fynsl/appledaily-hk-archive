中大Sonia：堅持活着作戰下去

【還政於民】
【本報訊】不論是上周四在中文大學的師生對話會中，還是在採訪當日，穿着同一件印上「we have to be very strong, if we want to do something‘very wrong’」黑底白字T恤的吳傲雪（Sonia）依然一臉倔強，「當其他人都指摘我哋嘅時候，我哋要更堅持，相信自己做緊正確嘅事。」只是，自脫下口罩指控遭受警方性暴力後，恐嚇電話和惡意中傷的網絡帖文至今未停。24歲的她坦言「好驚」，但無悔當日決定，「係香港人令我決定要同佢哋繼續並肩作戰。」在銀包中保存住一隻寫上「加油捱下去，努力別放棄」的紙鶴的她說。
記者：陳芷昕

中大女生Sonia首次現身，其實是於9月4日，在立法會議員毛孟靜陪同下，以「龍小姐」的名義示人。身為8.31太子站的三名「紅色傷者」之一，Sonia自9月2日從新屋嶺保釋出來後，得悉當日在太子站的傳聞惹起滿城風雨，她希望能以目擊者身份站出來發聲。「你問我有冇死到人，我唔知。但十個變七個，政府部門話因為移動咗傷者位置，呢個說法我唔接受。作為嚴重傷者，當時我哋移動範圍都係10步以內，而且係救護移動。」當時，穿得一身黑、戴上墨鏡和口罩的她在記者會正式開始前，問議員助理能不能給她變聲器，可惜時間倉促準備不及，她惟有自行變聲。

不停受內地來電滋擾

第二次在民間記者會自稱「S傷者」，和第三次在9.27新屋嶺人權集會中自稱「S同學」的Sonia依然蒙住臉。終於在上周四，她脫下口罩實名控訴，香港人都知道她叫「吳傲雪」。

Sonia知道公開身份將承受很大風險，但這不是一時衝動的決定。「我好清楚我企出嚟係返唔到轉頭，唔係細路仔玩泥沙，企出嚟彈返入去係唔得。」

9.27集會當日，Sonia在現場得悉原來有其他被捕人士比她承受更嚴重的性暴力，甚至涉及性侵犯及虐待，讓她決定要鼓起勇氣走多一步，「已經係有人承受住無法逆轉嘅傷痛，但佢哋都仲捱緊落去。我就不斷問我自己：我吳傲雪可以為呢班人做到咩呢？」

不只是性暴力受害者，還有這四個月來，被黑勢力斬斷腳筋的少年、爆眼少女和印尼記者、肺部中彈的中學生——Sonia希望能為他們發聲，「啲人成日話我哋隱藏係口罩背後，講嘅都唔知係咪真，演員嚟嘅，咁我𠵱家咪企出嚟實名指控。」

只是，自上周四起，Sonia就不斷面對着滋擾、惡意中傷和恐嚇。每分鐘她都收到內地電話，更有人傳來短信問她「幾錢一晚」，亦有人在網絡討論區上造謠指她做援交。對此，她一臉無所謂的說：「呢啲中傷預咗啦，比我想像中少㖭。中國有13億人，𠵱家都係幾百個電話啫。」她亦不把這些人放在眼內，「因為佢接受唔到警方所作所為，所以寧願指控我講大話。」

縱使說得雲淡風輕，24歲的Sonia還是對這一切感到不知所措。自9月2日保釋出來至今，她一直未能好好入睡，因為太子站和新屋嶺的一切依然瀝瀝在目。

9月3日，Sonia乘地鐵經過太子站時，突然渾身發抖、不能呼吸，只能一直半蹲在閘門附近，不能站直；一直至列車駛離太子站，她才回復正常。9月4日，她到旺角與朋友聚餐，怎料一到埗便在餐廳門口遇上防暴，讓她嚇得心驚膽顫。

這份壓力在Sonia實名指控後更嚴重，她說：「好多人嘅時候就會驚，驚會有人想對我不利，仲要𠵱家警方咁主動，會唔會又捉我去葵涌警署定新屋嶺，又多輪性暴力？」

「有一秒覺得自己好污糟」
