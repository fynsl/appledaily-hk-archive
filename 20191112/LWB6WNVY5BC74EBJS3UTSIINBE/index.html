【add名人時尚】減齡眼雙修

眼周肌膚厚度少於一毫米，相較其他部位更脆弱。加上不良的生活習慣及環境、頻繁的活動，如長時間對着電腦、手機、熬夜等，令眼周肌膚更早出現衰老迹象。想保持眼周肌膚緊致、亮麗的年輕狀態，除了日常要勤加保養外，借助化妝小撇步也能令你瞬間減齡。

滋潤為上

眼周肌膚少有皮脂腺，加上肌膚薄弱，鎖水力差，故此非常容易出現乾紋，看來更顯老。選擇眼霜時，應以滋潤度高為本，再針對細紋及黑眼圈集中修護，提升眼部自身屏障機能，而且用量不宜過多，每邊約4、5小米份量即可，用量太多不但不能吸收，反而會造成負擔，加速肌膚衰老。

眼周運動

塗抹眼霜時，以指腹或利用美容小工具為眼周肌肉做運動，可以促進眼部肌膚的循環，改善黑眼圈、水腫問題，令眼形輪廓更緊致。從內眼角經眼肚至眼尾，再從外眼角至上眼瞼，以打圈形式來回輕按，讓眼瞼肌肉得以放鬆，緩和黑眼圈和眼袋形成。

多活性脂肪酸氨基酸成份有助重新調整「生物時鐘」，可提高細胞活力，減少導致衰老的炎症問題，令眼周肌膚變得年輕、緊致。

蘊含松茸精華成份，含有豐富蛋白質及氨基酸，有助增強肌膚防禦力，降低因外部污染對眼周肌膚所造成的傷害。

含有32 種天然來源的成份，配合四倍神經胜啲技術，有助放鬆並減淡深刻的皺紋，岩薔薇及仙草成份可改善眼周肌膚保濕、堅實度、彈性及平滑度。

質地透薄，蘊含有水解植物蛋白，瞬間撫平肌膚，可以作為眼部化妝的基礎。乳蛋白成份有助提升眼部輪廓，減少浮腫和黑眼圈。

【逆齡小撇步】

Key 1.原生眉形

平順的原生眉形比幼眉感覺更年輕，使用眉粉從眉頭開始描畫至眼眉，再以眉筆加強眉頭及尾部。注意描畫眉𥧌要微向上揚，有效提升顯老下垂眼形。最後以眉蠟把眉毛向上刷，營造豐盈眉毛感覺更年輕。

Key 2.輕盈遮瑕

熟齡肌難免會出現細紋，特別是脆弱的眼周肌膚，要撫平細紋，切忌過度使用遮瑕，以免令乾紋更顯眼。宜選用質地輕盈，水感度高的遮瑕液，於眼尾位置順應紋理畫幾筆，以遮瑕掃輕力掃開，最後以指腹及掃輕力按壓，令遮瑕液更貼服。

Key 3. 自然鬈睫毛

自然鬈曲睫毛調整眼形更圓潤，可重點在中央加上睫毛液，眼頭及眼尾放輕，注意睫毛液也應以輕盈質感為上，隨着年紀越大，睫毛也會變得比較柔弱，過重的纖毛液反而會令睫毛下垂，更顯老態。

Key 4. 「小翅膀」線

年紀漸長，眼周肌肉鬆弛，下垂的眼瞼會令眼睛看來有「越來越細」及疲累。別以為畫上粗眼線就能解決問題，過粗眼線同樣令人顯老。不妨以「點」的方式填補睫毛間的縫隙，加強深邃度，再於眼尾1/3位置畫上少許上揚的「小翅膀」眼線，提拉眼形效果。

Key 5.打亮黃金三角

完成眼妝後，在眼下三角位置及外眼角的C zone 掃上光影，提亮後能營造骨膠原滿滿的膨脹感，能讓視覺上瞬間減齡。
