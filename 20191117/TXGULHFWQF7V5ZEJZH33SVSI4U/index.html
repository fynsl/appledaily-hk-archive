華為Mate X 5G
1分鐘賣清　炒價11萬

一機難求？華為首款屏幕摺芒機Mate X 5G前日網上開賣，官方價格高達16,999元人民幣（下同，約18,980港元）仍大受追捧，僅1分鐘即售罄。炒價亦因此水漲船高，有買家願加價兩萬收購，亦有賣家叫價近10萬（約11萬港元）一部，令網民驚呼：「搶到即賺到，利潤比蘋果高多了！」該手機將於明日及22日（周五）作第二、三輪銷售。

令一眾內地用家為之雀躍的Mate X，是華為推出的首款5G可摺叠屏幕手機。原定今年6月上市，但受美國對華為實施貿易禁令影響，最終兩度延期才推出。Mate X設計方面採用「外翻式」摺屏，未摺前是約6吋的手機螢幕，翻開變成8吋的流動平板。Mate X集合了目前最尖端技術，即使官方售價高達16,999元，仍然受大批用家追捧。在上周五早上10時在網上甫一開售便火速售罄，不少人大為失望：「30秒不到就沒了！」「搶不過黃牛……」

有泡沫成份　商戶拒囤貨

悉由於柔性屏幕生產難度大，Mate X 5G月產約5萬至10萬部。華為也採取限量發售模式，但未提供首輪貨量的數字，只指「備貨量不多」，暫不會在線下旗艦店和體驗店上架。一機難求下，網上炒價驚人，普遍2至3萬元以上，有賣家叫價至6萬多元。一家淘寶數碼店舖Mate X現貨價99,999元（約111,574港元），較原價貴4.2倍。

在深圳華強北專營手機生意的吳小姐向《蘋果》表示，Mate X 5G在市場上貨源極少，官網限售致外界瘋搶，現時華強北的機價比官網價貴7,000至8,000元（約7,800至8,900港元）。吳認為價錢有泡沫成份，因此不敢囤貨，也不建議現在入手Mate X 5G，認為最好過一段時間再買：「如果往後貨源充足的話，（售價）甚至比官網價更低。」

除華為以外，三星的摺叠屏幕手機Galaxy Fold近日也在內地發售，多個電商平台同樣搶空。市場人士分析稱「現在的折叠屏手機產品極具創新，對一些用戶具有吸引力和衝擊力。華為開售之後很快售罄，說明市場需求旺盛」。

不過，市場調研公司CSS Insight研究主管Ben Wood質疑，華為屏幕「外翻」的設計容易損壞，「這風險、過高價格及只在中國發售，讓我懷疑這只是華為實現可能性藝術的小眾產品，而不是一件可信的商品」。
