欠債千萬　咖啡店TOM N TOMS結業

【本報訊】攻港不足4年的韓國人氣連鎖咖啡店TOM N TOMS COFFEE，本月初突然全線結業。其在港註冊公司CAFE TOM N TOMS HONG KONG LIMITED的董事會，已於本月3日決議進行自動清盤。負責是次清盤的永聯會計師事務所臨時清盤人張鶴騫接受本報電話查詢時表示，近月商場生意難做，董事局決定「止血」清盤，公司現拖欠供應商、股東、業主等以千萬計欠款。

出售資產或不足還債

位於屯門市廣場的TOM N TOMS COFFEE正貼出有關CAFE TOM N TOMS正進行自動清盤的告示。該公司亦已於查冊中心上載自動清盤文件，文件中提到，經慎重考慮到公司的財政狀況及負債情況，公司業務未能持續經營，即使仍有顧客的定單尚未兌現，仍然需要清盤終止業務。被問到清盤原因，張鶴騫表示，公司董事局有感「香港近月商場做唔到生意」，決定清盤「止血」，不會找其他買家接手生意。張又指，本港五間TOM N TOMS COFFEE分店租約未到期，且全部錄得虧損，待出售資產後會把所得款項分發予供應商、股東等，惟相信款項不足以償還以千萬計的欠款。公司將於本月尾舉行債權人大會處理清盤後續事宜。至於受影響員工方面，大部份已獲發放薪金，員工需自行到勞工處索償餘下代通知金及假期等。

TOM N TOMS COFFEE於2016年3月攻港，首家分店位於沙田新城市廣場，不過，該分店連同觀塘apm的分店已於公司清盤前結業。最新在港餘下五間分店，分別位於尖沙嘴The ONE、屯門市廣場、銅鑼灣時代廣場、荃灣Citywalk及南昌V Walk。據TOM N TOMS COFFEE facebook官方專頁，公司於本月1日仍有發佈「Happy New Year」的帖子，更寫上「當然要係（喺）2020年重新開始，迎接新嘅一年啦！」，誰知兩日後就迎來清盤通知。有網民於最新帖子中留言指未兌現的團購券怎樣賠償，個別網民留言指有關團購平台表示會全數退款。
