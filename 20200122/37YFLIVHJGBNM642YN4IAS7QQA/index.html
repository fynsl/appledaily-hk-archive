【留名學跑】跑步界嘅「鯊魚衣」
揀啱裝備跑出PB

工欲善其事，必先利其器。在長跑比賽中要造出好成績，除了有好的體能、技術和心態之外，裝備都要好好配合， Nike Run Club香港總教練蘇凱男直言科技日新月異，現今跑鞋的回彈力更強，有助推遲跑手的疲勞感，對爭取佳績更有幫助：「馬拉松係講緊長時間保持速度，去到30公里後你有冇力，就取決於前面可唔可以慳力。」

眾人對如何選擇跑鞋有不同見解，蘇凱男指跑手應有兩雙跑鞋，分別應付練習和比賽，前者較為耐用，而後者則有助爭取佳績。不過，蘇凱男表示跑手需要一段時間適應鞋子，不宜穿全新跑鞋參加比賽，「我建議跑手着一對鞋，累計跑40至60公里先去比賽。好似馬拉松咁，可能試嘅距離要長少少，因為跑咁長時間，對腳個反應可能有少少唔同。」

雖然坊間有不少迷思，認為跑鞋薄底一定較為輕巧，但蘇凱男指現今科技進步，競賽型的跑鞋偏向厚底，保護性和速度並重。近日有英國傳媒報道，國際田聯（IAAF）正考慮禁用對馬拉松產生重大影響的Nike ZoomX Vaporfly NEXT%跑鞋，但蘇凱男直言此類跑鞋的碳纖板並非彈弓，跟當年使用浮水物料聚氨酯製造而加強浮力的「鯊魚衣」不能相提並論：「碳纖板個形狀係後面係比較平，前面有個匙羹型，目的係令你隻腳落地後有一個引導性向前，特別係對腳唔夠力時，可以好穩定咁落地，做一個重心轉移動作，就係碳纖板最重要嘅功能。」

除了跑鞋，蘇凱男指選擇衫、褲、襪也不能掉以輕心，這些衣物既要輕巧，也要排汗，不過最重要的是因應天氣而作出調整：「如果溫度太低，仲着背心短褲嘅話，溫度就會流失得太快。」
