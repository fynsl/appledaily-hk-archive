入場掃QR Code　填健康聲明　隔位坐
澳門賭場重開　荷官多過客

【肺炎大爆發】
【本報訊】因應武漢肺炎肆虐，澳門政府早前宣佈全部賭場停業15日，至昨凌晨零時29間賭場獲准重開，並設下多重關卡，包括賭客入場前必須填寫健康聲明，入場後要隔位坐，上班員工亦不可超過平日三分之一，賭枱不設企位，以防人群過度聚集。昨凌晨零時已有賭客急不及待排隊進場賭兩手。另外，澳門昨日加強入境檢測，要求曾到過武漢肺炎高發病地區的旅客及多次不正常往返澳門人士，須接受六至八小時醫學檢查，有疑似症狀即送院。

博彩監察協調局宣佈澳門41間賭場中有29間昨日凌晨重開，其餘12間延期開放，當中重開的賭場開放約1,800張賭枱，佔總體數量不足原有三成。

從疫區入境須強制檢查

澳門政府同時將14天內曾到過武漢肺炎高發病地區的旅客，包括廣東、河南、浙江、上海、北京及重慶，以及多次不正常往返內地的澳門居民，送往工人球場和氹仔客運碼頭的臨時醫學檢查站，接受六至八小時醫學檢查，其間每兩小時檢查一次。據統計，由凌晨零時截至中午12時，共有444名旅客和32名澳門居民被送往檢查站，有三人需送往醫院作進一步檢查，其中一人出現呼吸道症狀，另外兩人有其他症狀。有252名旅客不願送檢而選擇返回內地。

昨凌晨零時，澳門半島的美高梅娛樂場及新葡京已有人排隊等候進入賭場，有內地賭客稱已等候多時，心情特別興奮。有首批進場的澳門賭客稱，大約逗留一會，發現「基本幾十張賭枱，只有我一個喺度賭」，估計與遊客減少及經濟轉差有關。兩賭場約有10多名賭客，大部份都在玩角子機，包括小量內地賭客，稱來澳門數天感覺極為清靜，賭場重開便急不及待賭兩手，「有得賭沒有那麼悶！」有上班員工則稱，擔心感染武漢肺炎，已加強個人衞生防護、勤洗手及消毒。

位於氹仔的澳門威尼斯人度假村賭場則近乎無人，記者在入口守候15分鐘，只見有兩名賭客進內，他們進入前必須先掃二維碼（QR Code），於網上填寫澳門衞生局的個人健康聲明後，再用可在澳門接收短訊SMS的電話確認及填寫資料，並通過體溫檢測後，方可進入。

氹仔銀河娛樂場則有小量賭客消遣，賭客均戴上口罩，各人至少相隔一個座位。澳門人張先生表示，平日都有「賭一兩手」的習慣，故在賭場重開日亦專程入內觀察和賭錢，目測只有十數張賭枱有賭客，粗略估算只有70至80人在場內，以澳門人為主，相信因自由行客人大減，影響人流。
