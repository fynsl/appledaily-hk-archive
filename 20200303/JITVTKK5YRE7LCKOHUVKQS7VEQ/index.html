【財經專題】
銀公擬上半年招標　破環聯壟斷

載有本港逾500萬個人信貸資料的環聯資訊（TransUnion），是本地獨市營運的個人信貸資料服務機構（Credit Reference Agency，CRA），前年底該公司爆保安漏洞，令監管當局決心推動市場有多於一間CRA以減低風險。本報取得一家銀行的內部電郵，披露金管局及銀公檢討CRA的監管方法細節，並醞釀在今年上半年進行招標，引入新競爭者。

該電郵披露，銀公即將設立新架構，負責權力包括審批CRA申請、審視CRA提交的獨立評估報告及審計報告、委任評估機構對CRA進行調查的Multiple CRA User Group（MCRAUG），以及提供平台予業界與CRA進行交流的Multiple CRA Subscribers Forum（MCRAF）。文件亦顯示，該會有討論CRA招標程序，並羅列出一系列的招標資格要求，如公司歷史、財務狀況、行業經驗、技術能力、合規狀況等，並有相應的評估準則。

另有大銀行主管向本報表示，現階段未知銀公何時招標及招標條件如何，仍待工作小組建議上呈銀公大會確認，並與監管當局進一步磋商。不過無論如何，相關工作乃事在必行，相信透過招標引入競爭者，屬短期內可擴容多一間CRA營運商的最務實可行方法。

金管局發言人回應查詢指，鑑於有關項目較為複雜，銀公經內部討論後已委託顧問，協助研究落實相關安排。銀公發言人亦指，就本港設立多一家信貸資料庫，旗下相關工作小組正與金管局及第三方顧問積極研究，細節仍在商討，未有定案。

「中資」諾華有意申請

一直磨拳擦掌的潛在挑戰者諾華誠信（Nova Credit），行政總裁何佳意對本報稱，暫時未收到營辦多於一間CRA或入標等具體資料，「我哋任何時間都準備好，只要有清晰指引，便會遵守規定申請營運權。」有中資行股東色彩的諾華誠信於2017年底成立，中銀信用卡國際前舵手蘇誠信是該公司董事會主席兼創辦人之一。

對於銀公或招標引入競爭者，環聯香港發言人回應指，「信貸報告業務營運，需要良好往績並在世界各地具備相關經驗，環聯在全球超過30多個國家地區已營運逾30年。」環聯指一直尋找機會與金管局、銀公及其他政府部門合作。
記者：劉美儀　馬浩彬
