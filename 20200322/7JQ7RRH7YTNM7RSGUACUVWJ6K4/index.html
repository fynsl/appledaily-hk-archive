NBA球員點出糧 - 仙道彬

NBA球員可說是最令人艷羨的職業，球技高超，而且搵的錢也多，千萬年薪是平常事，近年億萬合約也越來越多；在比賽和訓練之餘，他們就似超級明星，行出來被球迷簇擁，一身奢華打扮，也將財富盡顯，可是搵得多，使得也多，加上黑人的「兄弟」文化，就算打一場比賽已等於普通人一年的薪金，很多時也只是掹掹緊。

所以當NBA自3月11日停擺後，雖不到兩星期，已有很多球員在社交平台表示關心出糧狀況，就連拓荒者球星麥高倫（C.J. McCollum）也在twitter出post，寫住no game no pay下，最好之前有好好儲蓄。其實NBA勞資合約中，的確有所謂的「末日條款」，要是真的認為今次武漢肺炎是觸發點，而餘下球季完能完成的話，那冇糧出絕對正常。

1.NBA球員薪金如何計算？

視乎球隊薪金上限，再按勞資合約中的條文說明簽約，例如新人和老將，都有不同指引，續約時也會因應年資及其他條款而制訂。以最低薪金計算，如從未有NBA經驗，如在海外打波，或大學時落選選秀的新人，最低薪金則是89萬8,310美元，至於10年以上的老將，例如安東尼（Carmelo Anthony、圖）加入拓荒者，他的最低薪金則是約256萬美元。當加入時餘下場數不足一季，則按比例計算。至於頂薪亦要視乎條件，包括要贏過MVP、總冠軍，或入選最佳隊伍等，今季人工最高的勇士球星史堤芬居里（Stephen Curry），年薪就高達4,020萬美元。

2. NBA球員如何出糧？他們是月薪還是年薪計算？

一如你我，NBA球員也是由球隊直接出糧到銀行戶口，但他們並非月薪或年薪，正常來說，薪金都是每兩周入賬，每月逢1號及15號就是出糧的日子。

3. 有沒有球員不喜歡每月出糧，要一炮過領取薪金？

當然有，好似陳年港產片《富貴迫人》般的橋段，拜恩（Kobe Bryant）在2013年就選擇在球季之初，就先收上期2,436萬美元，要是真的換成美金，就真的可以在銀紙山前面影相吧！NBA有規定，球員可與球會商討出糧方式，但這種一筆過方式，不可多於整體薪金的80%。該季Kobe的年薪為聯盟最高的3,050萬美元，第二名是奴域斯基（Dirk Nowitzki）的2,270萬美元；領取過大筆薪金後，之後由11月中到5月，Kobe一如其他球員，會每兩周領取一次餘下的610萬美元薪金。

4. 除了正常薪金，NBA球員是否有特別獎金？

當然有，為鼓勵球星有好表現，各隊都會度身訂造，訂下不同目標去讓球員「我要努力向上」。為免各球隊藉此走法律罅，所以也有嚴格規定，這些獎金的總額加起來，不得多於球員薪金的15%。

正常來說，這些獎金的達標方式都是贏得個人獎項或總冠軍等，但球隊也會視乎球員本身的問題，設定其他目標，例如易傷球員，就會在上陣50場至80場間，有數十萬獎金；也有不去控制體重的肥仔，會規定完季後仍要保持在一定磅數。

近年不時受傷的K艾榮（Kyrie Irving），就是其中一例，他今年有100萬額外獎金，分為8份各125,000美元，如上陣達70場，或罰球命中率達88.5%等，都可以有獎金。至於他的網隊隊友杜蘭特（Kevin Durant），也有100萬獎金，不過就相對易領取，如網隊打入季後賽，就有25萬。

5. 既已簽了合約，為何NBA球員還怕冇糧出？

要是球季不受影響，那有汗出有糧出，當然天經地義，可是球季暫停，NBA球隊也沒有了門票收入，那是否成為不出糧的理由？在NBA勞資合約中有個俗稱為「末日條款」（Doomsday pay provision）的細節，正式名稱為「不可抗力條款」（Force majeure event clause)，包括天災、戰禍和疫情等；究竟現時武漢肺炎是否瘟疫？可能要待勞資雙方再開會傾才有定案。

可以肯定的是，施華（Adam Silver）早前公佈了NBA暫停至少30日，但目前歐美疫情嚴重，要在四月重開NBA根本不切實際，那餘下約18場的常規賽，甚至是季後賽和總決賽，都難以全部完成，如果真的取消部份比賽，那每名球員將會每場損失了約1.08%的薪金，如super max合約的眾星為例，大約是每場不見了40多萬美元。之前聯盟有建議如順利重開，將會減少季後賽的數目，例如改為5場3勝制等，一切詳情，還是要待四月才有定案。
