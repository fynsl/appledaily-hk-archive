夥確診樂手表演　蘭桂坊歌手中招

【本報訊】昨日最少有兩宗新增確診感染武漢肺炎個案涉及蘭桂坊，本報翻查衞生防護中心資料，至今最少有16宗確診涉蘭桂坊及蘇豪區，合共最少三個感染群組。其中昨確診的第343宗39歲男子是一名歌手，3月21日喉嚨痛，無外遊紀錄，是第276宗29歲男樂手的同事，兩人到蘭桂坊多間酒吧表演，包括中環德己立街好利商業大廈地下的Insomnia（圖）。

女患者轉危殆需插喉

Insomnia昨晚照常營業，惟看不見客人，店員表示管理層指示不回答傳媒提問，未見有樂隊演出。另一新確診為第332宗32歲男子，潛伏期間去過英國，無病徵。衞生防護中心傳染病處主任張竹君指出，這名患者與第191宗34歲女子、第217宗35歲男子、第250宗27歲女子潛伏期間齊往蘭桂坊消遣，張不排除該患者是這群組的感染源頭。

第三蘭桂坊群組為第275宗30歲男子、第278宗28歲男子及第284宗26歲女子，三人曾到中環砵甸乍街45號H Code大廈消遣。位於H Code 6樓的酒吧Kagesha Lounge日前在facebook公佈，得悉一名確診患者曾於13日晚上10時到過該酒吧，酒吧已即時徹底消毒。

早前確診並曾到蘭桂坊消遣的第192宗46歲女患者，昨情況轉為危殆，需要插喉治療，據悉患者無慢性疾病，現於伊利沙伯醫院留醫。
