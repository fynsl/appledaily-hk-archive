【利字當頭】還得到又邊使借？ - 利世民

「賺夠了，也想別人賺一點。」話說，坊間有人發明了甚麼Loop Loop Loop大法。認真一睇，原來就是教人用銀行私人貸款買股票；以股息還債，當款項還清，股票便是永續有息收的生財資產。

Loop Loop Loop表面上萬無一失，但2008年滙控（005）才供過股，難道乜Sir不知道？其實，借錢買股票，也不用甚麼Loop Loop Loop來包裝；借就借，供就供。槓桿，當然可以放大回報，但不斷重複地對自己說獨門秘方有賺冇蝕，除了自欺欺人之外，沒有任何客觀現實因此而改變。

賭場常客，主要有兩類。第一種，以為自己有特異功能。第二種，知道自己是平凡人，所以找運氣好眼光準的人，跟手賭兩把。賭場裏真正賺錢的人，其實是老闆、叠碼仔、荷官等；要學，為甚麼不去學他們？「你估咁易咩??」「Bingo！你估咁易咩！貧窮局限你嘅想像呀！」

我沒有資格教人如何投機發達；但分享如何過舒泰小資產階級生活，我想自己是綽綽有餘。首先，假如你懶得去做，就要勤力去思考。假如懶得思考，就勤力去做。當然，若是撫心自問又勤力又思考不倦，但始終沒有甚麼成績，可能你真的非常不幸，又或者，真的十分蠢，無論如何，這種人更加不應信神話故事，認命就好。

若非法規所限，Loop Loop Loop的pitch，用來幫銀行和財務公司賣廣告，實在上乘。與其辰時卯時，亂打電話問人有沒有卡數要清，不如正面誘發貪念：「先生，你信唔信世界上有零風險不勞而獲嘅投資呀？」借定唔借？還得到又邊使借！
