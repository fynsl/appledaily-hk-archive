777盛讚警隊表現專業　劈炮專家嘲報告塑「新真相」

【撐警報告】
【本報訊】特首林鄭月娥昨日在「香港的真相」螢幕背景下回應監警會報告，盛讚警隊毋庸置疑是專業。因不滿監警會淪為無牙老虎而劈炮的國際專家對林鄭盲撐警隊表示無言，並指監警會報告只是一系列經協調下傳遞的「新真相」，不太可能消除港人對警方執法的憂慮，必須要有真正獨立的調查機構。

記者：趙雅婷　麥志榮

監警會曾經委託五名外國專家負責提供意見協助檢討警暴問題，惟專家小組因監警會缺乏調查及搜證能力，去年底集體請辭。五名專家之一的英國基爾大學自然科學研究院院長Clifford Stott昨日不停於Twitter上發文評點報告。對於特首林鄭月娥昨在記者會上指巿民對警察的仇恨是由媒體塑造而成，警隊「毋庸置疑」是專業，Stott回覆本報查詢時對此表示無言（Speechless）。

再倡設獨立調查

Stott昨在Twitter上評點多個報告要點，當中特別指出，報告內亦承認所有資料都屬自願提供，監警會並無調查權力強制披露資料。對報告提及到7.21元朗事件，指警方錯失黃金機會向公眾解釋事件，令警方成為公眾怨恨的目標。Stott的帖文指，7.21事件後，市民認為警方不再具合法性，惟監警會再次以「市民對警方有誤解」作為解釋。Stott又節錄報告指新屋嶺紀錄系統不足，指在8月11日的紀錄系統全由人手操作，造成個別資料以至相關就醫要求的資料不完整。

Stott指，報告雖然詳列各大型事件，惟欠缺了部分關鍵，指報告不太可能消除港人對警方執法的憂慮，他認為一個真正獨立及擁有調查、取證權力的調查機構是基本要求，這樣才可核實警方、示威者以至其他持份者說法的真確性，他認為只有這樣才可讓廣大市民對其作出的報告安心。

質疑報告經協調

Stott引用著名英國作家喬治˙歐威爾的名言「一個社會越是偏離真相，那些說出真相的就越是受到壓迫」，並指出，監警會報告似乎是經過一系列協調下所傳遞的「新真相」（It would seem the release of the IPCC report is part of a wider set of coordinated announcements designed to deliver the new 'truth'.）。

Clifford Stott去年辭職後曾公開指出，外國專家建議監警會應被賦予權力，可傳召證人及搜集證據，報告才能有公信力。
