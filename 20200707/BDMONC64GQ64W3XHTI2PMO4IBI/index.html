評F1講「武肺」被指言論不恰當　
陳恩能堅拒Now Shut Up

2020年F1賽季首場賽事周日落幕，惟作本地直播的Now TV卻有插曲，前晚正賽末段，評述員陳恩能的旁述突然被中斷，Now TV母公司電訊盈科其後發聲明表示，對陳恩能有關疫情的言論「深表憤怒和遺憾」，並指其言論極不恰當，兼帶歧視及誤導。陳恩能疑因以「武漢肺炎」形容疫症引爭議，網民議論紛紛，不滿Now TV自我審查，有人發起向電視台傳電郵支持陳恩能言論。

F1奧地利站前晚落幕，賽事於Now TV中的Fox Sports亞洲頻道直播，並由陳恩能負責廣東話旁述，及至直播末段其旁述突被中斷，網民指電視台初以「技術出錯」解釋，但電訊盈科其後發聲明，表示對陳恩能就「新型冠狀病毒疫情作出的若干言論，深表憤怒和遺憾，指其言論極不恰當，兼帶有歧視及誤導成分」，並指其言論絕不代表Now TV立場，就此向受到冒犯觀眾致歉，並向Fox Sports表達不滿，要求他們立即正視和交代。

網民轟電視台自我審查

網民懷疑陳恩能直播期間以「武漢肺炎」形容疫症惹爭議，炮轟Now TV自我審查，紛紛於陳恩能的fb專頁留言支持，並指會向Now TV發電郵投訴電視台滅聲。本報曾向陳恩能查詢，他指暫不回應，不過事後他於fb專頁開帖提及事件，表示「哦？乜俾人早cut咗咪咩？Any肥，多謝賽前《F1吹水站》破紀錄1,600人同時睇LIVE」。陳恩能為本地著名體育賽事評述員，並以藝名「丹尼爾」主持電台節目，過往他敢於發言，多次於社交平台就政治議題表態。

今次事件絕非空穴來風，賽前他於YouTube直播中，透露自己隨傳媒以「俗稱」武漢肺炎稱呼疫症卻接獲投訴，備受壓力，呼籲聽眾電郵予Now TV展示支持，「武漢肺炎就係俗稱，你可以叫新冠肺炎，我都係根據香港主流傳媒之一《蘋果日報》同《立場新聞》去講。我發現有朋友mon住我，你可以繼續投訴我，亦都可以幫手email去Now，讚我又得、鬧我又得，畀番啲支持我啦，因為都有啲壓力。」

反駁沒理由不准用俗稱

他指疫症名稱眾多，不理解為何被禁說「武漢肺炎」，「你可以講英文，你可以講學名，可以講拉丁學名，冇理由唔畀我講俗稱。」他明言早已預料會有後果，「講得俗稱我就預咗，係咪下下要講學名？咁我哋唔可以講『香港腳』，咁要講足癬。」
