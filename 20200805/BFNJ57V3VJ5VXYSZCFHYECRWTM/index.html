【小心駕駛】韓國藝人頻遇交通意外

韓國演藝圈近年頻生交通意外，有藝人更因此受重傷或身亡。

朴智允 全家受傷

韓國41歲女主持朴智允一家四口，上周於釜山遇車禍，當時朴智允一家的座駕與一輛逆行的貨車相撞，朴智允跟老公均告受傷，一對子女亦受輕傷，貨車司機因涉嫌酒駕而受查。

金柱赫 搶救不治

曾出演韓劇《布拉格戀人》以及綜藝節目《兩天一夜》的男星金柱赫，2017年10月於首爾三成洞發生車禍，疑因心臟病發導致意外，送院搶救後不治，終年45歲。

Ladies' Code 兩死五傷

五人女團Ladies' Code於2014年9月在大邱趕回首爾途中發生嚴重車禍，兩位成員高恩妃（中）及權梨世（左二）傷重不治，意外造成兩死五傷，餘下三名成員一度停工休養一年半。

大成 超速捱告

組合BIGBANG成員大成於2011年疑撞死人遭起訴，當時死者因酒駕而受傷倒地，隨後大成超速開車駛至，將他輾過並拖行約22.8米，不過調查後大成被判「冇嫌疑」。
