財經專題：中美逢源　班農拍檔來港覓食　
拓一帶一路　牽涉中信常振明

白宮前首席戰略顧問班農（Steve Bannon），早前眾籌2,500萬美元，於美墨邊界興建圍牆，近日涉從中欺詐被捕。本報發現，班農拍檔活躍香港股壇，當中包括上市公司先豐服務副主席普林斯（Erik Prince），協助內地搭建一帶一路項目，並牽扯央企中信集團及其前董事長常振明。

中美兩國目前關係雖然緊張，但過去一直有「政治中介」，借助香港資金能自由進出的特性，遊走於中美之間覓食，是繼白手套及間諜外，另一種活躍於香港的政治動物。

特朗普2016年總統競選最大功臣、白宮前首席戰略顧問班農，近日被控於「我們興建圍牆（We Build the Wall）」的眾籌中，挪用部份資金作私人開支。該計劃原是為支持政府在美國南部邊境興建圍牆，阻擋來自墨西哥的非法入境者；有外媒透露，班農被捕時身處中國富豪郭文貴的豪華遊艇上。

「僱傭兵之王」曾任先豐主席

據We Build the Wall網頁，班農為組織顧問團主席，顧問團中出現另一位特朗普競選團隊成員普林斯（Erik Prince）名字。Prince曾為美國海豹突擊隊成員，9.11後成立私人保安公司黑水（Blackwater），因伊拉克戰爭成名，被稱為「僱傭兵之王」；及後爆出虐待戰俘醜聞，普林斯2010年出售公司，因而轉到香港搵食。

2013年，有「殼王」之稱、曾是蔡東豪舊老闆的高振順，其持有的天地數碼（500，現稱先豐服務）向普林斯以300萬美元及授出期權，收購其一間名為Frontier Services的非洲物流保安公司。當時天地數碼的第二大股東正是中信集團（267，現稱中信股份）。中信早於2006年已入股並安插多個高層進入董事局，包括現任中信國安董事長羅寧等；普林斯其後獲委任為該公司主席，並將公司易名。高振順之後減持股份，退任為公司副主席。

普林斯藉先豐服務這個平台，多次收購非洲的大型物流公司及航空公司；此後發展老本行保安服務，並取得大量中資合約。據外媒報道，中資於一帶一路的大部份私人保安合約，都落入先豐手中；先豐服務還曾接到中冶合約，到巴基斯坦為中國員工安排安全培訓，包括面對恐怖分子的策略、綁架的預防與應對，以及簡易爆炸裝置的預防和應對等。

據港交所資料，先豐服務2017年收購北京國際安全防衞學院25%股權，該院校2011年成立，為中國首間防暴反恐培訓學院，也是內地唯一為軍警提供反恐培訓的民營公司，號稱亞洲最大私人安全防衞基地。至2018年，先豐服務年報指出，公司業務已橫跨一帶一路的途經路線國家包括西北的哈薩克斯坦、烏茲別克斯坦及巴基斯坦；西南的緬甸、老撾、泰國及柬埔寨，還有中東的阿聯酋、非洲的肯尼亞、剛果。

疑接新疆集中營項目

2018年尾，先豐服務向中信股份、中國太平（966）及國開金融滕榮松等配股，集資逾8億元；完成之後，普林斯股權攤薄至12%，並行使部份購股權，同時在場外賣走手上近4%股份，套現超過1億元。

原本是第二大股東的中信，取代普林斯成為最大股東，現時持股增至約25%，中信董事長常振明更加進入先豐服務擔任非執行主席；去年更傳出該公司接下新疆再教育營項目，每年為新疆8,000人提供訓練，惟集團其後否認。

普林斯及後退任公司副主席，與中資仍來往甚密；去年有報道指他計劃融資5億美元，投資可開採電動車電池礦物的礦場，該公司主要目標顧客亦為中資。

普林斯身為班農拍檔，能牽連到反共的郭文貴與央企中信及前董事長常振明，引證利益當前、背後複雜關係並不是一般股民所能想像。
