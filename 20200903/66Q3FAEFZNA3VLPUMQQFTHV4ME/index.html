論林鄭落台　周凱旋：萬物有時

【本報訊】香港制度崩壞，是非黑白顛倒，始作俑者林鄭月娥責無旁貸。「超人」李嘉誠紅顏知己周凱旋本周初在一個活動上罕有談到社會各項議題，當中被問到林鄭班子落台是否良方時，她語有深意地說：「我自己是基督徒，我信《聖經》所說，There is a time for everything，不用心急誰和誰要落台。」

去年初政府修訂《逃犯條例》觸發的社會事件至今仍未平息，李嘉誠去年8月曾以「黃台之瓜　何堪再摘」籲當權者對香港留手；9月更明確表達立場，提到希望年輕人能夠體諒大局，執政者亦能夠對未來社會主人翁網開一面，溫和言論竟然遭到猛烈抨擊。一年後的今天，維港投資創辦人、李嘉誠紅顏知己周凱旋在一個科技活動上再談香港現況，本報記者向她提到香港情況惡劣，連國安法都實施了，社會面臨激烈矛盾，林鄭政府及其班子是否要問責落台？

「整個經濟，應該擁抱甚麼方向，可以為年輕人創造更多未來，這是更值得大家深思的地方。」

周凱旋指：「我自己是基督徒，我信《聖經》所說，There is a time for everything（傳道書3章——「萬物有時」），不用心急誰和誰要落台。現在全世界的挑戰，無論在經濟上，或者在inequality（不平等）方面，都是很大的（問題），There is a time for changes coming，整個經濟，應該擁抱甚麼方向，可以為年輕人創造更多未來，這是更值得大家深思的地方。」

2017年香港特首選戰，李嘉誠未有明言支持林鄭月娥、曾俊華或胡國興，但曾經借古代神話「女媧補天」呼籲香港解決日益加劇的撕裂問題，認為一切都是事在人為，女媧當時被視為是隱喻女特首（林鄭）上場，惟今天撕裂卻遠比任何一個時期嚴重。

對於政府施政失當，周凱旋退一步說，「李生（李嘉誠）也常常說，批評別人是很容易的，我們大家也喜歡做armchair politicians（紙上談兵），全部都挑骨頭，我自己也天天在挑骨頭，但是事實上是否做得差到如很大陰謀論般，我個人又不覺得；但是否可以做得更好呢？永遠都可以做得更好」。

過去一年多每個重要日子，香港人未能平反，難以忘懷，有人繼續抗爭，有人選擇移民，香港面臨人才及資金流失，周凱旋引述李嘉誠說話，希望大家冷靜面對：「憤怒不是動力來的，李生或者我們很多同事，包括我自己，都是由很困難的環境走出來；如果只覺得憤怒，經常覺得被邊緣化，經常覺得不開心，對自己的未來是沒有幫助的，你要為自己的未來，無論去到哪個地方，或留在香港，或找到其他機遇，你都要帶着自己的心去，要勝過別人，先要勝過自己。」

對於警察處事手法無章法、無機制，導致警民嚴重對立，周凱旋直言：「如要挑剔，就總有錯，每個社會也是這樣。」但她未有完全排除警察的貢獻，包括治安和繁榮，「如個別案例，有些人做得不好，不要攻擊全個警隊，正如有個別示威者暴力，就把所有示威者視為很差，這也是不好，大家要從理性角度考慮問題，不要陰謀論，做到你們自己心目中理想的人，不要專注個別的人」。
