抗疫不疲勞｜運動達人教你隨時隨地Keep Fit 8招

【特約報道】減肥修身彷佛是很多人的終身事業，受疫情影響，越來越多人養成做運動的習慣，workout不止為了有完美體態，更重要是得到健康的身體。說到底，抗疫最基本就是透過運動提高與保持好的抵抗力。但其實剛剛接觸訓練的你，又要注意什麼？兩位來自adidas的運動達人Lawrence和Czon，與大家分享8招簡單而有效的workout法，使用在家中容易找到的「訓練器材」，讓你隨時隨地無論在室內或室外，都可以動起來！

Workout注意事項

運動前一定要熱身、熱身 & 熱身！這是我們都知道的道理，但不少人仍會忽略。以下的訓練雖然簡單，不熱身的話可能有受傷的風險，Lawrence亦絕不建議大家在身體疲勞的情況下進行訓練。

另外，Lawrence提醒我們選擇合適裝備亦相當重要，例如當大家專注做一些跑跳訓練時，運動鞋的選擇就很關鍵。「今日我特別穿上Adidas Alphatorsion Boost波鞋，它備有360度Torsion系統，能增加着地時的穩定性，減低腳踝扭傷的風險；鞋底的坑紋能提高抓地力，做運動時減低受傷的機會。」

而Czon穿的Adidas Alphatorsion Boost，Boost中底的技術有能量回饋的功效，令她做跳躍動作時增強爆發力，減少能量消耗，讓她做到最後仍然有力。「不少朋友喜歡跑街，波鞋上Adidas標誌性的三間以反光物料打造，增添跑者的能見度之餘，設計亦十分型格。」Czon補充。

動作1：Burpee Jump

訓練：全身肌肉的協調性、穩定性和心肺功能

注意：跳起時着地要輕力

步驟：

1. 雙腳與肩同寬自然站立 2. 彎曲膝蓋盡可能深蹲 3. 蹲下後雙手撐在地面 4. 雙腳向後用力蹬至高平板式姿勢 5. 雙腳跳回深蹲時姿勢 6. 向上跳起來回到站姿着地

動作2：Shuttle Run

訓練：大腿、平衡力和心肺功能

道具：水樽 X 2

注意：向後退時要注意保持平衡

步驟：

1. 將水樽放於前、後相隔兩米距離

2. 向前跑繞過水樽

3. 向後退至另一個水樽

4. 指尖輕碰水樽頂為一圈

動作3：Side Shuffle

訓練：大腿、平衡力和心肺功能

道具：水樽 X 2

注意：動作須要急停、轉向（膝蓋有傷患者要特別小心）

步驟：

1. 將水樽放於左、右相隔兩米距離

2. 從一個水樽橫移向另一個水樽

3. 急停、轉向至原來位置

動作4：Bear Crawl

訓練：四肢肌肉、軀幹協調性和穩定性

步驟：

1. 伏地挺身向前爬行兩米距離

2. 向後爬行退至原來位置

動作5：T Pose

訓練：大腿、手臂和平衡力

道具：毛巾

步驟：

1. 雙腳站立

2. 雙手用力抓實毛巾輕碰大腿

3. 雙手向前伸的同時，單腳向後伸直

4. 保持姿勢後返回原來位置

動作6：180′ Squat Jump

訓練：大腿、臀部及心肺功能

道具：水樽 X 2

步驟：

1. 將水樽放於你的前、後位置

2. 向其中一個水樽蹲下

3. 指尖輕碰水樽

4. 轉身跳180度

5. 向另一個水樽蹲下

6. 用另一隻手的指尖輕碰水樽

動作7：Russian Twist

訓練：下腹和大腿

道具：咕𠱸和水樽

步驟：

1. 坐在地上的咕𠱸上，水樽放在前面

2. 雙腳放在水樽的一邊

3. 提起雙腳跨過水樽至另一邊

4. 同時將上身扭動向相反方向

動作8：Knee to Elbow Jump

訓練：腹部和大腿

步驟：

1. 原地跳起

2. 提起左膝輕碰右手手肘

3. 回至原來位置再跳起

3. 提起右膝輕碰左手手肘

*以上動作重複做30秒，完成一個動作休息30秒，再做下一個動作。
