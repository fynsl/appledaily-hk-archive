錯過交瓶　長洲居民醫院潑口水　護士被濺中　醫管局無追究

錯過交瓶　長洲居民醫院潑口水　
護士被濺中　醫管局無追究

【本報訊】一名長洲居民因不滿收集深喉唾液樣本的安排，大鬧長洲醫院。該居民拿着裝有深喉唾液的樣本瓶到長洲醫院門診提交化驗，但因過了收集時間被拒，居民一怒之下將瓶中唾液潑在枱上便一走了之。有醫護人員透露，當時護士的制服被唾液濺濕，但醫管局無報警追究。有區議員指出，由於近日大量旅客湧入長洲度假，居民擔心疫症傳入，令樣本瓶需求大增。

醫院管理局總行政經理劉家獻昨表示，長洲醫院門診本月3日上午11時有一名居民遞交深喉唾液樣本，由於已過10時截止時間，護士建議他翌日重新提交，惟居民不接受，並將樣本瓶扭開，將樣本倒在枱上。當時只一位護士在場，無其他病人。護士有穿合適防護裝備，但為謹慎起見建議她做病毒測試及醫學監測。醫管局對該人的行為表示十分遺憾。

劉家獻表示，若有人故意在醫院或診所搗亂，醫管局附例可以追究，但該人在倒完樣本之後就立即離開，沒有留下資料。醫管局明白公眾可能會心急接受檢測，但呼籲切勿作出罔顧公眾健康的行為。醫管局亦會檢視收集時間是否能更方便公眾。

疑沒穿保護衣　醫護斥欠保障

不過有長洲醫院醫護人員向本報透露，當日的護士沒有穿保護衣，制服被撥出的口水濺濕，但管理層沒嚴肅處理事件，沒有即時召保安及報警，翻查閉路電視片段追究該缺德市民，讓事件不了了之，沒阻嚇力對醫護人員缺乏保障。

根據《醫院管理局附例》，任何人不得在醫院內使用可能令他人厭惡或煩擾的言語，或作出不雅或影響秩序的行為，如向職員施襲、嚴重妨礙醫院運作。違例者首次定罪可處罰款1,000元，再度定罪則可處罰款2,000元及監禁一個月。

每日派90瓶　難應付島民需要

離島區議員梁國豪表示，近日每逢假日都有大量旅客湧入長洲，居民擔心引致病毒傳播，所以紛紛取樣本瓶檢測，但長洲醫院每日限派90個樣本瓶，他已要求院方增加數目。

他指，長洲早上10時便截止收集樣本瓶，因要運回市區化驗，派樽的護士都有向居民講解清楚。今次事件可能源於溝通誤會引起，尤其是一些年長居民未必了解有關資訊，他會要求院方加強宣傳。
