Coach妝語

彩妝市場龐大，Z世代在奢侈品消費中已經佔着越來越大的比重，為了吸引更多年輕消費者，品牌都喜歡借推出限量系列，提高話題與吸引度。Coach近日與Sephora合作推出Capsule Collection，產品飾有Coach標誌的Sculpted C 標誌、玫瑰花，以及品牌過去用在服裝和配飾上的元素，包括恐龍Rexy眼影組合及Sharky眼影組合、具有四種光澤度的Tea Rose唇膏套裝等。

Bvlgari香檳

意大利高級珠寶品牌Bvlgari近年明顯注入不少新元素，最新是與頂級香檳品牌Dom Perignon合作，推出Bvlgari Serpenti x Dom Perignon Rose。限量版木箱可見到特製的內部，以Bvlgari標誌橘黃及深紫紅色縫製，每套作品均包含Serpenti頸鏈及三瓶1.5公升Dom Perignon x Bvlgari Limited Edition Vintage Rose 2004。這次聯乘只出四套，今年起於倫敦、米蘭及東京承訂後製作，絕對是「有錢都未必買得到」。
