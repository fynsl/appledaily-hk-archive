聽風的歌 - 林夕

林振強為王菲〈如風〉寫人「來又如風，離又如風」，如今聽來，是在說小粉紅嗎？愛之去來，亦如跟風者以恨咬人，今日砸這，明天燒那，以為發自本心，那心也不是自由的。

人造風是人禍，大自然之風過猛是天災，而這其實相通。何以有風？還記得當天課室的情形，還留住發問回答的熟性；老師問，什麼是風？本人身為舉手王，過早學會搶着爭先表態，立刻站起來回答：「空氣的流動叫做風。」老師說：「對。空氣冷縮熱脹，高氣壓流向低氣壓，就形成風。」

那麼，就是說：以大欺小、以高壓低，於是人間有風？

四十年代初，毛澤東在延安發起整風運動，就是整治不羈的風，「反對主觀主義以整頓學風，反對宗派主義以整頓黨風，反對黨八股以整頓文風」，整頓「三風」當代版即係批鬥教育、行政、文化三個界別，這場小文革風一吹，死掉上萬人。

迫害知識份子的跟風者，最後成就了毛氏一人絕對領導地位；只有老毛才有資格高歌「不羈」的風。共犯最後不只被只視為空氣，誰有點不羈，一個個都不得善終。

說到這裏，小鳳姐又出場了：「吹啊吹，讓這風吹」。少年的我懂得風之起源，念念不忘，到變成大人，終有迴響，大發悲憫之情：風啊風，以為你一向任性，哥哥張國榮用風的唱腔唱你：「從前如不羈的風，不愛生根，我說我最害怕誓盟。」

不羈也是一種自由，原來你比人還要身不由己，可憐啊。2008年，把明周「十方一念」專欄結合成詩集，收錄了<風的自由>，「風自覺任意而為 / 對滄海一聲豪笑 / 空氣靜坐時想 / 你不過是我一個嘆氣 / 我有壓力/ 你才能揮舞旗幟 / 天下萬物最身不由己的 / 就是所謂/自由的風」。

在這風的季節，香港身處於龍捲風眼，誰是那自覺任意而為的暴風，誰是那靜坐的空氣，請自行對號入座。不羈的風大部分都被羈押了，不得自由。至於我們要做疾風，搖得樹枝拋棄樹葉，做暴風，猛得樹身低頭認輸，還是要做和風，吻得樹葉舞出形態，這如呼吸空氣般的自由，還是沒有失去的。
