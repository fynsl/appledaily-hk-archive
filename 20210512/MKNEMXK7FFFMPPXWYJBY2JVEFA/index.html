投資加密貨幣｜新經濟淘金字典　入市前必須要知的Term

特約報道｜ETH、NFT、IPFS、DOGE、DOT，好像密碼一樣的英文縮寫，當然不及香港樓盤名字易記，但現在日日上網睇投資消息，總有一兩條是關於加密貨幣的，不單升幅驚人，而且越來越多大公司都開始接納加密貨幣，好像特斯拉容許客戶以比特幣買車，公司亦曾買入價值15億美元比特幣，而VISA、Mastercard、BlackRock及Bank of New York Mellon等大型機構，亦陸續接受或投資加密貨幣，作為投資或服務組合一部分。所以作為一個投資者，如果不認識加密貨幣，隨時錯失很多機會！

BTC不可忽視的加密貨幣一哥

比特幣（BTC）一直是加密貨幣的一哥，雖然4月中曾一度跌至5萬美元以下，但很快就重拾升軌，比特幣有點類似國際貨幣中的美元，市值也長年維持在整體加密貨幣市值的6至7成，不過隨著越來越多人開始買入其他加密貨幣，以追逐更高回報率，現時比特幣於加密市值佔有率為5成左右。由於過去兩年比特幣以至整個加密貨幣陣營都是大漲小回，所以比特幣（BTC）已經是每一個投資者都關注的項目，就連最近投資界盛事巴郡股東大會上，股神巴菲特拍檔芒格亦罕有地評論比特幣，可見比特幣已經不容忽視。加密基金Pantera Capital去年4月曾預期比特幣一年後可升至6萬美元，雖然最近略有回落，但的確曾衝破6萬美元，Pantera Capital最近又估算比特幣將在明年可達20萬美元。這些大利好的預測以及過去兩年比特幣的表現，都令到BTC達到神枱級的地步。

IPFS去中心化儲存技術衍生FIL幣

IPFS (InterPlanetary File System ) 是另一個近幾年炒得火熱的加密題材，原理是以P2P 網絡達成去中心化，與BT的原理相似，有低成本、高效率及更高隱私度等優點，目的想填補傳統網絡的HTTP協議，當然想IPFS技術成功普及，前提就要鼓勵用家提供數據儲存服務，分享自己的電腦儲存空間。所以提供IPFS數據儲存空間，可獲得虛擬貨幣Filecoin (FIL幣）作為獎勵，當然亦形成另一個投資Filecoin的市場，對比一年前不到8美元，今年4月已升至190美元，升幅比起一眾虛擬貨幣更瘋狂。

NFT令數碼作品終於有價

NFT（Non-Fungible Token）是以區塊鏈技術產生的獨有標記，由於有不可更改的特性，最適合斷定數碼物品的真偽及擁有權。最大的好處是以往只有實體藝術品因為其獨一無二的特性，才會有價有市，而數碼作品易於複製，一向都沒有交易或炒賣價值。現在透過NFT技術，「NFT藝術作品」都擁有獨有的數碼標記，擁有搶購與收藏價值。3月佳士得拍賣行就以6900萬美元售出美國藝術大師Beeple的加密作品《Everydays-The First 5000 Days》。值得留意的是NFT可以是數碼圖片、也可以是影片、音樂等，音樂人陳奐仁（Hanjin）日前就以NFT形式推出新歌《Nobody GeMe》，七秒MP4單曲連封面及與陳奐仁以zoom對話一小時公開競投，最後以七個以太幣（約11萬港元）成交。由此可見，不見加密貨幣有得炒，連NFT數碼作品也開始有投資價值，不過相信與真正藝術作品一樣，最後是否水漲船高，就要看你的眼光了！
