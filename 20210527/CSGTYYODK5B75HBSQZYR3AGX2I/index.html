【八十後投機客】深化市寬看後市 - 灝昇

投機大師Mark Minervini常鼓勵交易員專注於個股走勢而撇開指數，事關市場出現大牛市或大熊市的機會即使近年已越趨頻繁，但亦屬罕見。過去半年，不論港股美股，二三線股份和指數走向都差天共地。最近即使納指持續破頂，但大部份二三線股亦比高位跌近一半。所以要分析大市有沒有主題好炒，或是反彈有沒有持久力，亦應顧及市寬。

市寬是指大市強弱股份的分佈，較常用的是股份高於50天移動平均線比率，或是20天移動平均線高於50天移動平均線的比率，通常為單一數值。不過筆者習慣根據股份的強、中、弱三個狀態觀察大市市寬。普遍認為MACD處於階段4是下跌階段，為之弱股，而MACD階段2則是上升階段，其餘階段1、3則是整固/反彈浪，為之持平階段。

觀察階段4的市寬比例由5月14日（指數見低位當日）的30回落至26日的22.1，而同期階段2的市寬則由14日的23.3略升至30.11，代表市場有近8%由弱轉為持平走入反彈階段，同期更有6%股份轉為強勢，反映市底硬淨，可是5月25日時指數已升破5月3日位置，惟5月3日階段2市寬為32.67，而5月26日則只有30.11，即與指數有背馳，所以大市再向上動力未必太強，可推斷大市於29300附近容易遇上阻力，但不至於出現5月初的下跌趨勢，比較可能在28650附近找到支持，並形成波幅區間，資金將流入階段4轉1和階段2的股份，例如正在形成cup and handle set up的3998、正進行square set up的882都是考慮之列。
