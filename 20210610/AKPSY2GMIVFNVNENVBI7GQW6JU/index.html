【八十後投機客】贏家遊戲vs輸家遊戲 - 灝昇

投資是一個機率的遊戲，今天想和大家分享一下對「勝率」的看法。

當談及勝率時，大家往往會想到如何增加贏錢的機會率，這是一種勝出「贏家遊戲」的思維模式。反過來卻很易忽略如何減少犯錯，以勝出「輸家遊戲」的思維模式操作。「贏家遊戲」與「輸家遊戲」查爾斯．艾利斯（Charles Ellis）提出的概念。在「贏家遊戲」之中，勝出是透過做出比對手厲害的決策、技術等等而獲勝，在股市之中可以想像是選股技巧、止蝕止賺動作、資金運用的有效性控制等等。在「輸家遊戲」之中，勝出則是取決如何減少錯誤，避免做出不必要的錯誤，以順勢操作為例，在趨勢不明顯時如何減少出手，避開值博率不高的股份等等。

查爾斯提出這兩個概念是認為，投資者只要願意持有指數型ETF便可以勝出投資這個遊戲，而不必過多操作。當然這是一種極端，完全撇開「贏家遊戲」的思維，大多數時間我們可以同時擁有「贏家遊戲」和「輸家遊戲」的思維，至於判斷如何需要減少錯誤操作，最核心便是了解自己交易策略的強弱，亦敢於以少注測試市場。如同上周本欄所言，根據市寬，大市仍處於28650-29450的波幅區間，大市在早2天下試完28650，亦未見有突出版塊，所以維持區間睇法，相信區間底部要再調低至28200。

既然大市趨勢不明朗，下行空間亦未算太大，搬出「輸家遊戲」的思維模式，沒有一副靚牌時，盡量減少出錯，減少注碼是上策。如戰意仍高，新地（016）接近完成square box set up 形態，如升穿124元可上試130元，而MACD在階段2由收窄轉為拉闊，相對大市強勢，以20天線作止蝕，小注以衍生工具偷步，仍有一定值博率。
