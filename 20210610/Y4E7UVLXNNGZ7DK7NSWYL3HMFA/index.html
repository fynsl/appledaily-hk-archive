【add名人時尚】聯乘優勢

停不了的聯乘浪潮！今個月除了期待已久的Yeezy Gap系列登場外，還有Nike Lebron 19、Sacai x Nike Blazer Low、Rick Owens x Birkenstock，以及Netflix x Halston系列等。

YZY預售外套

由Kanye West主理的Yeezy與Gap去年６月宣佈推出聯乘YZY，等候超過一年，Gap在前日終於發放首件單品Round Jacket。外套採用再生尼龍物料，彩藍色設計，面料帶有亮光質感，整體呈Yeezy標誌性的簡約線條，售價200美元（約1,560港元），已於美國地區進行網上預售。早前外界曾估計YZY系列有望為營業額低迷的Gap帶來近10億美元（約78億港元）進賬，業界都非常關注Kanye West能否以一人之力令Gap翻身。

Nike Lebron 19接棒入樽

由NBA球星勒邦占士（LeBron James）接棒佐敦（Michael Jordan）演出的《太空也入樽：改朝換代》（Space Jam：A New Legacy）預定下月在港上映。片中勒邦占士穿上Nike特別為電影推出的LeBron 19別注系列。另外，Converse也分一杯羹，攜手為電影推出聯乘鞋款。

Sacai Nike Blazer Low夏日拼色

Sacai與Nike的合作雖然不是新鮮事，但每次推出的聯乘作都成為搶手貨。今回登場的Sacai x Nike Blazer Low系列分別有橙黃（圖）與灰綠兩種配色。設計上保留Blazer Low原型，配以交叠的拼色Nike logo及鞋舌，打造充滿夏日感的時尚設計，定價120美元（約936港元）。

Rick Owens Birkenstock暗黑涼鞋

Rick Owens x Birkenstock系列已來到第三回，對比之前的設計更顯暗黑大師Rick Owens的魅力，除了全黑色設計的經典涼鞋款外，更出現黑色漆皮及全黑金屬件設計，新品已登陸雙方官網。

Netflix Halston 上乘晚裝

Netflix時尚美劇《侯斯頓傳奇》（Halston）講述美國60年代設計師Halston的傳奇，其中他曾是美國第一夫人Jackie Kennedy的帽子設計師，Netflix將與品牌Halston推出限量晚裝系列，設計以真絲和雪紡等上乘物料打造多款高貴優雅的裙子，售價由945至1,595美元（約7,371至12,441港元），已於Halston官方網頁上架。
